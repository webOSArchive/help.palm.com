Customize Contacts
  1. Open Contacts  > Menu (upper left) > Preferences & Accounts.
  2. Set the following options:

    List Order: Tap to select the criterion by which the contact list is sorted. If you select Last name to sort on, the names are alphabetized by last name, but they still appear in the Contacts list as first name followed by last name. If you select First name to sort on, names are alphabetized by first name.

    Accounts: Tap an account to change login settings and sign in using the new settings.

    Default Account: Tap the current default account > tap an account. New contacts you create are stored in the default account unless you specify otherwise.

    Sync Now: Tap to force synchronization of your online accounts.

    Add An Account: Tap an account type > enter the account information. Before you can set up an account on your smartphone, you must already have an account online with the provider.
