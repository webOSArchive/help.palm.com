Set up email: Microsoft Exchange and Lotus Notes

Before You Begin

Get this info from your email provider or system administrator:

  • Mail server name or IP address for receiving mail
  • Server domain name
  • Your username and password

BYB Close

  1. Open  Email .
  2. Do one of the following:
    • The first time you open Email, tap Microsoft Exchange.
    • Any other time: In Email, tap Menu (upper left) > Preferences & Accounts > Add Account > Microsoft Exchange.
  3. Enter your email address and password > Manual Setup.
  4. Enter info for the incoming mail server as needed, based on the server information you got from your email provider or system administrator (see Before You Begin, above). You can enter either the server name or IP address in the Incoming Mail Server field.
  5. Tap Sign In.
  6. To keep your Exchange account secure, you may be required to create a password or PIN that is required to unlock the phone. Tap Set Password or Set PIN > enter a password or PIN > Done.
  7. Tap On/Off for each app that you want to download data to from this Exchange account.
  8. Tap Create > Done.
