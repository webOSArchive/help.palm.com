Première utilisation de l'application Contacts
  1. Ouvrir ! Contacts .
  2. Effectuez l'une des opérations suivantes :
    • Pour configurer un compte en ligne de manière à ce qu'il se synchronise : pour configurer un compte faisant partie de la liste, appuyez sur le type de compte et passez à l'étape 3. Pour configurer un type de compte ne faisant pas partie de la liste, appuyez sur Rechercher autres pour trouver une application, puis suivez les instructions à l'écran.
    • Pour utiliser les comptes déjà configurés dans une autre application : appuyez sur Aller. Vous ouvrez ainsi votre compte HP webOS et vos autres comptes en ligne, et vous pouvez commencer à créer des contacts dans ces comptes.
    • Pour utiliser l'application Contacts sans créer un compte en ligne : appuyez sur le nom de votre compte HP webOS. Vous pouvez alors commencer à créer des contacts dans votre compte HP webOS.
  3. Si vous configurez un compte Skype, appuyez sur Me connecter.
  4. Saisissez les informations de compte > appuyez sur Me connecter.
  5. Si vous configurez un compte Exchange, vous pourriez voir s'afficher un message vous indiquant que vous devez insérer un NIP ou un mot de passe pour l'appareil la prochaine fois que vous activez l'écran. Appuyez sur OK pour poursuivre la création du compte. La prochaine fois que vous activerez l'écran, appuyez sur Définir le NIP ou Définir le mot de passe. Définir un NIP ou un mot de passe > Terminé.
  6. Selon le compte, il est possible qu'on vous offre diverses options d'applications pouvant télécharger des données depuis ce compte. Pour chaque application, appuyez sur Activé ou Désactivé, puis sur Créer un compte.
