Parcourir les applications dans l'App Catalog HP webOS
  1. Ouvrir ! App Catalog .
  2. Sélectionnez votre pays si vous y êtes invité.
  3. Lisez les conditions générales si vous y êtes invités > Accepter.
  4. Effectuez l'une des opérations suivantes :
    • Pour consulter Pivot, un magazine mettant en vedette les nouvelles applications : appuyez sur Présenté(s) au bas de l'écran.
    • Pour effectuer une recherche par catégorie : appuyez sur Catégories au bas de l'écran > appuyez sur une catégorie. Pour afficher les sous-catégories d'une catégorie, appuyez sur Tout [nom de la catégorie] en haut de l'écran > appuyez sur une sous-catégorie. Si l'écran d'une sous-catégorie présente le nom de la sous-catégorie au haut de l'écran avec une flèche, appuyez sur ce nom pour afficher d'autres sous-catégories.
    • Pour rechercher parmi les applications que vous avez marquées d'un signet : appuyez sur Signets au bas de l'écran.
    • Pour une recherche par nom : appuyez sur Rechercher au bas de l'écran > saisissez un terme de recherche > . Une liste d'applications dont le nom ou la catégorie correspond à votre terme de recherche s'affiche.
    • Pour effectuer une recherche parmi les applications proposées, payantes, gratuites ou encore les nouveautés : après avoir lancé une recherche par catégorie, signet ou nom, appuyez sur Populaires, Payantes, Gratuites ou Nouveautés au haut de l'écran.
  5. Appuyez sur un nom d'application pour afficher les détails correspondants.
  6. À partir de l'écran affichant les détails de l'application sélectionnée, effectuez l'une des opérations suivantes :
    • Pour voir une vidéo de l'application : appuyez sur une miniature qui présente l'icône . Certaines applications n'ont pas de vidéo.
    • Pour voir une miniature d'image en plus grand, si cette option est disponible : appuyez sur une miniature qui ne présente pas l'icône .
    • Pour afficher d'autres miniatures : faites défiler les miniatures à l'écran vers la droite ou vers la gauche.
    • Pour marquer une application d'un signet : appuyez sur en haut de l'écran. Vous pouvez également marquer d'un signet une application dans la liste des résultats de recherche d'application en appuyant sur à droite du nom de l'application.
    • Pour lire des critiques : sous Critiques, appuyez sur Positive ou sur Négative. Faites défiler pour consulter plus de critiques positives ou négatives.
    • Pour partager des informations au sujet d'une application avec un ami : appuyez sur Partager, puis appuyez sur Courriel ou sur Message SMS.
    • Pour afficher d'autres applications de ce développeur particulier : appuyez sur Apps de développeur.
    • Pour accéder à la page de support produit du développeur : appuyez sur Support (si l'option est disponible).
    • Pour signaler un problème d'application : appuyez sur Signaler un problème. Appuyez sur Bogue > appuyez sur le type de problème que vous signalez > Appuyez sur Entrez votre problème ici > décrivez le problème > appuyez sur Envoyer.
