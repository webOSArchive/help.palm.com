Gestion des informations de positionnement transmises aux applications
  1. Ouvrir Services de positionnement .
  2. Dans Pour les applications, appuyez sur l'une des options suivantes :

    Toujours demander : les applications affichent une demande d'autorisation d'identifier votre position. Vous autorisez les applications à identifier votre position pour la session actuelle uniquement.

    Pos. automatique : permet aux applications d'identifier automatiquement votre position.

  3. (Facultatif) Appuyez sur Activé pour désactiver la Récupération des données contextuelles. L'option de Récupération des données contextuelles est disponible uniquement lorsque les services Google sont activés (procédure). Lorsque cette fonction est activée, des données anonymes de localisation sont recueillies à partir de votre appareil. Ces données sont utilisées pour améliorer la qualité des Services de positionnement.

L'application de Services de positionnement vous permet de contrôler si votre emplacement est transmis aux applications qui le demandent. Une application peut demander votre position actuelle pour vous fournir des informations pertinentes en fonction de votre localisation. Ainsi, Google Maps peut se servir de votre position actuelle pour vous fournir l'itinéraire vers une destination.
