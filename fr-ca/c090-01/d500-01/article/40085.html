Configuration de Quickoffice pour pouvoir accéder aux fichiers à partir d'un compte en ligne
  1. Ouvrir ! Quickoffice .
  2. Effectuez l'une des opérations suivantes :
    • Si une liste des comptes s'affiche : appuyez sur le type de compte.
    • Si le bouton Ajouter un compte s'affiche, appuyez sur Ajouter un compte > le type de compte.
    • Vous pouvez aussi ouvrir l'application Quickoffice, accéder à la liste des fichiers et appuyer sur Ajouter un compte au bas de l'écran.
  3. Saisissez le nom d'utilisateur ou l'adresse de courriel de votre compte ainsi que votre mot de passe > Me connecter.
  4. Sous Utiliser le compte avec, assurez-vous que l'option Documents est Activée.
  5. Appuyez sur Créer un compte.

      L'application Quickoffice vous permet de consulter les fichiers stockés sur votre appareil. Vous pouvez également configurer l'application Quickoffice de manière à pouvoir accéder à des fichiers dans des comptes en ligne tels que Google Docs, Dropbox ou Mobile Me. Vous devez posséder un compte en ligne pour pouvoir le configurer dans Quickoffice sur votre appareil.
