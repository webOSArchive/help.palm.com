Navigation dans une page Web
  • Déplacement vers l'arrière dans les pages précédemment consultées : appuyez sur .
  • Déplacement vers l'avant dans les pages précédemment consultées : appuyez sur .
  • Actualisation de la page actuelle : appuyez sur .
  • Défilement d'une page : faites la glisser vers le haut ou le bas de l'écran ou effectuez une pichenette vers le haut ou le bas de l'écran.
  • Affichage d'une page Web en mode paysage : faites pivoter l'appareil sur le côté.
  • Zoom avant ou arrière sur une page : effectuez le geste pincer/ouvrir pour agrandir les éléments affichés à l'écran. Effectuez le geste pincer/fermer pour réduire la taille des éléments affichés à l'écran. Appuyez deux fois sur l’écran pour effectuer un zoom avant ou arrière d’une taille définie.
  • Zoom avant sur une colonne de page Web : appuyez deux fois sur la page pour centrer une colonne et pour l'agrandir. Appuyez de nouveau deux fois pour faire un zoom arrière et afficher de nouveau toute la page.
  • Affichage d'une vue panoramique d'une page : appuyez sur la page et faites-la glisser tout en maintenant la pression.
  • Saisie d'informations dans un formulaire : appuyez sur le champ du formulaire > saisissez ou sélectionnez les informations > appuyez sur Entrée .
  • Partage d'une adresse Web sous forme d'un lien : appuyez sur > Partager le lien > rédigez le courriel.
