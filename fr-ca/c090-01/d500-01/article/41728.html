Copie de fichiers entre votre appareil et votre ordinateur
  1. Connectez l'appareil à l'ordinateur à l'aide du câble USB.
  2. Sur votre appareil, appuyez sur Lecteur USB.
  3. Windows : si des messages indiquent qu'un nouveau matériel a été détecté, fermez-les.
  4. Ouvrez Poste de travail (Windows XP), Ordinateur (Windows Vista) ou Finder (Mac), puis cliquez deux fois sur le lecteur représentant votre appareil.
  5. Vous pouvez déplacer les fichier par glisser-déposer dans les deux directions.
  6. Windows : lorsque vous êtes prêt à déconnecter l'appareil, cliquez avec le bouton droit de la souris sur le lecteur qui le représente, puis sélectionner Éjecter.
    Mac : à partir du bureau, faites glisser le lecteur représentant votre appareil jusqu'à la Corbeille. Le symbole de la corbeille se transforme en symbole Éjecter.
  7. Débranchez le câble USB de l'ordinateur lorsque l'écran Lecteur USB n'apparaît plus sur votre appareil.

Vous pouvez transférer tout fichier vers votre appareil, mais vous ne pourrez les ouvrir que si une application permettant de les lire y est installée.
