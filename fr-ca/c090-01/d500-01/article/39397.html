Entrée des paramètres de compte avancés

Ces préférences ne s'appliquent qu'au compte sélectionné.

  1. Ouvrir Courriel .
  2. Menu (en haut, à gauche) > Préférences et comptes > appuyez sur un compte.
  3. Pour régler l'un ou l'autre des paramètres répertoriés ci-dessous, appuyez sur l'option correspondante :
    • Nom du compte  : entrez le nom qui apparaît dans la liste de comptes.
    • Nom complet : entrez le nom que vous voulez voir apparaître dans le champ De des messages que vous envoyez.
    • Afficher l’icône : déterminez si une icône de notification doit s’afficher à l’arrivée d’un nouveau message.
    • Alerte : lorsque vous avez indiqué Oui à l'option Afficher l'icône, détermine si un avis sonore joue lors de la réception de nouveaux messages (procédure).
    • Signature : appuyez sur cette option pour créer une signature à ajouter aux messages sortants (procédure).
    • Adresse pour réponse : entrez l’adresse que les destinataires verront sur vos messages sortants, et à laquelle ils répondront, si elle est différente de l’adresse de courriel à partir de laquelle vous envoyez le message.
    • Afficher  : choisissez le nombre de jours de messages à récupérer sur le serveur.
    • Récupérer : indiquez la fréquence de synchronisation des courriels pour ce compte.
    • Synchroniser les courriels supprimés : comptes POP uniquement Déterminez si les messages doivent être supprimés du serveur lorsque vous les supprimez de votre appareil.
    • Synchroniser le serveur avec l'appareil : comptes POP uniquement Déterminez si les messages doivent être supprimés de votre appareil lorsque vous les supprimez du serveur.
    • Dossiers par défaut : comptes IMAP uniquement Indiquez le dossier dans lequel les messages que vous envoyez, enregistrez en tant que brouillons ou supprimez sont stockés.
    • Modifier config. ouv. session : procédure.
    • Supprimer de l'application : procédure.
  4. Appuyez sur Terminé > Terminé.
