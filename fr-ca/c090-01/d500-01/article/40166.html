Connexion de votre appareil à un téléphone intelligent HP webOS

 Avant de commencer

Le téléphone intelligent webOS avec lequel vous établissez une connexion doit être détectable. Consultez la documentation fournie avec votre téléphone intelligent pour obtenir des informations sur la façon de le rendre détectable.

BYB Close
  1. Ouvrez Téléphone  > Connecter le téléphone.
  2. Si la fonction Bluetooth est désactivée, appuyez sur Non pour l'activer.
  3. Effectuez l'une des opérations suivantes :
    • Si le téléphone intelligent auquel vous souhaitez vous connecter figure dans la liste Appareils, appuyez dessus.
    • Si le téléphone intelligent que vous souhaitez connecter ne figure pas dans la liste Appareils, appuyez sur Ajouter un appareil > Type > Téléphone webOS. Lorsque le téléphone intelligent auquel vous souhaitez vous connecter est trouvé, appuyez dessus.
  4. Sur le téléphone intelligent, appuyez sur Oui, autoriser pour réaliser le couplage.
  5. Sur le téléphone intelligent, si vous êtes invité à le faire, appuyez sur Autoriser pour autoriser le partage des messages entre le téléphone intelligent et votre appareil. Cette fonction n'est prise en charge que sur certains téléphones intelligents HP webOS.
  6. (Facultatif) Appuyez sur en regard du nom du téléphone intelligent et configurez les options d'utilisation de votre appareil avec le téléphone intelligent :
    • Connexion automatique : L'appareil et le téléphone intelligent se connectent chaque fois qu'ils se trouvent à proximité l'un de l'autre.
    • Recevoir des appels : gérez vos appels entrants et sortants directement sur votre appareil.
    • Recevoir des SMS : envoyez et recevez des SMS et répondez-y directement sur votre appareil. Cette fonction n'est prise en charge que sur certains téléphones intelligents HP webOS.
REMARQUE : vous ne pouvez pas envoyer ou recevoir de MMS sur votre appareil. Si vous recevez un MMS sur votre téléphone intelligent connecté, un avis apparaît sur votre appareil. Vous devez toutefois visionner le message sur le téléphone intelligent.
