Ouverture d’un fichier
  1. Ouvrir ! Quickoffice .
  2. La première fois que vous ouvrez l'application Quickoffice, vous avez la possibilité de configurer l'application pour travailler avec des fichiers à partir d'un compte en ligne (procédure). Si vous souhaitez uniquement travailler avec des fichiers sur votre appareil ou avec un compte déjà configuré, appuyez sur OK.
  3. Appuyez sur un compte pour afficher les fichiers de ce compte. Appuyez sur Mon TouchPad pour afficher les fichiers enregistrés localement sur votre appareil.
  4. Appuyez sur Afficher > Documents, PDF, PowerPoint ou Feuilles de calcul pour voir une liste de ces types de documents, ou appuyez sur Tous les fichiers. Si vous appuyez sur un fichier PDF, celui-ci s'ouvre dans l'application Adobe Reader de votre appareil.
  5. Appuyez sur le fichier.
  6. Pour retourner à la liste de fichiers, appuyez sur Documents dans le coin supérieur droit de l'écran.

Vous pouvez afficher un fichier dans un compte en ligne que vous configurez (procédure), un fichier que vous copiez de votre ordinateur (procédure) ou un fichier que vous recevez en pièce jointe à un courriel (procédure). Vous pouvez afficher des fichiers aux formats DOC, DOCX, TXT, XLS, XLSX, PPT et PPTX.
