Recherche d'informations dans une application de votre appareil
  1. Ouvrez la vue Cartes (procédure) > Tapez maintenant.
  2. Commencez la saisie des informations que vous recherchez. En fonction de l'application dans laquelle vous souhaitez effectuer la recherche, il peut s'agir de l'objet d'un courriel, de l'artiste qui a créé une chanson, du nom d'un signet ou d'un autre terme.
  3. (Facultatif) Pour filtrer les résultats de la recherche afin de n'afficher les informations que sur votre appareil, appuyez sur l'onglet Contenu.
  4. En fonction de l'application dans laquelle vous souhaitez effectuer une recherche, procédez comme suit.

    Artistes : Votre appareil indique combien de chansons présentent le nom indiqué comme terme de recherche. Appuyez sur Artistes pour afficher la liste de chansons. Appuyez sur une chanson pour l'écouter.

    Événements de calendrier : Votre appareil indique combien d'événements de calendrier contiennent votre terme de recherche dans le nom. Appuyez sur Événements de calendrier pour afficher la liste d'événements. Appuyez sur un événement pour l'ouvrir.

    Signets et historique : Votre appareil affiche le nombre de sites Web dans votre liste de signets et dans votre historique de navigation dont le nom contient le terme recherché. Appuyez sur Signets et historique pour afficher la liste des sites. Appuyez sur une entrée pour accéder au site.

    Courriel : Votre appareil affiche le nombre de messages qui contiennent le terme recherché dans le nom de l'expéditeur et l'objet. Appuyez sur Courriel pour afficher la liste de messages concordants. Appuyez sur un message pour l'afficher.

Si le terme recherché ne vous donne aucun résultat pour une certaine application, peut-être devrez-vous activer une préférence pour que cette application soit incluse dans les recherches Tapez maintenant (procédure).
