Personnalisation de Tapez maintenant
  1. Ouvrir ! Tapez maintenant .
  2. Sélectionnez l'une des options suivantes :

    Applications : Par défaut, la fonction Tapez maintenant trouve toute application installée sur votre appareil dont le nom correspond au terme saisi dans la recherche. Si vous ne voulez pas inclure les applications dans les recherches Tapez maintenant, appuyez sur Activé pour remplacer la valeur Activé par la valeur Désactivé.

    Contacts : Par défaut, la fonction Tapez maintenant trouve tout contact figurant sur votre appareil dont le nom, le numéro de téléphone ou l'adresse de courriel correspond au terme de recherche saisi. Si vous ne voulez pas inclure les contacts dans les recherches Tapez maintenant, appuyez sur Activé pour remplacer la valeur Activé par la valeur Désactivé. Si vous disposez d'un compte Exchange configuré sur votre appareil, cochez la case Lancer la recherche globale d'adresses automatiquement pour que votre recherche de contacts comprenne toujours la liste d'adresses globale (GAL) de votre entreprise. Lorsque cette case n'est pas cochée, les résultats de recherche Tapez maintenant proposent une option permettant d'effectuer une recherche globale d'adresses.

    Moteur de recherche par défaut : moteur de recherche utilisé par votre appareil pour effectuer une recherche sur le Web lorsque vous saisissez un terme à rechercher. Si vous ne voulez pas que la fonction Tapez maintenant affiche un moteur de recherche par défaut, appuyez sur Activé pour remplacer la valeur Activé par la valeur Désactivé. Pour changer le moteur de recherche, appuyez sur le moteur affiché, puis appuyez sur le moteur de recherche de votre choix. Cette liste contient deux types de moteurs de recherche : moteurs effectuant des recherches sur le Web, tels que Google, et moteurs effectuant des recherches sur des sites Web particuliers, tels que Wikipedia.

    Contenu : Par défaut, la fonction Tapez maintenant trouve des entrées de certaines applications installées sur votre appareil correspondant au terme saisi dans la recherche. Si vous ne voulez pas inclure les entrées d'applications dans les recherches Tapez maintenant, appuyez sur Activé pour remplacer la valeur Activé par la valeur Désactivé. Pour personnaliser les entrées de quelles applications doivent être considérées dans les recherches, cochez ou décochez les cases situées à gauche du nom de chaque application.

    Rechercher en utilisant : Par défaut, la fonction Tapez maintenant vous permet d'effectuer des recherches sur le Web à l'aide d'autres moteurs de recherche ou d'effectuer des recherches à l'aide d'autres applications installées sur votre appareil. Si vous ne voulez pas que ces autres moteurs de recherche et applications s'affichent dans les recherches Tapez maintenant, appuyez sur Activé pour remplacer la valeur Activé par Désactivé. Pour personnaliser les autres moteurs de recherche Web et applications disponibles pour les recherches Tapez maintenant, cochez ou décochez les cases situées à gauche de chaque entrée. Les moteurs de recherche sont désignés par Recherche Web; tous les moteurs de recherche répertoriés peuvent être utilisés en tant que moteur de recherche par défaut. Les applications incluses dans vos recherches sont désignées par Recherche d'applications.

    Actions rapides : Par défaut, Tapez maintenant vous permet de créer un élément ou de démarrer une action en fonction du terme recherché. Si vous ne voulez pas inclure ces Actions rapides dans les recherches Tapez maintenant, appuyez sur Activé pour remplacer la valeur Activé par la valeur Désactivé. Pour personnaliser les actions rapides incluses dans les recherches Tapez maintenant, cochez ou décochez les cases situées à gauche de chaque entrée.

    Autres : grâce à cette option, vous pouvez rechercher dans l'App Catalog des applications et des services tiers que vous pouvez utiliser pour effectuer l'une ou l'autre des actions suivantes à l'aide des fonctions Tapez maintenant : recherche sur le Web, recherche parmi les application ou création d'un nouvel élément (Actions rapides). Par exemple, vous pouvez utiliser Rechercher autres pour télécharger un nouveau moteur de recherche sur le Web. Pour utiliser cette fonction, appuyez sur Rechercher autres, puis téléchargez une application à partir de la liste qui s'affiche dans App Catalog (procédure).
