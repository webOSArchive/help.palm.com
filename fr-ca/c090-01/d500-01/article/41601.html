Première utilisation de l'application Facebook Tablet

Si aucun compte Facebook n'est configuré sur votre appareil, vous devez en configurer un lorsque vous ouvrez l'application Facebook Tablet pour la première fois.

  1. Ouvrir ! Facebook Tablet .
  2. (Facultatif) Procédez de l'une des façons suivantes :
    • Pour changer la fréquence à laquelle l'application Facebook Tablet vérifie les nouvelles notifications : appuyez sur Fréquence > appuyez sur une fréquence.
    • Pour activer ou désactiver les avis HP WebOS : appuyez sur Activé ou Désactivé. Quand ce paramètre est activé, les nouvelles notifications Facebook apparaissent sur le tableau de bord.
  3. Appuyez sur Ajouter un compte > saisissez les informations du compte > Me connecter.
  4. Vous êtes invité à sélectionner les applications téléchargeant des données depuis ce compte. Pour chaque application, appuyez sur Oui ou sur Non > Créer un compte.
  5. Appuyez sur Terminé.
