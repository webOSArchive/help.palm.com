Utilisation du menu d'application
  1. Dans une application, effectuez l'une des opérations suivantes :
    • Faites glisser depuis le coin supérieur gauche de l'appareil vers l'écran.
    • En haut, à gauche de l'écran, appuyez sur le nom de l'application.
  2. Pour ouvrir un élément de menu, appuyez sur cet élément.

Lorsqu'une flèche pointant vers la droite apparaît près d'un élément de menu ,vous pouvez appuyer sur l'élément pour afficher son sous-menu.

Lorsqu'une flèche pointant vers le bas apparaît au bas du menu, cela signifie que d'autres éléments sont disponibles. Pour afficher d'autres éléments, faites défiler le menu vers le bas.

Dans certaines applications, telles que Contacts, vous pouvez supprimer les informations à l'aide d'une option de menu.
