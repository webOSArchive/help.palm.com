Insertion des coordonnées d'un contact dans Contacts

Si vous souhaitez intégrer les coordonnées d'un contact à l'application Contacts, vous avez le choix entre plusieurs procédures :

  • Se connecter à un carnet d'adresses en ligne : si vous disposez d'un carnet d'adresses dans un compte en ligne fonctionnant avec la fonction HP Synergy (Google ou Microsoft Exchange, par exemple), vous pouvez configurer l'application Contacts de votre appareil pour qu'elle se synchronise et affiche les contacts que vous enregistrez dans votre carnet d'adresses en ligne. Une fois que vous avez configuré la connexion au carnet d'adresses en ligne, les contacts que vous entrez en ligne apparaissent automatiquement dans vos Contacts, et les contacts que vous entrez sur votre appareil se synchronisent au compte en ligne, à condition que ce dernier permette l'écriture à partir de votre appareil.

    Vous pouvez configurer vos comptes contacts à partir de l'application Comptes (procédure) ou de l'application Contacts.

    La fonction Synergy offre un autre avantage : durant le même processus de configuration d'un compte contacts, vous configurez également ce même compte pour toutes les autres applications qui peuvent afficher des données provenant de ce compte.

    Par exemple, supposons que vous voulez configurer votre compte contacts de Google sur votre appareil. La dernière étape de la configuration consiste à sélectionner toutes les autres applications pour lesquelles vous souhaitez recevoir des données de votre compte Google (outre l'application Contacts, vous pouvez inclure Messagerie, Calendrier, Courriel et Documents). Lorsque la configuration est terminée, votre appareil commence à télécharger les contacts et toutes les autres données de toutes les autres applications dont vous avez autorisé la synchronisation.

  • Utiliser une solution de synchronisation tierce : si vous souhaitez utiliser une application de bureau pour stocker vos contacts, vous pouvez être en mesure d'utiliser une solution tierce (vendue séparément) pour synchroniser vos contacts directement avec votre ordinateur. Visitez hpwebos.com/sync-solutions pour voir si des solutions tierces sont disponibles pour votre appareil. Si aucune solution n'est disponible pour votre appareil, vérifiez auprès des développeurs de logiciels de synchronisation pour d'autres appareils webOS s'ils mettent au point une solution pour votre appareil.
  • Entrez les contacts directement dans Contacts (procédure). Assignez le contact à l'un de vos carnets d'adresses en ligne ou créez le contact dans votre compte HP webOS. Les informations figurant dans votre compte HP webOS ne sont pas synchronisées, mais elles sont incluses à la sauvegarde régulière des informations HP webOS.
  • Transfert de contacts depuis un compte HP webOS existant : lors de la configuration de votre appareil, vous pouvez vous connecter à un compte HP webOS existant dont vous vous servez déjà avec un téléphone intelligent webOS. Si vous décidez de vous connecter à un compte existant, vous pouvez effectuer un transfert unique des données stockées dans le compte HP webOS existant vers votre nouvel appareil, notamment des contacts.
  • Télécharger les informations des vCard dans les Contacts : les vCard représentent un type de fichiers conçu spécialement pour contenir les informations des contacts. Une vCard peut contenir une ou plusieurs entrées de contacts. Une vCard peut être jointe à un courriel, et vous pouvez télécharger les informations d'une vCard dans votre application Contacts.
