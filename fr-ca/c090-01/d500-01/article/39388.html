Configuration du compte de courriel lorsque la configuration automatique échoue

Procédez comme suit uniquement si la configuration automatique de votre compte de courriel (procédure) échoue.

Avant de commencer

Communiquez avec votre fournisseur de courriel ou votre administrateur système pour obtenir les renseignements suivants :

  • Type de compte (POP ou IMAP)
  • Noms des serveurs de courrier entrant et sortant
  • Nom d’utilisateur et mot de passe sur le serveur de courrier entrant
  • Numéros de ports du serveur pour le courrier entrant et sortant
  • Tout renseignement requis pour SSL, pour le courrier entrant et/ou sortant
  • Tout renseignement requis en matière d’authentification (ESMTP), pour le courrier sortant
  • Dossier racine (comptes IMAP uniquement)

BYB Close

  1. Si vous avez déjà commencé la configuration du compte et que le message « Impossible d'ouvrir une session » apparaît, appuyez sur Configuration manuelle et passez à l'étape 6. Dans le cas contraire, commencez à l'étape 2.
  2. Ouvrir Courriel .
  3. Effectuez l'une des opérations suivantes :
    • Si vous n'avez pas configuré de compte dans une autre application qui utilise la fonction Synergy, appuyez sur Compte de courriel.
    • Si vous avez déjà configuré un compte qui utilise la fonction Synergy et souhaitez en configurer un autre, appuyez sur Ajouter un compte > Compte de courriel.
    • À tout autre moment : dans Courriel, Menu (en haut, à gauche) > Préférences et comptes > Ajouter un compte > Compte de courriel.
  4. Saisissez votre adresse de courriel et votre mot de passe > Configuration manuelle.
  5. Appuyez sur le champ Type de compte > appuyez sur votre type de compte.
  6. Saisissez les informations du serveur de messagerie entrante et sortante, en fonction des informations du serveur que vous avez obtenues auprès de votre administrateur système (voir Avant de commencer, ci-dessus).
  7. Appuyez sur Me connecter.
  8. Appuyez sur Oui/Non pour les applications pour lesquelles vous souhaitez télécharger des données à partir de ce compte.
  9. Appuyez sur Créer un compte > Aller.
