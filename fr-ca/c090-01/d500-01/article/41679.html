Personnalisation de l'application Facebook Tablet
  1. Ouvrez ! Facebook Tablet  > Menu (en haut, à gauche) > Préférences et comptes.
  2. Définissez les options suivantes :
    • Fréquence : définit l'intervalle de vérification des notifications. L'intervalle de vérification par défaut est de 20 minutes. Pour modifier cet intervalle, appuyez sur Toutes les 20 minutes > appuyez sur un nouvel intervalle.
    • Notifications HP webOS : appuyez pour activer ou désactiver cette fonction. Lorsque cette fonction est activée, les nouvelles alertes s'affichent sur le tableau de bord.
    • Comptes : Appuyez sur un compte pour modifier les paramètres de connexion ou pour retirer le compte. Vous ne devriez changer ces paramètres que si vous avez modifié ces informations dans votre compte Facebook en ligne. En effet, les informations saisies ici doivent correspondre à celles du compte en ligne.
    • Ajouter un compte : ajoutez d'autres comptes Facebook. Par défaut, le dernier compte ajouté devient votre compte principal. Pour changer le compte principal, appuyez sur Compte principal > appuyez sur un autre compte. Facebook n'affiche les informations que sur votre compte principal.
  3. Appuyez sur Terminé.
