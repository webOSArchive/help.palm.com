Partage d'une page Web à l'aide de Toucher-partager
  1. Allez sur le Web > ouvrez la page Web que vous voulez envoyer vers votre téléphone intelligent.
  2. Appuyez le téléphone intelligent sur le Bouton central de l'appareil.
  3. Si le Bluetooth n'a pas été activé sur un ou sur les deux appareils, appuyez sur Activer Bluetooth sur un ou sur les deux appareils > avec votre téléphone intelligent, touchez de nouveau le Bouton central de votre appareil pour terminer le couplage.

    La page Web s'ouvre en plein écran sur le téléphone intelligent.

Vous pouvez aussi partager une page Web ouverte sur votre téléphone intelligent avec votre appareil. Cela est utile lorsque vous voulez voir la page Web sur un plus grand écran. Répétez les étapes 1 et 2 indiquées ci-dessus, sauf celle consistant à ouvrir la page Web sur votre téléphone intelligent, puis utilisez la fonction Toucher-partager pour l'envoyer vers votre appareil.
