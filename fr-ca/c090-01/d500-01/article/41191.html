Impression d'images
  1. Ouvrir ! Photos et vidéos .
  2. Appuyez sur l'album contenant les images.
  3. Procédez de l’une des façons suivantes :
    • Pour imprimer une seule image : appuyez sur la miniature de l'image.
    • Pour imprimer plusieurs images : appuyez sur > Tout sélectionner pour sélectionner toutes les images d'un album. Pour supprimer un fichier de la sélection, appuyez dessus.
  4. Appuyez sur . Appuyez sur l'écran si n'est pas disponible.

    Les imprimantes disponibles sur votre réseau sans fil actuel sont affichées.

  5. Effectuez l'une des opérations suivantes :
    • Si c'est la première fois que vous utilisez une imprimante sur ce réseau : si une liste d'imprimantes s'affiche, appuyez sur l'une des imprimantes indiquées.
    • Si vous avez déjà utilisé une imprimante sur ce réseau : la dernière imprimante à voir été utilisée sera affichée et sélectionnée. Si vous voulez sélectionner une autre imprimante, faites défiler vers le bas et appuyez sur Sélectionner une autre imprimante > appuyez sur le nom d'une imprimante.
    • Pour configurer manuellement une imprimante ou pour la configurer sur un réseau d'entreprise : pour ajouter une imprimante, faites défiler vers le bas et appuyez sur Ajouter une imprimante. Appuyez sur Entrer une adresse IP > saisissez l'adresse IP de l'imprimante. (Facultatif) Appuyez sur Nommer cette imprimante > donnez un nom à l'imprimante afin de l'identifier. Appuyez sur Terminé.
  6. (Facultatif) Selon l'imprimante utilisée, vous pouvez configurer les attributs suivants de l'imprimante :

    Nombre de copies : Appuyez sur 1 puis sur le nombre de copies à imprimer.

    Taille : appuyez sur la taille actuelle > appuyez pour sélectionner une autre taille. Les tailles affichées sont celles qui sont disponibles pour votre imprimante.

    Impression couleur : Appuyez sur Activé pour désactiver l'impression couleur.

  7. Appuyez sur Imprimer.

    Pour annuler une tâche d'impression, appuyez sur Annuler lorsque le fichier est en cours de préparation de l'impression ou pendant l'impression, appuyez sur dans le haut de la page > appuyez sur l'avis qui apparaît > appuyez sur à droite de la tâche d'impression dans la File d’attente à l’impression.
