Configuration d'un compte en ligne dans l'application Calendrier
  1. Ouvrir ! Calendrier .
  2. Effectuez l'une des opérations suivantes :
    • Configuration d'un compte en ligne lors de la première utilisation de l'application Calendrier (procédure).
    • Pour configurer un compte en ligne à tout autre moment : Menu (en haut, à gauche) > Préférences et comptes > Ajouter un compte > appuyez sur un type de compte > passez à l'étape 3. Pour configurer un type de compte ne faisant pas partie de la liste, appuyez sur Rechercher autres pour trouver une application, puis suivez les instructions à l'écran.
  3. Saisissez les informations de compte > appuyez sur Me connecter.
  4. Si vous configurez un compte Exchange, vous pourriez voir s'afficher un message vous indiquant que vous devez insérer un NIP ou un mot de passe pour l'appareil la prochaine fois que vous activez l'écran. Appuyez sur OK pour poursuivre la création du compte. La prochaine fois que vous activerez l'écran, appuyez sur Définir le NIP ou Définir le mot de passe. Définir un NIP ou un mot de passe > Terminé.
  5. Selon le compte, il est possible qu'on vous offre diverses options d'applications pouvant télécharger des données depuis ce compte. Pour chaque application, appuyez sur Activé ou Désactivé, puis sur Créer un compte.
