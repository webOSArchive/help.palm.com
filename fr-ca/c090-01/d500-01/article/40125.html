Configuration du mode de numérotation internationale
  1. Ouvrez Téléphone  > Menu (en haut, à gauche) > Préférences et comptes.
  2. Dans Utiliser, appuyez sur l'une des options suivantes :
    • Pour utiliser un téléphone intelligent connecté pour les appels sortants : appuyez sur Bluetooth.
    • Pour utiliser votre compte Skype pour les appels sortants, appuyez sur Skype.
    • Pour qu'un message s'affiche vous demandant le mode d'utilisation souhaité pour chaque appel : appuyez sur Toujours demander.

Si vous possédez un compte Skype, vous pouvez définir si votre appareil compose les numéros par l'intermédiaire d'un téléphone intelligent connecté ou de votre compte Skype. Vous pouvez également choisir qu'on vous demande quel compte vous préférez utiliser chaque fois que vous effectuez un appel.
