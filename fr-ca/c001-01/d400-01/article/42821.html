Clavier

  1. Option : appuyez sur cette touche pour entrer les chiffres, la ponctuation et les symboles qui figurent au-dessus des lettres sur les touches, ou pour déplacer le curseur.
  2. Maj : appuyez deux fois sur cette touche pour verrouiller les majuscules.
  3. Espace : appuyez sur cette touche pour entrer un espace ou, dans Appareil photo, pour prendre une photo.
  4. Retour arrière
  5. Entrée : appuyez sur cette touche pour insérer un retour chariot (par exemple : dans un mémo ou un courriel que vous composez) ou pour accepter les renseignements que vous avez saisis dans un champ.
  6. Sym : appuyez sur cette touche pour entrer des symboles et des caractères accentués qui n'apparaissent pas sur les touches.
