Configuration d'un compte de messagerie instantanée
  1. Ouvrir Messagerie .
  2. Effectuez l'une des opérations suivantes :
  • La première fois que vous ouvrez l'application Messagerie : appuyez sur Ajouter un compte.
  • À tout autre moment : Menu (en haut, à gauche) > Préférences et comptes > Ajouter un compte MI.

3. Procédez de l’une des façons suivantes :
  • Pour configurer une messagerie avec l'un des comptes de la liste : appuyez sur le type de compte.
  • Pour configurer un type de compte ne faisant pas partie de la liste, appuyez sur Rechercher autres et sur une application, puis suivez les instructions à l'écran pour ajouter le compte.

4. Entrez votre nom d'utilisateur et votre mot de passe > Me connecter.

5. Appuyez sur Oui/Non pour les applications pour lesquelles vous souhaitez télécharger des données à partir de ce compte.

6. Appuyez sur Créer > Terminé.

Votre téléphone intelligent prend en charge les messageries de la plupart des services de messagerie instantanée en ligne, tels que GoogleTalk et AIM. Si vous disposez déjà d'un compte de messagerie instantanée, vous pouvez vous y connecter à partir de l'application Messagerie ou Comptes (procédure).

La fonction HP Synergy offre un autre avantage : durant le même processus de configuration d'un compte de messagerie, vous configurez également ce même compte pour toutes les autres applications qui peuvent afficher des données provenant de ce compte.

Par exemple, supposons que vous voulez configurer votre compte Google Talk de Google sur votre téléphone intelligent. La dernière étape de cette configuration consiste à sélectionner toutes les autres applications qui, à votre choix, recueilleront des données de votre compte Google (en plus de l'application Messagerie, vous pouvez choisir les applications Contacts, Calendrier et Courriel). Lorsque la configuration est terminée, votre téléphone intelligent commence à télécharger les messages et toutes les autres données de toutes les autres applications dont vous avez autorisé la synchronisation.
