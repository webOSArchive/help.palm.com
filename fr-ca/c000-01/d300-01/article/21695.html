Personnalisation de Tapez maintenant
  1. En vue Cartes ou dans l'écran de lancement, commencez à saisir du texte pour afficher l'écran des résultats de recherche.
  2. Faites défiler jusqu'au bas de l'écran des résultats de recherche et appuyez sur Préférences.
  3. Sélectionnez l'une des options suivantes :

    Moteur de recherche par défaut : moteur de recherche utilisé par votre téléphone intelligent pour effectuer une recherche sur le Web lorsque vous saisissez un terme à rechercher. Pour changer le moteur de recherche, appuyez sur le moteur affiché, puis appuyez sur le moteur de recherche de votre choix. Cette liste contient deux types de moteurs de recherche : moteurs effectuant des recherches sur le Web, tels que Google, et moteurs effectuant des recherches sur des sites Web particuliers, tels que Wikipedia.

    Applications : si vous incluez des applications dans vos paramètres de recherche, tous les éléments correspondants compris dans ces applications apparaissent dans les résultats de recherche. Pour ajouter une application à vos résultats de recherche, appuyez sur Ajouter des recherches d'application, puis sur le nom de l'application.

    Lancement et recherche : cette liste affiche d'autres moteurs de recherche et les applications dans lesquels votre téléphone intelligent effectue une recherche lorsque vous saisissez un terme de recherche. Les moteurs de recherche sont désignés par Recherche Web; tous les moteurs de recherche répertoriés peuvent être utilisés en tant que moteur de recherche par défaut. Les applications incluses dans vos recherches sont désignées par Recherche d'applications. Pour inclure d'autres moteurs de recherche ou applications dans vos recherches, appuyez sur Ajouter d'autres recherches, puis sur l'élément souhaité.

    Actions rapides : permet d'afficher les types d'éléments que vous pouvez créer à partir des résultats de votre recherche, tel qu'un courriel ou une note. Pour ajouter un nouveau type d'élément, appuyez sur Ajouter des actions rapides, puis sur l'élément souhaité.

    Autres : grâce à cette option, vous pouvez rechercher dans l'App Catalog des applications et des services tiers que vous pouvez utiliser pour effectuer l'une ou l'autre des actions suivantes à l'aide des fonctions Tapez maintenant : recherche sur le Web, recherche parmi les applications ou création d'un nouvel élément (Actions rapides). Par exemple, vous pouvez utiliser Rechercher autres pour télécharger un nouveau moteur de recherche sur le Web. Pour utiliser cette fonction, appuyez sur Autres, puis téléchargez une application à partir de la liste qui s'affiche dans l'App Catalog (procédure).

  4. Pour changer l'ordre des éléments d'une liste : appuyez sur l'élément et maintenez-le enfoncé, attendez le marqueur visuel, puis faites glisser l'élément à un autre endroit.
