Comment insérer des événements dans Calendrier?

Plusieurs options s'offrent à vous si vous souhaitez que des événements s'affichent dans Calendrier :

  • Se connecter à un calendrier en ligne : si vous disposez d'un calendrier dans un compte en ligne fonctionnant avec la fonction HP Synergy (Google ou Microsoft Exchange, par exemple), vous pouvez configurer l'application Calendrier de votre téléphone intelligent pour qu'elle se synchronise et affiche les événements que vous enregistrez dans le calendrier en ligne. Après avoir configuré la connexion au calendrier en ligne, les événements que vous saisissez en ligne apparaissent automatiquement dans le Calendrier et les événements que vous entrez sur votre téléphone se synchronisent avec le compte en ligne, à condition que ce dernier permette l'écriture à partir du téléphone intelligent.

    Vous pouvez configurer vos comptes calendrier à partir de l'application Comptes (procédure) ou de l'application Calendrier.

    La fonction Synergy offre un autre avantage : durant le même processus de configuration d'un compte calendrier, vous configurez également ce même compte pour toutes les autres applications qui peuvent afficher des données provenant de ce compte.

    Par exemple, supposons que vous voulez configurer votre compte calendrier de Google sur votre téléphone intelligent. La dernière étape de cette configuration consiste à sélectionner toutes les autres applications qui, à votre choix, recueilleront des données de votre compte Google (en plus de l'application Calendrier, vous pouvez choisir les applications Contacts, Messagerie et Courriel). Lorsque la configuration est terminée, votre téléphone intelligent commence à télécharger les événements de calendrier et toutes les autres données de toutes les autres applications dont vous avez autorisé la synchronisation.

  • Saisie directe d'un événement dans Calendrier (procédure) : vous pouvez assigner l'événement à l'un de vos calendriers en ligne. Il se synchronise alors avec le calendrier en ligne. Vous pouvez également créer l'événement dans votre compte de profil Palm. Dans ce cas, l'événement s'affiche uniquement sur le téléphone. Les événements du profil Palm sont inclus dans la sauvegarde quotidienne de vos informations de profil Palm, ce qui permet de les restaurer s'ils sont involontairement effacés.

  • Utilisez une solution de synchronisation tierce : si vous souhaitez utiliser une application de bureau pour stocker les événements de votre calendrier, vous pouvez utiliser une solution tierce (vendue séparément) pour synchroniser vos événements de calendrier directement avec votre ordinateur (voir palm.com/sync-solutions).

  • Exportation d'événements à l'aide de l'Assistant de transfert de données (ATD) : lorsque vous utilisez l'ATD pour exporter des événements de calendrier depuis un logiciel de bureau (procédure), vous devez sélectionner le compte auquel vous souhaitez assigner ces événements. Recherchez ces événements dans le compte sélectionné.
