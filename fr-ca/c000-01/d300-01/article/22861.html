Définition des préférences de courriel

Cette configuration s'applique à tous vos comptes de courrier électronique.

  1. Ouvrez Courriel  > Menu (en haut, à gauche) > Préférences et comptes.

  2. Configurez les options de votre choix :

    • Dossiers intelligents : indiquez si les messages de toutes les boîtes de réception et/ou tous les messages marqués d’un drapeau doivent être inclus dans les favoris figurant en haut de la vue de la liste des comptes.
    • Confirmation de la suppression : définissez si vous souhaitez être invité à confirmer la suppression des courriels.
    • Comptes : appuyez sur un nom de compte pour entrer des paramètres avancés (procédure).
    • Ajouter un compte : procédure.
    • Compte par défaut : appuyez sur le compte, puis appuyez sur un autre dans la liste. Le compte que vous sélectionnez est utilisé pour envoyer un message lorsque vous créez un nouveau message dans la liste des comptes ou dans un dossier fusionné comme celui contenant toutes les boîtes de réception ou tous les messages marqués d'un drapeau.
