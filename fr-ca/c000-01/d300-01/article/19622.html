Configuration d'un compte de facturation
  1. Ouvrir App Catalog .
  2. Effectuez l'une des opérations suivantes :
    • Appuyez sur une application que vous souhaitez acheter > Continuer.
    • Appuyez sur Menu (en haut, à gauche) > Préférences et comptes > Ajouter un compte.
  3. Entrez le mot de passe associé à votre profil Palm > Continuer.

    4. Si vous entrez trois fois un mot de passe de profil Palm erroné, un message s'affiche et vous indique comment communiquer avec le Service à la Clientèle pour obtenir de l'aide.


  4. Appuyez sur un type de carte de crédit > saisissez les données de votre carte de crédit ainsi que l'adresse de facturation > Soumettre.
    5. IMPORTANTIMPORTANT :

    Les détails de votre carte de crédit ne sont pas enregistrés sur votre téléphone. Ils sont chiffrés et enregistrés dans votre profil Palm.

    Important Close
  5. Appuyez sur OK pour vérifier votre adresse de courriel.

    6. Les factures sont envoyées à cette adresse. Appuyez sur Changer cette adresse si vous souhaitez changer l'adresse de courriel.


  6. Dans Le mot de passe est exigé, sélectionnez l'une des options suivantes :
    • Une fois toutes les 4 heures : vous devez entrer votre mot de passe lors de votre premier achat. Si vous faites un nouvel achat au cours des quatre heures qui suivent, vous n'avez pas besoin d'entrer votre mot de passe. Au bout de quatre heures, vous devrez entrer à nouveau votre mot de passe, ce qui le validera pour une nouvelle période de quatre heures.
    • Lors de chaque achat : vous devrez entrer votre mot de passe à chaque achat, quel que soit le temps écoulé depuis le précédent.
