Indiquer où vous vous trouvez
  1. Ouvrez Facebook  > .
  2. Si vous y êtes invité, appuyez sur Autoriser une fois pour activer les Services de positionnement.
  3. Si vous y êtes invité, appuyez sur Accepter.
  4. Effectuez l'une des opérations suivantes :
    • Pour créer un nouveau lieu : appuyez sur > entrez un nom pour le lieu > (facultatif) saisissez une description > Créer un lieu.
    • Pour sélectionner un lieu existant : saisissez le lieu où vous voulez indiquer que vous vous trouvez, ou appuyez sur un lieu dans la liste des lieux à proximité.
  5. Appuyez sur Je suis là.
  6. Procédez de l'une des façons suivantes :
    • Pour indiquer à vos amis ce que vous faites : appuyez sur Qu'êtes-vous en train de faire? > saisissez un message.
    • Pour indiquer à vos amis quelles personnes se trouvent avec vous : appuyez sur Identifier des amis avec vous > appuyez sur les amis > Terminé.
  7. Appuyez sur Je suis là.
  8. Procédez de l'une des façons suivantes :
    • Pour visualiser quelles autres personnes s'y trouvent : appuyez sur Ici maintenant.
    • Pour afficher les mises à jour de statut pour les amis qui se trouvent à proximité : appuyez sur Activité et regardez dans Activité récente.
    • Pour afficher des informations sur le lieu : appuyez sur Infos.
    • Pour afficher votre emplacement dans Google Maps : appuyez sur la carte.

Vous pouvez utiliser l'application Lieux de Facebook pour indiquer à vos amis où vous vous trouvez, avec qui vous êtes, entrer en contact avec des amis qui sont à proximité et plus encore.
