Utiliser le mode Exhibition
  • Lorsque vous lancez le mode Exhibition, la dernière application utilisée s'affiche. Pour passer à une autre application, appuyez sur le coin supérieur gauche de l'écran > appuyez sur l'application souhaitée.
  • Lorsque le mode Exhibition est lancé, les notifications apparaissent comme en temps normal, et le fait d'appuyer sur une notification donne le même résultat qu'en temps normal.
  • Lorsque votre téléphone intelligent est placé sur une station de charge HP Touchstone, le mode Exposition s'active automatiquement lorsque le téléphone intelligent est resté inactif pendant la période définie dans les paramètres de verrouillage d'écran. L'écran n'est pas verrouillé.
  • Si la fonction de Déverrouillage sûr est activée, vous devez saisir votre NIP ou mot de passe pour utiliser le téléphone intelligent lorsque le mode Exhibition est lancé.
