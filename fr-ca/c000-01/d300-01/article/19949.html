Achat d'une application directement à partir du lien d'un code promo
  1. Si vous recevez un lien de code promo (comme dans un courriel ou un message SMS), cliquez sur ce lien.
  2. Effectuez l'une des opérations suivantes :
    • Si la page principale de l'App Catalog apparaît : appuyez sur OK pour effacer la notification concernant le code promo > naviguez pour trouver l'application que vous souhaitez > appuyez sur le nom de l'application.
    • Si la page d'une application en particulier apparaît : Appuyez sur OK pour effacer la notification concernant le code promo.
  3. Appuyez sur Télécharger pour [montant].
  4. Si l'application utilise des services de positionnement, une notification apparaît. Appuyez sur Continuer.
  5. Si le système vous le demande, entrez le mot de passe de votre profil Palm pour autoriser l'achat > Continuer.
  6. Si vous n'avez pas configuré un compte de facturation, vous serez invité à en configurer un (procédure).
  7. Effectuez l'une des opérations suivantes :
    • Pour acheter l'application en utilisant le code promo : appuyez sur Utiliser le code promo.
    • Pour acheter l'application par l'entremise de votre compte de facturation : Appuyez sur Acheter.
  8. Pour ouvrir une application téléchargée, appuyez sur Appuyez pour lancer. L'icône de l'application apparaît aussi sur la première page de l'écran de lancement. Cela vous permet de l'ouvrir aussi de cette manière.

Un code promo peut s'appliquer à une application en particulier ou à n'importe quelle application (jusqu'à un certain montant). Vous devez détenir un compte de facturation configuré dans l'App Catalog pour acheter des applications à l'aide d'un code promo; toutefois, les achats ne sont pas portés à votre compte lorsque vous utilisez le code.
