Réception et lecture des courriels
  1. Ouvrir Courriel .
  2. Appuyez sur pour afficher les fichiers des comptes > appuyez sur un fichier.
  3. Appuyez sur un message pour l’ouvrir.
  4. (Facultatif) Procédez de l'une des façons suivantes :
    • Pour appeler un numéro : appuyez sur le numéro > .
    • Pour envoyer un courriel à une adresse de courriel : appuyez sur l'adresse de courriel > saisissez votre message > .
    • Pour visiter une page Web : appuyez sur l'adresse.
    • Pour partager un lien vers un site Web : appuyez sur la touche Option  et maintenez-la enfoncée > appuyez sur le lien > Échanger le lien.
    • Pour copier un lien vers un site Web : appuyez sur la touche Option  et maintenez-la enfoncée > appuyez sur le lien > Copier l’URL.
  5. Appuyez sur ou sur pour afficher le message précédent ou suivant.

L’application Courriel synchronise les messages chaque fois que vous ouvrez un dossier de courrier. L'application synchronise également les messages sur un calendrier que vous pouvez définir (procédure). Pour synchroniser les messages lorsque vous avez un dossier déjà ouvert : appuyez sur .
