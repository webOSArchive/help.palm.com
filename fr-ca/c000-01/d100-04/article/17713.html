Mise à jour des applications téléchargées
  1. Ouvrir Gestionnaire de logiciels .
  2. Appuyez sur Applications pour rechercher les mises à jour potentielles pour les applications téléchargées qui apparaissent dans l'écran de lancement ou appuyez sur Autres pour vérifier la disponibilité de mises à jour pour les applications et services en arrière-plan qui n'apparaissent pas dans l'écran de lancement.
  3. Si des mises à jour sont disponibles pour certaines des applications ou certains des services que vous avez téléchargés, une flèche accompagnée d'un nombre apparaît dans le coin inférieur gauche de l'écran. Ce chiffre indique le nombre d'éléments pour lesquels des mises à jour sont disponibles. Appuyez sur la flèche.
  4. Effectuez l'une des opérations suivantes :
    • Pour afficher les informations concernant l'application et la mise à jour disponible, appuyez sur le nom de l'application.
    • Pour mettre à jour une application, appuyez sur l'icône en forme de flèche à droite du nom de l'application.
    • Pour mettre à jour toutes les applications : appuyez sur Installer les mises à jour.
