Utilisation de l'Assistant de transfert de données (ATD)

L'Assistant de transfert de données installé sur votre téléphone permet d'exporter vos contacts, événements de calendrier, tâches et notes par le biais d'un processus qui commence sur votre ordinateur. Pour amorcer la partie de ce processus qui s'accomplit sur votre ordinateur, visitez le site palm.com/migrate-pre pour télécharger l'ATD et préparer les données à exporter vers votre téléphone.

Il est recommandé d'utiliser l'Assistant de transfert de données une seule fois pour chaque source de données de votre ordinateur. Si vous utilisez l'Assistant de transfert de données pour transférer des donnés plusieurs fois à partir d'une même source, vous risquez d'obtenir des données dupliquées.

L'Assistant de transfert de données n'est pas une application de synchronisation. Si vous apportez des modifications aux données une fois qu'elles sont dans votre téléphone, elles ne sont pas reflétées dans le logiciel de l'ordinateur. Si vous désirez maintenir la synchronisation avec le logiciel de bureau, visitez le site palm.com/sync-solutions pour en savoir plus sur les logiciels de synchronisation tiers (vendus séparément).

L'Assistant de transfert de données vous permet de transférer les données vers l'un de vos comptes en ligne (Exchange ou Google, par exemple) ou vers votre compte de profil Palm. Il est possible de transférer une partie seulement des donnés vers le compte en ligne sélectionné. L'Assistant de transfert de données vous indique les donnés transférées et l'emplacement auquel elles sont transférées au moment où vous sélectionnez le compte. À ce stade, vous pouvez encore changer d'avis. Par la suite, ce ne sera plus possible.

 

  1. Lorsque vous avez fini de transférer les données vers votre téléphone, débranchez celui-ci de l'ordinateur en utilisant la procédure sécurisée.
  2. Sur votre téléphone, sélectionnez le compte sur lequel vous souhaitez transférer vos données. Effectuez l'une des opérations suivantes :

    • Si votre compte apparaît dans la liste : appuyez dessus. Au besoin, entrez les informations sur le compte > Me connecter.
    • Si vous souhaitez ajouter un compte en ligne à votre téléphone : appuyez sur Ajouter un compte > appuyez sur un type de compte > entrez les informations sur le compte > Me connecter.

      Pour configurer un type de compte non répertorié : appuyez sur Autres > appuyez sur une application > suivez les instructions à l'écran pour ajouter le compte.

      IMPORTANTIMPORTANT :

      Si vous souhaitez annuler ou reporter le transfert, vous devez le faire avant d'effectuer l'étape suivante. Pression

      Menu (en haut, à gauche) > Annuler le transfer > Annuler le transfert ou Continuer plus tard.

      Pour reprendre un transfert que vous avez reporté : branchez votre téléphone à votre ordinateur à l'aide du câble USB > Lecteur USB. Fermez ensuite la connexion en utilisant la procédure sécurisée. Appuyez sur un compte ou ajoutez-en un à l'écran de Transfert de données.

       

      Important

  3. Appuyez sur Terminer l'importation > OK.

    REMARQUE : l'importation s'effectue application par application et peut prendre de quelques minutes à une heure.

  4. Lorsqu'une notification indique que le transfert de données est terminé, appuyez sur Terminé.

Qu'advient-il des données une fois le transfert effectué? Si vous transférez des données vers un compte en ligne, les modifications apportées aux données sur le téléphone sont synchronisées avec le compte figurant sur le serveur et inversement. Si vous transférez les données vers votre compte de profil Palm, les données sont stockées sur le téléphone et sauvegardées dans votre profil Palm. Ces données ne sont disponibles par le biais d'aucun compte en ligne.

Une version locale d'Outlook et Palm® Desktop est installée sur mon ordinateur. Est-il possible de transférer les données des deux applications? Oui, c'est possible. Toutefois, si des donnés identiques sont stockées aux deux emplacements, le téléphone risque de contenir des données dupliquées.
