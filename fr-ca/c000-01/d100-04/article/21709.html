Recherche d'informations dans une application de votre téléphone
  1. Ouvrez la vue Cartes (procédure) ou l'écran de lancement (procédure).
  2. Commencez la saisie des informations que vous recherchez. En fonction de l'application dans laquelle vous souhaitez effectuer une recherche, ces informations peuvent être les suivantes : objet d'un courriel; adresse physique à localiser sur une carte, nom de signet, nom d'une application à télécharger ou tout autre terme.
  3. En fonction de l'application dans laquelle vous souhaitez effectuer une recherche, procédez comme suit. Vous devrez peut-être faire défiler l'écran pour afficher les résultats de recherche d'applications.
    • Signets et historique : soit votre téléphone affiche seulement le nombre de sites Web dans votre liste de signets et dans votre historique de navigation dont le nom contient le terme recherché, soit il affiche ce nombre accompagné d'une liste de ces sites. Si seul le nombre de sites Web est affiché, appuyez sur Signets et historique pour afficher la liste des sites Web. Appuyez sur une entrée pour accéder au site.
    • Courriel : soit votre téléphone affiche seulement le nombre de messages de courriel qui contiennent le terme recherché dans les expéditeurs ou les objets, soit il affiche ce nombre accompagné d'une liste de ces messages. Si seul le nombre de messages est affiché, appuyez sur Courriel pour afficher la liste des messages correspondants. Appuyez sur un message pour l'afficher.
    • Google Maps : pour localiser le terme que vous recherchez sur une carte, dans Lancement et recherche, appuyez sur Google Maps.
    • App Catalog : pour trouver dans l'App Catalog les applications qui correspondent au terme recherché, dans Lancement et recherche, appuyez sur Plus > HP webOS App Catalog.
    • YouTube : pour visualiser les vidéos sur YouTube qui correspondent au terme recherché, dans Lancement et recherche, appuyez sur Plus > YouTube.

Si le terme recherché ne vous donne aucun résultat pour une certaine application, peut-être devrez-vous activer une préférence pour que cette application soit incluse dans les recherches Tapez maintenant (procédure).
