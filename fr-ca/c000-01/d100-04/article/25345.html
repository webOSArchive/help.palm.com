Insertion des coordonnées d'un contact dans Contacts

Si vous souhaitez intégrer les coordonnées d'un contact à l'application Contacts, vous avez le choix entre plusieurs procédures :

  • Se connecter à un carnet d'adresses en ligne : si vous disposez d'un carnet d'adresses dans un compte en ligne fonctionnant avec la fonction HP Synergy (Google ou Microsoft Exchange, par exemple), vous pouvez configurer l'application Contacts de votre téléphone intelligent pour qu'elle se synchronise et affiche les contacts que vous enregistrez dans votre carnet d'adresses en ligne. Une fois que vous avez configuré la connexion au carnet d'adresses en ligne, les contacts que vous entrez en ligne apparaissent automatiquement dans vos Contacts, et les contacts que vous entrez sur votre téléphone intelligent se synchronisent au compte en ligne, à condition que ce dernier permette l'écriture à partir du téléphone intelligent.

    Vous pouvez configurer vos comptes contacts à partir de l'application Comptes (procédure) ou de l'application Contacts.

    La fonction Synergy offre un autre avantage : durant le même processus de configuration d'un compte contacts, vous configurez également ce même compte pour toutes les autres applications qui peuvent afficher des données provenant de ce compte.

    Par exemple, supposons que vous voulez configurer votre compte contacts de Google sur votre téléphone intelligent. La dernière étape de cette configuration consiste à sélectionner toutes les autres applications qui, à votre choix, recueilleront des données de votre compte Google (en plus de l'application Contacts, vous pouvez choisir les applications Messagerie, Calendrier et Courriel). Lorsque la configuration est terminée, votre téléphone intelligent commence à télécharger les contacts et toutes les autres données de toutes les autres applications dont vous avez autorisé la synchronisation.

  • Utiliser une solution de synchronisation tierce : si vous souhaitez utiliser une application de bureau pour stocker vos contacts, vous pouvez utiliser une solution tierce (vendue séparément) pour synchroniser vos contacts directement avec votre ordinateur (voir palm.com/sync-solutions).
  • Exporter les contacts à l'aide de l'Assistant de transfert de données (ATD) : lorsque vous utilisez l'ATD pour exporter des contacts depuis un logiciel de bureau (procédure), vous devez sélectionner le compte auquel vous souhaitez assigner ces contacts. Recherchez ces événements dans le compte sélectionné.
  • Entrez les contacts directement dans Contacts (procédure). Assignez le contact à l'un de vos carnets d'adresses en ligne ou créez le contact dans votre compte de profil Palm. Les informations figurant dans votre compte de profil Palm ne sont pas synchronisées, mais elles sont incluses à la sauvegarde régulière des informations du profil Palm.
  • Télécharger une vCard qui vous est envoyée par Courriel : les vCards peuvent être envoyées en pièces jointes à un courriel ou à un message multimédia, et vous pouvez télécharger le contenu d'une vCard dans l'application Contacts (procédure).
