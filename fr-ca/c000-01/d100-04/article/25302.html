Indiquer à Facebook des informations sur le lieu
  1. Ouvrez ! Facebook  >  .
  2. Si vous y êtes invité, appuyez sur Autoriser une fois pour activer les Services de positionnement.
  3. Effectuez l'une des opérations suivantes :
    • Pour créer un nouveau lieu : appuyez sur > entrez un nom pour le lieu > (facultatif) saisissez une description > Créer un lieu.
    • Pour sélectionner un lieu existant : saisissez le lieu où vous voulez indiquer que vous vous trouvez, ou appuyez sur un lieu dans la liste des lieux à proximité.
  4. Menu (en haut, à gauche) > Lieu.
  5. Appuyez sur l’une des options ci-après :

    Indiquer comme info incorrecte : permet d'identifier les informations incorrectes.

    Indiquer comme Abusif : permet d'identifier les informations inconvenantes.

    Indiquer comme définitivement fermé : permet d'informer Facebook de la fermeture définitive d'un lieu.

    Indiquer comme doublon : permet d'identifier un doublon dans la liste des lieux à proximité.

Lorsque vous consultez les lieux sur l'application Lieux de Facebook, il est possible que vous trouviez des informations que vous souhaitez porter à l'attention de Facebook.
