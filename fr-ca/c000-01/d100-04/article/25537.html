Première utilisation de l'application Contacts
  1. Ouvrir Contacts .
  2. Effectuez l'une des opérations suivantes :
    • Pour configurer un compte en ligne à synchroniser : appuyez sur Ajouter un compte, puis passez à l'étape 3.
    • Pour ouvrir Contacts sans créer de compte en ligne : appuyez sur Terminé. Votre compte Profil Palm s'ouvre.
  3. Pour configurer un compte répertorié, appuyez sur le type de compte, saisissez les informations relatives à ce compte > appuyez sur Me connecter > Terminé.
  4. Si vous configurez un compte Exchange, pour garantir la sécurité de votre compte Exchange, vous devez entrer un mot de passe ou un NIP qui seront exigés pour déverrouiller votre téléphone. Appuyez sur Définir le mot de passe ou Définir le NIP > saisissez un mot de passe ou un NIP > Terminé.
  5. Pour configurer un type de compte non répertorié, appuyez sur Autres > appuyez sur une application > suivez les instructions à l'écran pour ajouter le compte.
