Suppression d'une application

Appuyez sur la touche Option et maintenez-la enfoncée > appuyez sur l'icône > Supprimer.

Attention :

  • Vous ne pouvez supprimer que les applications que vous avez installées.
  • L'icône de l'application doit se trouver sur l'écran de lancement; si elle se trouve dans la barre de lancement rapide, déplacez-la d'abord vers l'écran de lancement (procédure).
  • Si vous devez libérer de la mémoire pour installer une nouvelle application, vous pouvez supprimer une application ou supprimer des fichiers en utilisant le mode Lecteur USB (procédure).
  • Si vous supprimez une application achetée à partir d'App Catalog HP webOS, vous pouvez la réinstaller gratuitement à tout moment (procédure).
