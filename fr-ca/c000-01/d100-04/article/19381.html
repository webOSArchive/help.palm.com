Utilisation de la composition restreinte

Avant de commencer

Vous pouvez obtenir votre numéro NIP2 auprès de votre fournisseur de service sans fil. Si vous ne saisissez pas votre code PIN2 correctement et que votre carte SIM est verrouillée, demandez votre code PUK2 à votre fournisseur de service sans fil.

BYB Close

  1. Ouvrez ! Téléphone  > Menu (en haut, à gauche) > Préférences et comptes > Composition restreinte.
  2. Pour activer ou désactiver la composition restreinte : appuyez sur Oui ou Non > saisissez le code PIN2 > Suivant.
  3. Pour modifier la liste de numéros restreints : lorsque la fonction Composition restreinte est activée, appuyez sur Afficher la liste des numéros restreints et procédez de l'une des manières suivantes :
    • Pour ajouter une entrée : Ajouter une entrée > saisissez le code PIN2 > saisissez le nom et le numéro de téléphone > Terminé.
    • Pour modifier une entrée : appuyez sur l'entrée > saisissez le code PIN2 > modifiez les informations > Terminé.
    • Pour effacer une entrée : faites défiler l'entrée jusqu'à ce qu'elle sorte de l'écran > saisissez le code PIN2 > Supprimer.

Lorsque la composition restreinte est activée, vous ne pouvez passer des appels que vers les numéros figurant sur une liste de numéros restreints et vous ne pouvez recevoir que les appels provenant des numéros de cette liste.
