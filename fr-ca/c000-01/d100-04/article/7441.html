Utilisation des liens d'une page Web
  • Pour accéder à une autre page par le biais du lien correspondant : effectuez un zoom avant sur la page (en appuyant deux fois dessus ou en effectuant le geste pincer/ouvrir) > appuyez sur le lien.
  • Pour composer un numéro de téléphone affiché sous forme de lien : appuyez sur le lien pour ouvrir le clavier numérique avec le numéro déjà saisi > appuyez sur .
  • Pour ouvrir un lien dans une nouvelle carte de navigateur : appuyez sur la touche Option  et maintenez-la enfoncée > appuyez sur le lien > appuyez sur Ouvrir dans une nouvelle carte.
  • Pour partager un lien : appuyez sur la touche Option  et maintenez-la enfoncée > appuyez sur le lien > appuyez sur Partager le lien. Un nouveau courriel s'affichera, et le lien y sera joint.
  • Pour copier un lien ou une URL : appuyez sur la touche Option  et maintenez-la enfoncée > appuyez sur le lien. Appuyez sur Copier l'URL, puis coller-la dans toute application autorisant les modifications de texte.
