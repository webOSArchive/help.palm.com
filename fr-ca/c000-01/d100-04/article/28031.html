Écran tactile

  1. Nom de l'application en cours : cette zone affiche le nom de l'application présentement ouverte sur le téléphone. Faites glisser votre doigt vers le bas sur le nom de l'application pour ouvrir le menu de l'application.
  2. Écran de lancement : affiche toutes vos applications. Appuyez sur une icône pour ouvrir une application.
  3. Icônes de connexion : cette zone affiche l'état des connexions sans fil ainsi que le niveau de charge de la batterie. Faites glisser votre doigt vers le bas sur les icônes pour ouvrir le menu de connexion.
  4. Indicateur de page : indique que certaines pages de l'écran de lancement se trouvent à droite de la page actuelle. Appuyez pour passer à la suivante.
  5. Nom de la page : appuyez de manière prolongée pour modifier le nom.
  6. Icônes de notification : signalent les appels manqués, les nouveaux messages vocaux reçus, les nouveaux courriels reçus, etc. Appuyez au bas de l'écran pour afficher des renseignements détaillés sur ces avis.
