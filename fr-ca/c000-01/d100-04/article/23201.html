Changer les paramètres par défaut des applications
  1. Ouvrez Info sur l'appareil  > Menu (en haut, à gauche) > Applications par défaut.
  2. Pour trouver quels types de données et quels types de fichiers disposent de plus d'une application, cherchez les noms d'applications qui apparaissent en caractères noirs.
  3. Effectuez l’une des opérations suivantes ou les deux :
    • Pour modifier l'application par défaut à utiliser pour diverses fonctions : sous Types de données > appuyez sur l'application actuellement affichée pour un type particulier de données > appuyez sur une autre application.
    • Pour modifier l'application par défaut à utiliser pour divers types de fichier : sous Types de fichier > appuyez sur l'application actuellement affichée pour un type particulier de fichier > appuyez sur une autre application.

Pour utiliser une autre application que celle par défaut configurée lorsque vous recherchez l'emplacement d'une adresse sur une carte : appuyez sur la touche Option en la maintenant enfoncée, appuyez sur l'adresse, puis appuyez sur une application.

Si vous disposez de plusieurs applications sur votre téléphone intelligent qui peuvent effectuer une fonction particulière, il est possible de modifier l'application par défaut à utiliser pour cette fonction.

En outre, si vous disposez sur votre téléphone intelligent de plusieurs applications permettant d'ouvrir un certain type de fichier, vous pouvez modifier l'application par défaut à utiliser pour ouvrir ce type de fichier.
