Recherche d'informations

La fonction Taper maintenant permet de rechercher des contacts et des applications sur votre téléphone intelligent, de rechercher des informations au sein d'une application ou d'effectuer une recherche sur Internet.

  1. Ouvrez la vue Cartes (procédure) ou l'écran de lancement (procédure).
  2. Commencez par taper le terme recherché.
  3. Effectuez l'une des opérations suivantes :
    • Pour ouvrir une application : appuyez sur l'icône.
    • Pour communiquer avec une personne : appuyez sur son nom pour afficher ses coordonnées. Appuyez sur une adresse de courriel pour commencer un nouveau message. Appuyez sur un numéro de téléphone pour appeler une personne ou appuyez sur pour envoyer un message SMS.
    • Pour rechercher un contact dans la liste d'adresse globale de votre entreprise : appuyez sur Recherche d'adresses globale.
    • Pour faire une recherche sur le Web : appuyez sur une option de recherche Web.
    • Pour recherchez un courriel contenant le terme de recherche saisi : appuyez sur Courriel.
    • Pour rechercher des signets ou pages Web que vous avez visités contenant le terme de recherche saisi : appuyez sur Signets et historique.
    • Pour afficher une carte situant l'emplacement que vous avez saisi : appuyez sur Cartes.
    • Pour chercher des applications dans l'App Catalog HP webOS qui correspondent à votre terme de recherche : appuyez sur App Catalog HP.
    • Pour créer un nouvel événement, message ou autre élément dans la vue Calendrier : appuyez sur Nouveau [élément].

    Pour rechercher une application, vous pouvez saisir soit le nom de l'application soit un mot clé, par exemple, saisissez Réunions pour le calendrier.

    Lorsqu'aucun contact ni aucune application ne correspond aux critères de recherche saisis, les options de recherche Web s'affichent automatiquement.
