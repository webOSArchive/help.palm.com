Gestion des applications téléchargées
  1. Ouvrir Gestionnaire de logiciels .
  2. Appuyez sur Applications pour gérer les applications téléchargées figurant sur l'écran de lancement ou appuyez sur Autre pour gérer les applications et services en arrière-plan qui ne figurent pas sur l'écran de lancement.
  3. Appuyez sur le nom d'une application ou d'un service.
  4. À partir de l'écran des détails de l'application sélectionnée, effectuez l'une des opérations suivantes.
    • Pour consulter les critiques disponibles : appuyez sur Critique de l'application > appuyez sur une critique.
    • Pour ajouter une critique : appuyez sur Critique de l'application > Critique.
    • Pour partager des informations sur l'application : appuyez sur Critique de l'application > Partager > Courriel ou Message SMS.
    • Pour signaler un problème d'application : appuyez sur Signaler un problème > Bogue > appuyez sur un type de problème. Saisissez la description du problème > .
    • Pour supprimer une application : appuyez sur Supprimer > Supprimer.
    • Pour réinstaller une application supprimée : Cherchez cette application (procédure) > appuyez sur le nom de l'application > Télécharger à nouveau gratuitement. Si vous avez déjà payé pour l'application, vous n'avez pas besoin de l'acheter à nouveau.
