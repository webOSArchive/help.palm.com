Parcourir les applications dans l'App Catalog HP webOS
  1. Ouvrir App Catalog .
  2. Prenez connaissance des conditions d'utilisation et appuyez sur Accepter.
  3. Appuyez sur l'icône d'une application présentée ou effectuez l'une des opérations suivantes :
    • Pour une recherche par nom : saisissez un terme de recherche > Entrée .
    • Pour une recherche d'applications par catégories : appuyez sur l'en-tête Parcourir les catégories, puis appuyez sur une catégorie. Si l'en-tête continue d'apparaître, appuyez dessus pour sélectionner les sous-catégories.
    • Pour restreindre votre recherche à l'aide de filtres : appuyez sur une icône au bas de l'écran.

    • Pour lancer une recherche uniquement parmi les applications en vedette, les plus populaires parmi les applications en vente ou les applications gratuites ou les applications les plus récentes, appuyez d'abord sur le bouton correspondant en bas de l'écran, puis saisissez un terme à rechercher.
  4. Appuyez sur un nom d'application pour afficher les détails correspondants.
  5. À partir de l'écran affichant les détails de l'application sélectionnée, effectuez l'une des opérations suivantes :
    • Pour agrandir la vue : appuyez dessus.
    • Pour consulter les critiques disponibles : appuyez sur Critiques.
    • Pour partager des informations au sujet de cette application avec un ami : appuyez sur Partager > Courriel ou Message SMS.
    • Pour accéder à la page d'accueil du développeur : appuyez sur Accueil développeur.
    • Pour accéder à la page de support produit du développeur : appuyez sur Support (si l'option est disponible).
    • Pour visualiser une vidéo de l'application dans l'application YouTube de votre téléphone intelligent : appuyez sur le lien YouTube (si cette option est disponible).
