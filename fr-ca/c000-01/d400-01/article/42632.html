Partage d'une page Web à l'aide de Toucher-partager
  1. Allez sur le Web > ouvrez la page Web que vous voulez envoyer vers votre tablette.
  2. Appuyez le téléphone intelligent sur le Bouton central de la tablette.
  3. Si la fonction Bluetooth n'a pas été activée sur l'un ou sur les deux appareils, appuyez sur Activer Bluetooth sur l'un ou sur les deux appareils > appuyez de nouveau le téléphone intelligent sur le Bouton central de votre tablette pour terminer le couplage.

    La page Web s'ouvre en plein écran sur la tablette.

Vous pouvez également partager une page Web ouverte sur votre tablette HP webOS avec votre téléphone intelligent. Répétez les étapes 1 et 2 ci-dessus, sans toutefois ouvrir la page Web sur votre tablette, puis mettez votre téléphone intelligent en contact avec votre tablette de façon à partager la page.
