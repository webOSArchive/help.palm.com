Configuration de l'application Courriel
  1. Ouvrir Courriel .
  2. Effectuez l'une des opérations suivantes :
    • Lorsque vous ouvrez Courriel pour la première fois, appuyez sur un type de compte ou appuyez sur Compte de courriel.
    • Toutes les autres fois où vous ouvrirez l'application Courriel, Menu (en haut, à gauche) > Préférences et comptes > Ajouter un compte > appuyez sur un type de compte ou sur Compte de courriel.
  3. Entrez votre courriel et votre mot de passe, puis appuyez sur Me connecter.
  4. Appuyez sur Oui/Non pour les applications pour lesquelles vous souhaitez télécharger des données à partir de ce compte.
  5. Appuyez sur Créer > Terminé.

REMARQUE : Si la configuration automatique échoue pour votre compte de courriel, un message d'erreur apparaît. Appuyez sur Configuration manuelle ou sur Trouver un Service Synergy, rassemblez les données requises pour la configuration et entrez les paramètres du compte.
