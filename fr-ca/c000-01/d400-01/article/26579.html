Connexion à un réseau

Souhaitez-vous vous connecter au Web à l'aide de la connexion de votre réseau téléphonique ? Vérifiez les éléments suivants :

  • Le mode avion est-il activé ? Désactivez-le : appuyez en haut, à droite de l'écran, puis appuyez sur Désactiver le mode avion.
  • Captez-vous un signal ? Dans le cas contraire ou si l'intensité du signal est faible, placez-vous à un endroit auquel l'intensité du signal est adéquate.
  • L'icône , ou  s'affiche-t-elle dans le coin supérieur droit de l'écran ? Dans le cas contraire, assurez-vous que le service de transmission de données est activé : Écran de lancement > Téléphone > Menu (en haut, à gauche) > Préférences et comptes > Utilisation des données > Activé. Si les services de données sont activés, mais que vous ne voyez pas l'icône, attendez de vous trouver dans une zone où l'icône est affichée; cela signifie que vous vous trouvez dans une zone où les services de données sont pris en charge. Vous pouvez aussi utiliser une connexion Wi-Fi, si vous y avez accès (voir ci-dessous).
  • Utilisez-vous l'itinérance ? Activez les services de données pour l'itinérance : Lancement rapide > Téléphone > Menu (en haut, à gauche) > Préférences > Itinérance de données > Activé.

Souhaitez-vous vous connecter à l'aide de la technologie Wi-Fi ? Vérifiez les éléments suivants :

  • Le mode Wi-Fi est-il désactivé ? Activez-le : appuyez en haut, à droite de l'écran, puis appuyez sur Wi-Fi > Activer Wi-Fi.
  • Captez-vous un signal Wi-Fi ? Dans le cas contraire ou si l'intensité du signal est faible, placez-vous à un endroit auquel l'intensité du signal est adéquate.
  • Souhaitez-vous vous connecter à un point d'accès Wi-Fi ? Écran de lancement > Wi-Fi > appuyez sur un réseau de la liste. Vous pouvez également appuyer sur Joindre le réseau et saisir les informations requises (demandez à une personne connaissant le point d'accès de vous fournir ces informations).
  • Avez-vous besoin de vous connecter à un site Web pour pouvoir vous connecter au point d'accès ? Il est parfois nécessaire de saisir des informations sur un point d'accès sur un site Web. Écran de lancement > Web > saisissez les informations.

Une erreur de connexion de données s'est-elle produite ? Appliquez les solutions suivantes :

  • Patientez quelques minutes, puis réessayez.
  • Mettez à jour la configuration du réseau et réessayez : barre de lancement rapide > Téléphone > Menu (en haut, à gauche) > Préférences et comptes > Mettre à jour la configuration du réseau (si l'option est disponible).
