Utiliser du contenu Flash

Lorsque vous ouvrez une page Web, le contenu Flash de la page apparaît sous la forme d'un bouton  de lecture sur un fond noir. Procédez de l’une des façons suivantes :

  • Pour lire le contenu Flash dans son format actuel, appuyez sur .
  • Pour lire le contenu Flash et le visualiser en faisant un zoom, appuyez deux fois dessus.

Gardez à l’esprit les points suivants :

  • Lorsque vous voyez dans le coin inférieur droit de l'écran, cela vous indique que vous travaillez avec un contenu Flash et non la page Web.
  • Pour utiliser les commandes de lecture avec le contenu Flash et interagir avec d'autres zones actives : appuyez sur le contenu Flash.
  • Il n'est pas possible d'effectuer de zoom avant/arrière lorsque vous travaillez avec un contenu Flash.
  • Pour cesser de travailler avec le contenu Flash et revenir à la page Web : appuyez sur .

La possibilité de visualiser du contenu Flash est activée par défaut. Pour la désactiver : Menu (en haut, à gauche) > Préférences. Dans Activer le contenu Flash, appuyez sur Activé pour le désactiver.

Pour que le contenu Flash soit lu dès l'ouverture d'un site Web sans avoir à appuyer sur le bouton de lecture : Menu (en haut, à gauche) > Préférences. Dans Chargement automatique du contenu Flash, appuyez sur Désactivé pour l'activer.
