Personnalisation de la configuration du navigateur Web
  1. Ouvrez ! Web  > Menu (en haut, à gauche) > Préférences.
  2. Configurez les options de votre choix :

    Moteur de recherche Web par défaut : appuyez pour modifier votre moteur de recherche par défaut.

    Bloquer les fenêtres publicitaires : empêche les sites Web d'ouvrir de nouvelles fenêtres de navigateur. Appuyez sur Oui pour passer de Oui à Non.

    Accepter les cookies : autorise les cookies de tous les sites Web que vous visitez, même si vous êtes redirigé vers un site depuis un autre. Appuyez sur Oui pour passer de Oui à Non. 

    JavaScript : la désactivation de JavaScript peut nuire au bon fonctionnement de certains sites Web. Appuyez sur Activé pour le désactiver.

    Activer le contenu Flash : désactiver cette option empêche l'affichage du contenu Flash. Appuyez sur Activé pour le désactiver.

    Charger automatiquement le contenu Flash : permet au contenu Flash d'être lu dès l'ouverture du site Web. Appuyez sur Désactivé pour l'activer.

    Effacer l'historique : appuyez sur cette option pour supprimer l'historique de navigation.

    Effacer les cookies : appuyez sur cette option pour supprimer les cookies utilisés par certains sites Web pour enregistrer l'historique des visites et les paramètres utilisateur.

    Effacer la mémoire cache : appuyez sur cette option pour supprimer tout contenu de page Web stocké dans la mémoire cache. Vous êtes ainsi assuré de consulter le contenu le plus à jour lorsque vous visitez une page Web.
