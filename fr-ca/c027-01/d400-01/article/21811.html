Personnaliser Facebook
  1. Ouvrez ! Facebook  > Menu (en haut, à gauche) > Préférences et comptes.
  2. Définissez les options suivantes :
    • Afficher les commentaires dans le fil de nouvelle : appuyez sur Oui/Non pour déterminer si les commentaires sont affichés avec votre fil de nouvelles.
    • Secouer pour recharger : Appuyez sur Oui/Non pour vous permettre de rafraîchir la page d'accueil des nouvelles en secouant l'appareil.
    • Toutes les 20 minutes : Définit l'intervalle des interrogations concernant les alertes reçues. Pour modifier cet intervalle, appuyez sur Toutes les 20 minutes > appuyez sur un nouvel intervalle.
    • Notifications HP webOS : appuyez pour activer ou désactiver cette fonction. Lorsque cette fonction est activée, les nouvelles alertes s'affichent sur le tableau de bord.
    • Comptes : Appuyez sur un compte pour modifier les paramètres de connexion ou pour retirer le compte. Vous ne devriez changer ces paramètres que si vous avez modifié ces informations dans votre compte Facebook en ligne. En effet, les informations saisies ici doivent correspondre à celles du compte en ligne.
    • Ajouter un compte : pour ajouter d'autres comptes Facebook. Par défaut, le dernier compte ajouté devient votre compte principal. Pour changer le compte principal, appuyez sur Compte principal > appuyez sur un autre compte. Facebook n'affiche les informations que sur votre compte principal.
