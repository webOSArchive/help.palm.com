Ouverture d’un fichier
  1. Ouvrir ! Quickoffice .
  2. Appuyez sur un compte pour afficher les fichiers de ce compte. Appuyez sur Fichiers stockés sur le Palm pour afficher les fichiers stockés localement sur le téléphone intelligent.
  3. Appuyez sur Documents, Feuilles de calcul, ou Présentations pour afficher une liste de ces types de fichiers uniquement ou sur Tous les fichiers.
  4. Appuyez sur le fichier.

Vous pouvez afficher un fichier dans un compte en ligne que vous configurez (procédure), un fichier que vous copiez de votre ordinateur (procédure) ou un fichier que vous recevez en pièce jointe à un courriel (procédure). Vous pouvez afficher des fichiers aux formats DOC, DOCX, TXT, XLS, XLSX, PPT et PPTX.
