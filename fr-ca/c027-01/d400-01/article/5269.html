Ajout d'un moteur de recherche Web
  1. Appuyez sur au bas de l'écran > appuyez sur Appuyer pour ajouter un moteur de recherche.
  2. Pour ajouter ce moteur de recherche à la liste de ceux disponibles dans Tapez maintenant : appuyez sur le nom du site Web répertorié sous Moteurs de recherche récemment visités.

En naviguant sur certains sites Web qui utilisent une fonction de recherche, comme dictionnary.com, vous verrez peut-être la mention Moteur de recherche Web disponible. Cela signifie que vous pouvez ajouter ce site Web à la liste des moteurs de recherche que votre téléphone intelligent consulte lorsque vous utilisez la fonction Tapez maintenant pour saisir un terme de recherche dans l'écran de lancement ou dans la vue Cartes.
