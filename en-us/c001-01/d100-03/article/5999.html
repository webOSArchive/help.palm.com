Download files from the web
  1. Open  Web .
  2. Go to the web page with the file you want to download.
  3. Tap the file name.
  4. If an app on your smartphone supports the file type, the file name appears at the bottom of the screen. Tap to open the file.

All files are downloaded to the smartphone's USB storage. If an app on your smartphone supports the file type, you can also find the file in that app. For example, if it's a PDF file, find it in PDF View, and if it's an MP3 file, find it in Music.
