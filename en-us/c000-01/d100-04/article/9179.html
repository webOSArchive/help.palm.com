Add a VPNC VPN profile

Before You Begin

Get this info from your email provider or system administrator:

  • Profile name
  • VPN server
  • Username and password
  • Group ID
  • Group secret
  • Domain
  • Dead peer detection setting
  • Encryption method: Secure, Weak, or None
  • Nat traversal method: Cisco-UDP, NAT-T (always), NAT-T (auto-detect), or Disabled

BYB Close

  1. Open  VPN  > Add Profile.
  2. If VPNC is not selected already in the Connection Type field, tap the Connection Type field > VPNC.
  3. Tap Enter server name > enter the server name.
  4. Tap Next > enter the information you got from your system administrator > Connect.
  5. If required, enter your security token or other password > Done.
  6. Tap Continue.

NOTE:

Dead Peer Protection: When selected, if no VPN traffic is received for five minutes, the client checks to see if the VPN server is still available.

Encryption Method: Tap an encryption method: Secure forces the client to use more secure AES or 3DES algorithms. Weak allows the weaker 1DES algorithm. Strong encryption may still be used, depending on what the VPN server decides. None enables no encryption. Encryption is not disabled and may still be used, depending on the VPN server. With the None option, the VPN profile is simply allowed to use no encryption.
