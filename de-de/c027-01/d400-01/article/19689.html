Kurzwahlnummern verwenden
  1. Öffnen Sie ! Telefon  > Menü (links oben) > Einstellungen und Konten > Kurzwahlnummern > Nummer hinzufügen.
  2. Sie können folgende Optionen festlegen:

    Bei Wahl von: Tippen Sie auf dieses Feld und wählen Sie die Anzahl der Ziffern aus, die eingegeben werden müssen, damit das Smartphone eine Kurzwahlnummer hinzufügt.

    Dieses Wahlpräfix verwenden: Geben Sie das Präfix ein, das am Anfang der gewählten Nummer hinzugefügt werden soll. Die Kombination aus Präfix und Ziffern muss eine vollständige Telefonnummer ergeben.

  3. Tippen Sie auf Fertig.

Kurzwahlnummern fügen am Anfang einer Nummer ein Präfix hinzu, so dass Sie statt der vollständigen Nummer eine Durchwahlnummer wählen können.

Beispiel: Eine vollständige Telefonnummer im Telefonsystem Ihres Unternehmens hat 11 Ziffern, z. B. +1 (408) 555-1122. Wenn Sie diese Nummer mit einem Telefon wählen, das an die interne Telefonanlage angeschlossen ist, müssen Sie lediglich die letzten fünf Ziffern wählen (51122). Wenn Sie auf dem Smartphone eine Kurzwahlnummer erstellen, wählen Sie unter Bei Wahl von die Option „5-stellig“ aus und geben Sie in das Feld Dieses Wahlpräfix verwenden „140855“ ein. Wenn Sie die fünfstellige Anschlussnummer Ihres Kollegen wählen (51122), wählt das Smartphone automatisch die ganze Nummer: 1-408-555-1122.
