Verknüpfungen auf Webseiten verwenden
  • So folgen Sie einer Verknüpfung zu einer anderen Seite: Vergrößern Sie die Seite (ziehen Sie sie auf oder tippen Sie zweimal darauf) > Tippen Sie auf die Verknüpfung.
  • So wählen Sie eine Telefonnummer, die als Verknüpfung angezeigt wird: Tippen Sie auf die Verknüpfung, um die Displaytastatur zu öffnen. Die Displaytastatur wird angezeigt und die Nummer ist bereits eingegeben. Tippen Sie auf .
  • So öffnen Sie einen Link in einer neuen Browserkarte: Drücken und halten Sie die Optionstaste  > Tippen Sie auf den Link > Tippen Sie auf In neuer Karte öffnen.
  • So geben Sie einen Link frei: Drücken und halten Sie die Optionstaste  > Tippen Sie auf den Link > Tippen Sie auf Link freigeben. Es wird eine neue E-Mail mit dem Link im Anhang geöffnet.
  • So kopieren Sie einen Link oder eine URL: Drücken und halten Sie die Optionstaste  > Tippen Sie auf den Link. Tippen Sie auf URL kopieren und fügen Sie diese in eine Anwendung ein, die Texteingaben unterstützt.
