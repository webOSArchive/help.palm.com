Die Facebook-Seite eines Freundes anzeigen
  1. Öffnen Sie ! Facebook  > .
  2. Sie haben folgende Möglichkeiten:
    • Eine Freundschaftseinladung auf Facebook senden: Tippen Sie auf Suche > Alle Benutzer > Geben Sie den Namen einer Person ein > Tippen Sie auf den Namen > Als FreundIn hinzufügen.
    • Nach einem Freund suchen: Tippen Sie auf Suche > Geben Sie den Namen der Person ein > Tippen Sie auf den Namen.
    • Facebook-Seite eines Freundes anzeigen: Tippen Sie auf Suche > Geben Sie den Namen der Person ein > Tippen Sie auf den Namen. Tippen Sie unten auf dem Bildschirm für Beiträge auf Pinnwand, tippen Sie für Informationen zu Ihrem Freund auf Info und für Fotoalben auf Bilder.
    • Beiträge auf die Pinnwand eines Freundes schreiben: Tippen Sie auf Suche > Geben Sie den Namen der Person ein > Tippen Sie auf den Namen. Tippen Sie auf Schreib etwas > Geben Sie einen Kommentar ein > Tippen Sie auf Veröffentlichen.
    • Geburtstage Ihrer Freunde: Tippen Sie auf Veranstaltungen > Geburtstage.
    • So zeigen Sie von Ihren Freunden oder Kontakten erstellte Seiten an: Tippen Sie auf Suchen > Seiten.
