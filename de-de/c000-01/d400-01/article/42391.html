Anwendung direkt über einen Werbelinkcode kaufen
  1. Wenn Sie z. B. in einer E-Mail oder SMS einen Link mit einem Rabattcode erhalten, klicken Sie auf den Link.
  2. Sie haben folgende Möglichkeiten:
    • Wenn die App Catalog-Hauptseite angezeigt wird: Tippen Sie auf OK, um den Rabattcode-Hinweis auszublenden > Suchen Sie die gewünschte Anwendung > Tippen Sie auf den Anwendungsnamen.
    • Wenn die Seite einer bestimmten Anwendung angezeigt wird: Tippen Sie auf OK, um die Werbelinkbenachrichtigung zu schließen.
  3. Tippen Sie auf Herunterladen für: [Betrag].
  4. Für den Fall, dass diese Anwendung standortbasierte Dienste verwendet, wird eine Meldung angezeigt. Tippen Sie auf Weiter.
  5. Wenn Sie dazu aufgefordert werden, geben Sie das Kennwort für Ihr HP webOS-Konto ein, um den Kauf zu autorisieren > Weiter.
  6. Falls Sie kein Rechnungskonto besitzen, werden Sie zum Einrichten eines Kontos aufgefordert (Weitere Informationen).
  7. Sie haben folgende Möglichkeiten:
    • Anwendungen mit Rabattcode bezahlen: Tippen Sie auf Promocode verwenden.
    • Anwendung mithilfe des Rechnungskontos kaufen: Tippen Sie auf Kaufen.
  8. Um eine heruntergeladene Anwendung zu öffnen, tippen Sie auf Zum Starten tippen. Das Anwendungssymbol wird auch auf der ersten Seite des Startbildschirms angezeigt und kann auch von dort aus geöffnet werden.

Der Rabattcode kann für eine bestimmte Anwendung gelten oder für eine beliebige Anwendung (bis zu einem festgelegten Betrag) eingelöst werden. Es muss ein Rechnungskonto in App Catalog eingerichtet sein, damit Sie Anwendungen mit einem Promocode erwerben können. Ihr Konto wird jedoch nicht belastet, wenn Sie den Code verwenden.
