E-Mails empfangen und lesen
  1. Öffnen Sie E-Mail .
  2. Tippen Sie auf , um die Ordner der einzelnen Konten anzuzeigen > Tippen Sie auf einen Ordner.
  3. Tippen Sie auf eine Nachricht, um sie zu öffnen.
  4. (Optional) Sie haben folgende Möglichkeiten:
    • Nummer anrufen: Tippen Sie auf die Nummer > .
    • Eine SMS senden: Tippen und halten Sie die Nummer gedrückt > tippen Sie auf SMS.
    • Kontakten eine Nummer hinzufügen: Tippen Sie auf die Nummer und halten Sie sie > tippen Sie auf Zu Kontakten hinzufügen. Tippen Sie auf Als neuen Eintrag speichern, um einen neuen Kontakt zu erstellen, oder tippen Sie auf Zu vorhandenem Kontakt hinzufügen, um diese Nummer einem vorhandenen Kontakt hinzuzufügen.
    • E-Mail senden: Tippen Sie auf die E-Mail-Adresse > Geben Sie die Nachricht ein > .
    • Eine Internetadresse öffnen: Tippen Sie auf die Adresse.
    • Einen Link auf einer Website freigeben: Halten Sie die Optionstaste gedrückt > Tippen Sie auf den Link > Link freigeben.
    • Einen Link auf eine Website kopieren: Drücken und halten Sie die Optionstaste  > Tippen Sie auf den Link > URL kopieren.
    • E-Mail kopieren, um sie in ein Memo oder ein anderes Dokument einzufügen: Menü (oben links) > Bearbeiten > Alle kopieren.
  5. Tippen Sie auf bzw. , um die vorherige bzw. nächste Nachricht anzuzeigen.

Die E-Mail-Anwendung synchronisiert die Nachrichten jedes Mal, wenn Sie einen Mailordner öffnen. Außerdem können Sie einen Zeitplan für das Synchronisieren festlegen (weitere Informationen). Nachrichten synchronisieren, wenn bereits ein Ordner geöffnet ist: Tippen Sie auf .
