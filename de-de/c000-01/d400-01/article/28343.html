Dateien zwischen dem Smartphone und dem Computer kopieren
  1. Verbinden Sie das Smartphone über das USB-Kabel mit Ihrem Computer.
  2. Tippen Sie auf Ihrem Smartphone auf USB-Laufwerk.
  3. Windows: Falls auf Ihrem Computer der Hardware-Assistent angezeigt wird,  schließen Sie ihn.
  4. Öffnen Sie Arbeitsplatz (Windows XP), Computer (Windows Vista/Windows 7) oder den Finder (Mac) und doppelklicken Sie auf das Laufwerk, das Ihrem Smartphone entspricht.
  5. Verschieben Sie Dateien zwischen Computer und Telefon durch Ziehen und Ablegen.
  6. Windows: Wenn Sie das Gerät vom Computer trennen möchten, klicken Sie mit der rechten Maustaste auf das Laufwerk, das Ihrem Smartphone entspricht, und klicken Sie auf Auswerfen.
    Mac: Ziehen Sie am Desktop das Laufwerk, das Ihrem Smartphone entspricht, in den Papierkorb. Anstelle von „Papierkorb“ wird Auswerfen angezeigt.
  7. Ziehen Sie das USB-Kabel vom Computer ab, wenn der USB-Laufwerk-Bildschirm nicht mehr auf dem Smartphone angezeigt wird.

Sie können beliebige Dateien auf Ihr Smartphone übertragen. Wenn Sie eine Datei auf dem Smartphone öffnen möchten, muss darauf eine Anwendung installiert sein, die den Dateityp lesen kann.
