Fotos in Alben anordnen

Zum Erstellen von Alben und zum Hinzufügen von Bildern müssen Sie Ihr Smartphone in den USB-Laufwerkmodus versetzen.

  1. Schließen Sie Ihr Smartphone mit dem USB-Kabel an Ihren Computer an > USB-Laufwerk.
  2. Tippen Sie auf OK, wenn Sie dazu aufgefordert werden.
  3. Öffnen Sie auf dem Computer das Laufwerk, das dem Smartphone entspricht.
  4. Verwenden Sie die Steuerelemente Ihres Computers, um für das neue Album einen neuen Ordner zu erstellen. Damit das neue Album von der Fotos-Anwendung erkannt wird, sollten Sie neue Alben im Stammordner des Laufwerks erstellen, das dem Smartphone entspricht, und nicht als Unterordner.

    Ein von Ihnen erstelltes Album wird in der Fotoanwendung nur dann angezeigt, wenn es mindestens ein Foto im JPG-, BMP- oder PNG-Format enthält.

  5. Verschieben oder kopieren Sie Fotos in die Alben. Nutzen Sie dazu die Funktionen des Computers.
  6. Verbindung sicher trennen: Falls Sie zum Trennen der Verbindung nicht auf Auswerfen klicken, wird Ihr Smartphone zurückgesetzt, wenn Sie das USB-Kabel abziehen:

    Auf einem Windows-Computer müssen Sie mit der rechten Maustaste auf das Laufwerk klicken, das Ihrem Smartphone entspricht > Auswerfen.

    Wenn Sie einen Mac-Computer verwenden: Ziehen Sie das Laufwerk, das Ihrem Smartphone entspricht, in den Papierkorb. Anstelle von „Papierkorb“ wird Auswerfen angezeigt.

  7. Ziehen Sie das USB-Kabel vom Computer und vom Smartphone ab, wenn der Bildschirm „USB-Laufwerk“ nicht mehr auf dem Smartphone angezeigt wird.

Wenn Sie ein von Ihnen erstelltes Album löschen möchten, schließen Sie Ihr Smartphone an Ihren Computer an und tippen Sie auf USB-Laufwerk. Öffnen Sie auf Ihrem Computer das Laufwerk, das dem Smartphone entspricht, und löschen Sie den Ordner, der dem Album entspricht.

Sie können unter „Fotos“ ein beliebiges Foto als Hintergrund auswählen (Weitere Informationen). Das Foto muss sich nicht im Album „Hintergründe“ befinden.
