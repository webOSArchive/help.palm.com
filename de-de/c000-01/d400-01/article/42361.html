Nachrichten löschen

In einer Liste: Werfen Sie das Element über die Seite des Bildschirms.
In einer geöffneten Nachricht: Tippen Sie auf .

Wenn Sie eine Nachricht löschen, wird sie in den Ordner der gelöschten Elemente verschoben. Sie befindet sich also immer noch auf dem Smartphone. Abhängig vom E-Mail-Konto heißt dieser Ordner „Gelöschte Elemente“, „Gelöscht“, „Papierkorb“ oder Ähnliches.

Wenn Sie verhindern möchten, dass eine Nachricht versehentlich gelöscht wird, können Sie die Einstellung „Löschbestätigung“ aktivieren (weitere Informationen). Wenn Sie versehentlich eine Nachricht löschen, können Sie sie zum ursprünglichen Ordner zurückkehren (Weitere Informationen). Um eine Nachricht endgültig von Ihrem Smartphone zu entfernen, löschen Sie diese aus dem Ordner „Gelöschte Objekte“. Alle Nachrichten aus dem Ordner der gelöschten Elemente löschen: Tippen Sie im Ordner auf Menü (oben links) > Papierkorb leeren > Papierkorb leeren.
