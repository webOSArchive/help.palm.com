Verbindung zu einem neuen gesicherten WLAN-Netzwerk herstellen
  1. Öffnen Sie ! WLAN .
  2. Stellen Sie sicher, dass das WLAN aktiviert ist (Weitere Informationen). Sie haben folgende Möglichkeiten:
    • Wenn das gewünschte Netzwerk angezeigt wird: Tippen Sie auf den Netzwerknamen > Geben Sie Benutzername und/oder Kennwort ein > Anmelden. Sie haben alle erforderlichen Schritte ausgeführt.
    • Wenn das gewünschte Netzwerk nicht angezeigt wird: Tippen Sie auf Netzwerk wählen > Geben Sie den Netzwerknamen ein.
  3. Tippen Sie in das Feld Sicherheit > Tippen Sie auf den Netzwerktyp: Offen, WPA-privat, WEP oder Enterprise.
  4. Tippen Sie auf Verbinden.
  5. Es wird (je nach Netzwerktyp) eine Reihe unterschiedlicher Fenster angezeigt. Geben Sie Folgendes ein bzw. wählen Sie Folgendes aus, wenn Sie dazu aufgefordert werden: Benutzername, Kennwort und/oder Zertifikat.
  6. Tippen Sie auf Anmelden.
