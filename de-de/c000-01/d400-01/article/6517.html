HP webOS App Catalog nach Anwendungen durchsuchen
  1. Öffnen Sie App Catalog .
  2. Lesen Sie die Nutzungsbedingungen durch und tippen Sie auf Akzeptieren.
  3. Tippen Sie auf das Symbol einer ausgewählten Anwendung oder gehen Sie wie folgt vor:
    • Nach Name suchen: Geben Sie einen Suchbegriff ein > Eingabetaste .
    • Anwendungen nach Kategorie durchsuchen: Tippen Sie auf die Überschrift Kategorien durchblättern und tippen Sie dann auf eine Kategorie. Tippen Sie auf die Überschrift, wenn diese weiterhin angezeigt wird, um Unterkategorien auszuwählen.
    • Suche durch Filtern eingrenzen: Tippen Sie unten im Bildschirm auf ein Symbol.


      		Beworbene Anwendungen (nur auf der Startseite)

      		 Alle Anwendungen in der Kategorie (nur auf dem Bildschirm einer Kategorie/Unterkategorie)
      Top-Anwendungen (kostenpflichtig)

      		 Top-Anwendungen (kostenlos)

      		 Neueste Anwendungen

       

    • Ausschließlich in beworbenen Anwendungen, Top-Anwendungen (kostenpflichtig bzw. kostenlos) oder neuen Anwendungen suchen: Tippen Sie unten im Bildschirm zunächst auf das entsprechende Symbol und geben Sie dann einen Suchbegriff ein.
  4. Tippen Sie auf einen Anwendungsnamen, um Informationen zur Anwendung anzuzeigen.
  5. Verfahren Sie auf dem Bildschirm mit den Anwendungsdetails wie folgt:
    • Ansicht vergrößern: Tippen Sie auf einen Screenshot.
    • Rezensionen anzeigen: Tippen Sie auf Rezensionen.
    • Informationen über eine Anwendung an einen Freund weitergeben: Tippen Sie auf Freigeben > E-Mail oder SMS.
    • Website des Entwicklers anzeigen: Tippen Sie auf Entwickler-Startseite.
    • Supportseite des Entwicklers für das Produkt anzeigen: Tippen Sie auf Support (falls verfügbar).
    • Ein Video der Anwendung in der YouTube-Anwendung Ihres Smartphones anzeigen: Tippen Sie auf den YouTube-Link (falls vorhanden).
