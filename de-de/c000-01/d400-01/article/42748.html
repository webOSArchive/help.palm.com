Informationen suchen

Mit der Funktion "Einfach lostippen" können Sie nach Kontakten und Anwendungen auf Ihrem Smartphone oder nach Informationen in einer Anwendung suchen. Sie können aber auch im Internet suchen.

  1. Öffnen Sie die Kartenansicht (Weitere Informationen) oder den Startbildschirm (Weitere Informationen).
  2. Geben Sie den Suchbegriff ein.
  3. Sie haben folgende Möglichkeiten:
    • Anwendung öffnen: Tippen Sie auf das Symbol.
    • Kontakt zu einer Person aufnehmen: Tippen Sie auf einen Namen, um die Kontaktdaten der Person anzuzeigen Tippen Sie auf  eine E-Mail-Adresse, um eine Nachricht zu erstellen. Tippen Sie auf eine Telefonnummer, um die Person anzurufen, oder tippen Sie auf , um eine SMS zu senden.
    • So durchsuchen Sie die globale Adressenliste (GAL) Ihres Unternehmens nach einem Kontakt: Tippen Sie auf Globale Adressensuche.
    • Im Internet suchen: Tippen Sie auf eine Internetsuchoption.
    • E-Mail-Nachrichten mit dem eingegebenen Begriff suchen: Tippen Sie auf E-Mail.
    • Nach Webfavoriten oder aufgerufenen Seiten mit dem eingegebenen Begriff suchen: Tippen Sie auf Favoriten und Verlauf.
    • Einen eingegebenen Standort auf der Karte anzeigen: Tippen Sie auf Karten.
    • HP webOS App Catalog nach Anwendungen durchsuchen, die mit Ihrem Suchbegriff übereinstimmen: Tippen Sie auf HP App Catalog.
    • Ein neues Kalenderereignis, eine neue Nachricht oder ein anderes Element erstellen: Tippen Sie auf Neu [Element].

    Um nach einer Anwendung zu suchen, können Sie entweder den Namen der Anwendung oder ein Stichwort eingeben, z. B. Meetings für Kalender.

    Wenn bei der Suche weder Kontakte noch Anwendungen gefunden werden, werden automatisch die Optionen für die Internetsuche angezeigt.
