Rechnungskonten aktualisieren oder löschen
  1. Öffnen Sie App Catalog  > Menü (oben links) > Einstellungen und Konten.
  2. Tippen Sie auf den Kontonamen.
  3. Geben Sie das Kennwort für das Palm-Profil ein > Weiter.

    4. Wenn Sie das Kennwort für das Palm-Profil dreimal falsch eingeben, wird eine Nachricht mit Informationen darüber angezeigt, wie Sie den Support von Palm kontaktieren können.


  4. Sie haben folgende Möglichkeiten:
    • Kontoinformationen bearbeiten: Ändern Sie die Kontoinformationen > Absenden.
    • Kontoinformationen löschen: Tippen Sie auf Konto entfernen.
