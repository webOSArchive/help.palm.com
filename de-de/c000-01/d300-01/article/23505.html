Datenübertragungsassistent (DTA) verwenden

Mit dem Datenübertragungsassistenten (DTA) auf Ihrem Telefon können Sie einen Übertragungsvorgang abschließen, den Sie auf Ihrem Computer begonnen haben, um Ihre Kontakte, Kalenderereignisse, Aufgaben und Notizen auf das Telefon zu exportieren. Besuchen Sie zum Starten des Exports auf Ihrem Computer palm.com/migrate-pre, um den Datenübertragungsassistenten (DTA) herunterzuladen und Ihre Daten für den Export auf Ihr Telefon vorzubereiten.

Es wird empfohlen, den DTA für jede Datenquelle auf Ihrem Computer nur einmal auszuführen. Wenn Sie mit dem DTA mehrmals Daten aus der gleichen Datenquelle übertragen, besteht die Gefahr, dass sich danach viele Daten doppelt auf Ihrem Telefon befinden.

Der Datenübertragungsassistent (DTA) ist keine Synchronisierungssoftware. Änderungen, die Sie an Daten vornehmen, nachdem diese auf Ihr Telefon übertragen wurden, werden in der Desktop-Software nicht angezeigt. Wenn Sie weiterhin mit der Desktop-Software Synchronisierungen vornehmen möchten, besuchen Sie palm.com/sync-solutions, um Informationen zu Synchronisierungssoftware von Drittanbietern zu erhalten (separat erhältlich).

Im DTA können Sie festlegen, ob Sie die Daten in eines Ihrer Online-Konten (z. B. Exchange oder Google) oder in Ihr Palm-Profil übertragen möchten. Es können nicht immer alle ausgewählten Daten vom Computer in ein Online-Konto übertragen werden. Wenn Sie ein Konto auswählen, weist der DTA darauf hin, welche Daten übertragen werden und wohin. Sie können dann ein anderes Konto auswählen. Später ist dies nicht mehr möglich.

 

  1. Wenn das Übertragen der Daten auf Ihr Telefon beendet ist, trennen Sie das Telefon sicher von Ihrem Computer.
  2. Wählen Sie auf Ihrem Telefon das Konto aus, in das Sie Ihre Daten übertragen möchten. Sie haben folgende Möglichkeiten:

    • Wenn Ihr Konto aufgeführt ist: Tippen Sie darauf. Geben Sie falls erforderlich die Kontoinformationen ein > Anmelden.
    • Wenn Sie ein Online-Konto zu Ihrem Telefon hinzufügen möchten: Tippen Sie auf Konto hinzufügen > Tippen Sie auf einen Kontotyp > Geben Sie die Kontoinformationen ein > Anmelden.

      Wenn Sie einen nicht aufgeführten Kontotyp einrichten möchten: Tippen Sie auf Weitere suchen, tippen Sie auf eine Anwendung, und befolgen Sie die Anweisungen auf dem Bildschirm, um das Konto hinzuzufügen.

      WICHTIGWICHTIG:

      Wenn Sie die Übertragung abbrechen oder verschieben möchten, müssen Sie dies tun, bevor der nächste Schritt abgeschlossen ist. Tippen

      Menü (oben links) > Übertragung abbrechen > Übertragung abbrechen oder Später fortfahren.

      Übertragungen fortsetzen, die Sie verschoben haben: Verbinden Sie Ihr Telefon mit dem USB-Kabel mit Ihrem Computer > USB-Laufwerk. Trennen Sie die Verbindung dann sicher. Tippen Sie im Bildschirm „Datenübertrag.“ auf ein Konto oder fügen Sie es hinzu.

       

      Wichtig

  3. Tippen Sie auf Import fertig stellen > OK.

    HINWEIS: Der Import erfolgt Anwendung für Anwendung und kann wenige Minuten bis eine Stunde dauern.

  4. Wenn die Benachrichtigung angezeigt wird, dass die Datenübertragung abgeschlossen ist, tippen Sie auf Fertig.

Was geschieht mit meinen Daten, nachdem sie übertragen wurden? Wenn Sie die Daten in ein Online-Konto übertragen haben, werden Änderungen, die Sie auf dem Telefon an den Daten vornehmen, mit dem Konto auf dem Server synchronisiert und umgekehrt. Wenn Sie die Daten in Ihr Palm-Profilkonto übertragen haben, werden die Daten auf dem Telefon gespeichert und regelmäßig in Ihr Palm-Profil gesichert. Auf diese Daten können Sie nicht mit Online-Konten zugreifen.

Ich habe eine lokale Version von Outlook und Palm® Desktop auf meinem Computer installiert. Kann ich Daten von beiden Anwendungen übertragen? Ja, das ist möglich. Wenn Sie an beiden Speicherorten identische Daten gespeichert haben, kann es jedoch passieren, dass sich nach dem Übertragen viele Daten doppelt auf Ihrem Telefon befinden.
