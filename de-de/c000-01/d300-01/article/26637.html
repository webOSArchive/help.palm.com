Dateien von Ihrem Smartphone löschen
  1. Verbinden Sie das Smartphone über das USB-Kabel mit Ihrem Computer.
    Windows: Falls auf Ihrem Computer der Hardware-Assistent angezeigt wird,  schließen Sie ihn. 
  2. Tippen Sie auf Ihrem Smartphone auf USB-Gerät.
  3. Öffnen Sie auf Ihrem Computer das Laufwerk, das dem Smartphone entspricht.
  4. Navigieren Sie mithilfe der entsprechenden Optionen auf dem Computer zu den Dateien, und löschen Sie diese.
  5. Windows: Wenn Sie das Gerät vom Computer trennen möchten, klicken Sie mit der rechten Maustaste auf das Laufwerk, das Ihrem Smartphone entspricht, und klicken Sie auf Auswerfen.
    Mac: Ziehen Sie am Desktop das Laufwerk, das Ihrem Smartphone entspricht, in den Papierkorb. Anstelle von „Papierkorb“ wird Auswerfen angezeigt.
  6. Ziehen Sie das USB-Kabel vom Computer ab, wenn der Bildschirm USB-Laufwerk nicht mehr auf dem Smartphone angezeigt wird.

Verwenden Sie den USB-Laufwerkmodus, um Dateien auf Ihrem Smartphone zu löschen. Sie können alle Dateien löschen, die Sie auf dem Smartphone hinzugefügt haben, egal, ob diese von einem Computer kopiert oder aus einer Nachricht heruntergeladen wurden.
