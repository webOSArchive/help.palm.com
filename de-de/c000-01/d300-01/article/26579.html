Mit Netz verbinden

Sie möchten eine Verbindung zum Internet über das Mobilfunknetz herstellen? Überprüfen Sie Folgendes:

  • Ist der Flugzeugmodus eingeschaltet? Schalten Sie ihn aus: Tippen Sie auf die obere rechte Bildschirmecke und dann auf Flugzeugmodus ausschalten.
  • Haben Sie ein Signal? Wenn nicht oder wenn das Signal schwach ist, begeben Sie sich an einen Ort mit einem stärkeren Signal.
  • Sehen Sie oben rechts auf dem Bildschirm das Symbol , oder ? Wenn nicht, überprüfen Sie, ob Datendienste aktiviert sind: Startbildschirm > Telefon > Menü (oben links) > Einstellungen und Konten > Datenverwendung > Aktiviert. Wenn Datendienste aktiviert sind, das Symbol aber weiterhin nicht angezeigt wird, warten Sie, bis das Symbol zu sehen ist. Es wird angezeigt, sobald Sie sich in einem Empfangsbereich befinden, in dem Datendienste verfügbar sind. Sie können, falls vorhanden, auch eine WLAN-Verbindung verwenden (siehe unten).
  • Roamen Sie? Aktivieren Sie die Datendienste beim Roaming: Schnellstart > Telefon > Menü (oben links) > Einstellungen und Konten > Datenroaming > Aktiviert.

Sie möchten eine Verbindung über WLAN herstellen? Überprüfen Sie Folgendes:

  • Ist WLAN ausgeschaltet? Schalten Sie es ein: Tippen Sie auf die obere rechte Bildschirmecke und dann auf WLAN > WLAN einschalten.
  • Empfangen Sie ein WLAN-Signal? Wenn nicht oder wenn das Signal schwach ist, begeben Sie sich an einen Ort mit einem stärkeren Signal.
  • Müssen Sie eine Verbindung zu einem WLAN Access Point herstellen? Startbildschirm > WLAN > Tippen Sie auf ein Netzwerk in der Liste. Sie können auch auf Netzwerk wählen tippen und die erforderlichen Zugangsdaten eingeben (diese erhalten Sie von der Person, die über den Access Point Bescheid weiß).
  • Müssen Sie sich anmelden, um die Verbindung herzustellen? Manchmal müssen Access-Point-Daten auf einer Website eingegeben werden. Startbildschirm > Internet > Geben Sie die Daten ein.

Ist ein Datenverbindungsfehler aufgetreten? Versuchen Sie Folgendes:

  • Versuchen Sie es nach einigen Minuten erneut.
  • Aktualisieren Sie die Netzwerkeinstellungen und versuchen Sie es erneut: Schnellstart > Telefon > Menü (oben links) > Einstellungen und Konten > Netzwerkeinstellungen aktualisieren (falls verfügbar).
