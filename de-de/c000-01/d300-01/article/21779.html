Im Internet suchen
  1. Öffnen Sie die Kartenansicht (Weitere Informationen) oder den Startbildschirm (Weitere Informationen).
  2. Geben Sie den Suchbegriff ein.
  3. Sie haben folgende Möglichkeiten. Möglicherweise müssen Sie nach unten rollen, um die Ergebnisse der Internetsuchmaschinen anzuzeigen.
    • Die Internetsuche mit Ihrer Standard-Suchmaschine durchführen: Tippen Sie auf den Namen der Suchmaschine. Die Suchmaschine wird direkt unter den übereinstimmenden Kontakteinträgen angezeigt.
    • In Ihrer Standard-Suchmaschine auf Grundlage der eingegebenen Zeichen Vorschläge für Suchbegriffe anzeigen: Tippen Sie rechts vom Namen der Standard-Suchmaschine auf Vorschläge . Sollten bei der Suche keine Kontakte, Anwendungen oder andere Elemente auf Ihrem Telefon gefunden werden, werden automatisch Vorschläge angezeigt. Tippen Sie auf einen Vorschlag, um das betreffende Element zu suchen.
    • Die Internetsuche mit einer anderen Suchmaschine durchführen: Tippen Sie unter Starten und Suchen auf den Namen einer Suchmaschine. Wenn die Suchmaschine für eine bestimmte Website gilt, z. B. Wikipedia, wird nur diese Website durchsucht. Tippen Sie zum Anzeigen weiterer Suchoptionen unter Starten und Suchen auf Mehr.

Um direkt zu einer Website zu wechseln, öffnen Sie die Kartenansicht oder den Startbildschirm und geben Sie die Adresse einer Website wie palm.com ein. Tippen Sie dann auf Zur Website wechseln.
