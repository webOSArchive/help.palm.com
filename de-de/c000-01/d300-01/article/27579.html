Anwendungen löschen

Halten Sie die Optionstaste gedrückt > Tippen Sie auf das Symbol > Löschen.

Beachten Sie:

  • Sie können nur Anwendungen löschen, die Sie selbst installiert haben.
  • Das Anwendungssymbol muss sich auf dem Startbildschirm befinden. Wenn es sich im Schnellstart befindet, müssen Sie es zunächst auf den Startbildschirm verschieben (Weitere Informationen).
  • Wenn Sie vor der Installation einer Anwendung Speicherplatz freigeben müssen, können Sie entweder eine andere Anwendung oder einzelne Dateien im USB-Laufwerkmodus löschen (Weitere Informationen).
  • Wenn Sie eine über den Anwendungskatalog erworbene Anwendung löschen, können Sie die Anwendung jederzeit ohne weitere Kosten wieder installieren (Weitere Informationen).
