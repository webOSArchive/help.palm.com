Rechnungskonto einrichten
  1. Öffnen Sie App Catalog .
  2. Sie haben folgende Möglichkeiten:
    • Tippen Sie auf die Anwendung, die Sie kaufen möchten > Weiter.
    • Tippen Sie auf Menü (oben links) > Einstellungen und Konten > Konto hinzufügen.
  3. Geben Sie das Kennwort für das Palm-Profil ein > Weiter.

    4. Wenn Sie das Kennwort für das Palm-Profil dreimal falsch eingeben, wird eine Nachricht mit Informationen darüber angezeigt, wie Sie den Kundendienst kontaktieren können.


  4. Tippen Sie auf einen Kreditkartentyp > Geben Sie die Daten der Kreditkarte und eine Rechnungsadresse ein > Absenden.
    5. WICHTIGWICHTIG:

    Die Daten der Kreditkarte werden nicht auf dem Telefon gespeichert. Sie werden verschlüsselt und in Ihrem Palm-Profil gespeichert.

    Important Close
  5. Tippen Sie auf OK, um Ihre E-Mail-Adresse zu bestätigen.

    6. Rechnungen werden an diese Adresse gesendet. Tippen Sie auf Diese Adresse ändern, wenn Sie die E-Mail-Adresse ändern möchten.


  6. Wählen Sie im Abschnitt Kennwort erforderlich eine der folgenden Optionen:
    • Alle vier Stunden: Sie müssen Ihr Kennwort zunächst beim Kauf eingeben. Wenn Sie innerhalb der nächsten vier Stunden einen weiteren Einkauf tätigen, müssen Sie das Kennwort nicht erneut eingeben. Nach vier Stunden müssen Sie das Kennwort erneut eingeben, um dann wieder vier Stunden ohne weitere Eingabe einkaufen zu können.
    • Jeder Kauf: Sie müssen Ihr Kennwort bei jedem Kauf eingeben.
