Alben löschen
  1. Schließen Sie Ihr Smartphone mit dem USB-Kabel an Ihren Computer an > USB-Laufwerk.
  2. Tippen Sie auf OK, wenn Sie dazu aufgefordert werden.
  3. Öffnen Sie auf dem Computer das Laufwerk, das dem Smartphone entspricht.
  4. Löschen Sie mithilfe der Steuerelemente Ihres Computers den Ordner, der dem Album entspricht.
  5. Verbindung sicher trennen: Falls Sie zum Trennen der Verbindung nicht auf Auswerfen klicken, wird Ihr Smartphone zurückgesetzt, wenn Sie das USB-Kabel abziehen:

    Auf einem Windows-Computer müssen Sie mit der rechten Maustaste auf das Laufwerk klicken, das Ihrem Smartphone entspricht > Auswerfen.

    Wenn Sie einen Mac-Computer verwenden: Ziehen Sie das Laufwerk, das Ihrem Smartphone entspricht, in den Papierkorb. Anstelle von „Papierkorb“ wird Auswerfen angezeigt.

  6. Ziehen Sie das USB-Kabel vom Computer ab, wenn der Bildschirm USB-Laufwerk nicht mehr auf dem Smartphone angezeigt wird.
