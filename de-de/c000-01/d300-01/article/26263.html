Erste Schritte: Kalender
  1. Öffnen Sie Kalender .
  2. Sie haben folgende Möglichkeiten:
    • Ein Online-Konto zum Synchronisieren einrichten: Tippen Sie auf Konto hinzufügen > Fahren Sie mit Schritt 3 fort.
    • Kalender öffnen, ohne ein Online-Konto zu erstellen: Tippen Sie auf Fertig. Dadurch wird Ihr Palm-Profilkonto geöffnet.
  3. Tippen Sie zum Einrichten eines aufgeführten Kontos auf den Kontotyp > Geben Sie die Kontoinformationen ein > Tippen Sie auf Anmelden > Fertig.

    4. Wenn Sie ein Exchange-Konto einrichten, müssen Sie ein Kennwort oder eine PIN zum Entsperren des Telefons eingeben, um Ihr Exchange-Konto zu schützen. Tippen Sie auf Kennwort festlegen oder PIN festlegen > Geben Sie ein Kennwort oder eine PIN ein > Fertig.

  4. Wenn Sie einen nicht aufgeführten Kontotyp einrichten möchten, tippen Sie auf Weitere suchen, tippen Sie auf eine Anwendung und befolgen Sie die Anweisungen auf dem Bildschirm, um das Konto hinzuzufügen.
