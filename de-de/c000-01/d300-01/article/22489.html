Erweiterte Kontoeinstellungen eingeben

Diese Einstellungen gelten nur für das Konto, das Sie in Schritt 2 ausgewählt haben.

  1. Öffnen Sie E-Mail .
  2. Menü (oben links) > Einstellungen und Konten > Tippen Sie auf einen Kontonamen.
  3. Sie können folgende Optionen festlegen:
    • Kontoname: Der Name, der in der Kontoliste angezeigt wird.
    • Vor- und Nachname: Der Name, der im Feld „Von“ angezeigt werden soll, wenn Sie Nachrichten versenden.
    • Symbol anzeigen: Legen Sie fest, ob beim Eingang einer neuen Nachricht eine Benachrichtigung auf dem Bildschirm angezeigt werden soll.
    • Benachrichtigung: Wenn „Symbol anzeigen“ aktiviert ist, legen Sie fest, ob ein Ton abgespielt werden soll, wenn neue Nachrichten eingegangen sind (Weitere Informationen).
    • Signatur: Tippen Sie hier, um eine Signatur zu erstellen, die in alle ausgehenden Nachrichten eingefügt wird (weitere Informationen).
    • Antwortadresse: Adresse, die Empfängern Ihrer Nachrichten als Antwortadresse angezeigt und als solche verwendet wird.
    • E-Mail anzeigen: Legen Sie fest, wie viele E-Mails vom Server abgerufen werden sollen (Zeitraum in Tagen).
    • E-Mail abrufen: Legen Sie fest, wie häufig E-Mails für dieses Konto synchronisiert werden sollen.
    • Gelöschte E-Mails synchronisieren: Diese Einstellung gilt nur für POP-Konten. Legen Sie fest, ob Nachrichten vom Server gelöscht werden sollen, wenn Sie diese auf dem Smartphone löschen.
    • Server mit Gerät synchronisieren: Nur bei POP-Konten: Nachrichten werden auf Ihrem Smartphone gelöscht, wenn sie auf dem Server gelöscht werden.
    • Standardordner: Diese Einstellung gilt nur für IMAP-Konten. Legen Sie fest, in welchen Ordnern gesendete E-Mails, Entwürfe und gelöschte E-Mails gespeichert werden sollen.
    • Anmeldeeinstellungen ändern: Weitere Informationen.
    • Konto entfernen: Weitere Informationen.
