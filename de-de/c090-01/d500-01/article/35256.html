SMS erstellen und senden

Bevor Sie beginnen

  • Verbinden Sie Ihr Gerät mit einem webOS-Smartphone (weitere Informationen).
  • Bevor Sie mit Ihrem Gerät SMS senden oder empfangen, erkundigen Sie sich, ob dieser Nachrichtentyp Bestandteil des Mobilfunkvertrags für Ihr Smartphone ist und welche Preise dafür gelten.
  • Stellen Sie sicher, dass das Smartphone eingeschaltet ist, mit dem Ihr Gerät verbunden ist, und dass Sie sich im Empfangsbereich des Netzes befinden. Weitere Informationen zum Einschalten drahtloser Dienste finden Sie in der Dokumentation, die sich im Lieferumfang Ihres Smartphones befand.

  1. Öffnen Sie Messaging .
  2. Tippen Sie auf Neue Unterhaltung > Tippen Sie in das Feld An > Legen Sie die Empfängeradresse mit einer der folgenden Methoden fest:
    • Geben Sie den Vor- oder Nachnamen eines Kontakts, seine Initialen, seinen Anzeigenamen, seine Telefonnummer oder seine E-Mail-Adresse ein > Tippen Sie auf die Telefonnummer, wenn sie angezeigt wird.
    • Tippen Sie auf , um Ihre vollständige Kontaktliste zu öffnen > Tippen Sie auf den gewünschten Kontakt oder geben Sie einen Namen bzw. eine Adresse ein.
    • Geben Sie für einen Empfänger, bei dem es sich nicht um einen Kontakt handelt, eine Telefonnummer ein.
  3. (Optional) Tippen Sie in das Feld An und wiederholen Sie Schritt 2, um weitere Empfänger einzugeben. Tippen Sie in das Feld An, um weitere Empfänger einzugeben.
  4. Tippen Sie auf Nachricht hier eingeben > Geben Sie den Nachrichtentext ein > Tippen Sie auf die Eingabetaste. Sie können im Nachrichtentext auch Emoticons verwenden (Weitere Informationen).

In der Kartenansicht können Sie auch eine SMS-Nachricht schreiben. Tippen Sie auf Einfach lostippen > Beginnen Sie mit dem Eingeben einer Nachricht. Tippen Sie unter „Schnellaktionen“ auf Neue Nachricht. Die Anwendung „Messaging“ wird mit einer neuen Nachricht geöffnet, die den eingegebenen Text als Nachrichtentext enthält.
