Standardanwendungen ändern
  1. Öffnen ! Geräteinfos  > Menü (oben links) > Standardanwendungen.
  2. Achten Sie beim Suchen nach Daten- und Dateitypen, für die mehrere Anwendungen verfügbar sind, auf Anwendungsnamen, die in schwarzer Schrift angezeigt werden.
  3. Sie haben folgende Möglichkeiten:
    • Standardanwendung für verschiedene Funktionen ändern: Tippen Sie unter Datentypen auf die aktuell angezeigte Anwendung für einen bestimmten Datentyp, und tippen Sie anschließend auf eine andere Anwendung.
    • Standardanwendung für verschiedene Dateitypen ändern: Tippen Sie unter Dateitypen auf die aktuell angezeigte Anwendung für einen bestimmten Dateityp, und tippen Sie anschließend auf eine andere Anwendung.

Wenn Sie mehrere Anwendungen für die gleiche Funktion auf Ihrem Gerät installiert haben, können Sie die Standardanwendung für diese Funktion ändern.

Ebenso können Sie, wenn Sie mehr als eine Anwendung auf Ihrem Gerät installiert haben, mit der ein bestimmter Dateityp geöffnet werden kann, die Standardanwendung zum Öffnen dieses Dateityps ändern.
