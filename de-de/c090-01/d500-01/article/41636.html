Nachrichten anzeigen, senden und beantworten
  1. Öffnen Sie ! Facebook Tablet  > Nachrichten.
  2. Sie haben folgende Möglichkeiten:
    • Nachricht an einen Facebook-Freund senden: Tippen Sie auf > Tippen Sie in das Feld An > Geben Sie den Vor- oder Nachnamen eines Freundes ein > Tippen Sie auf einen Freund. Tippen Sie auf Nachricht eingeben > Geben Sie eine Nachricht ein > Tippen Sie auf Senden.
    • Auf eine Nachricht antworten: Tippen Sie auf eine Nachricht > > Geben Sie eine Nachricht ein > Senden.
    • Gesendete Nachrichten anzeigen: Tippen Sie auf Gesendet.
    • Nachrichten aus Facebook und Facebook-Anwendungen anzeigen: Tippen Sie auf Aktualisierungen.
    • Posteingang anzeigen: Tippen Sie auf Posteingang.
