Das Gerät mit einem HP webOS-Smartphone verbinden

 Bevor Sie beginnen

Das webOS-Smartphone, mit dem Sie eine Verbindung herstellen, muss erkennbar sein. Lesen Sie in der Dokumentation des Smartphones nach, wie Sie es erkennbar machen können.

BYB Close.
  1. Öffnen Sie Telefon .
  2. Sie haben folgende Möglichkeiten:
    • Falls die Option „Telefon verbinden“ angezeigt wird: Tippen Sie auf Telefon verbinden.
    • Falls die Option nicht angezeigt wird: Menü (oben links) > Einstellungen und Konten > Telefon verbinden unten auf dem Bildschirm.
  3. Wenn Bluetooth deaktiviert ist, tippen Sie auf Aus, um Bluetooth zu aktivieren.
  4. Sie haben folgende Möglichkeiten:
    • Falls das Smartphone, mit dem Sie eine Verbindung herstellen möchten, in der Liste „Geräte“ aufgeführt ist, tippen Sie darauf.
    • Wenn das zu verbindende Smartphone in der Liste „Geräte“ nicht angezeigt wird, tippen Sie auf Gerät hinzufügen > Typ > WebOS-Telefon. Wenn das Smartphone, mit dem Sie eine Verbindung herstellen möchten, gefunden wurde, tippen Sie darauf.
  5. Tippen Sie auf dem Smartphone auf Ja, zulassen, um den Pairing-Vorgang abzuschließen.
  6. Wenn Sie dazu aufgefordert werden, tippen Sie auf dem Smartphone auf Zulassen, um die Nachrichtenfreigabe zwischen Smartphone und Gerät zuzulassen. Diese Funktion wird nur von bestimmten HP webOS-Smartphones unterstützt.

HINWEIS: Sie können auf dem Gerät keine MMS senden oder empfangen. Wenn Sie eine MMS auf einem verbundenen Smartphone erhalten, wird auf dem Gerät eine Benachrichtigung angezeigt. Sie müssen die Nachricht jedoch auf dem Smartphone anzeigen.
