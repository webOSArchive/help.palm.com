E-Mails empfangen und lesen
  1. Öffnen Sie E-Mail .
  2. Wenn die Ordner für das gewünschte Konto ausgeblendet sind, tippen Sie im linken Fenster auf den Kontonamen, um die Ordner einzublenden.
  3. Tippen Sie auf den Ordner mit den Nachrichten, die Sie abrufen oder lesen möchten. Falls die Synchronisierung nicht startet, tippen Sie auf .
  4. Tippen Sie im mittleren Bereich auf eine Nachricht, um diese zu öffnen.
  5. (Optional) Wenn Sie eine Nachricht auf den Vollbildmodus vergrößern möchten, tippen Sie im rechten Bereich auf .
  6. (Optional) Sie haben folgende Möglichkeiten:
    • Nummer anrufen: Tippen Sie auf die Nummer > . Um Anrufe tätigen zu können, muss Ihr Gerät mit einem webOS-Smartphone verbunden sein (weitere Informationen).
    • Eine E-Mail an eine E-Mail-Adresse senden: Tippen Sie auf die E-Mail-Adresse > Geben Sie Ihre Nachricht ein > Tippen Sie auf Senden.
    • Eine Internetadresse öffnen: Tippen Sie auf die Adresse.
    • Den Text einer E-Mail kopieren und in ein Memo oder anderes Dokument einfügen: Tippen Sie auf den Nachrichtentext > Menü (oben links) > Bearbeiten > Alle auswählen. Tippen Sie auf den markierten Text > Tippen Sie auf Kopieren.

Die E-Mail-Anwendung synchronisiert die Nachrichten jedes Mal, wenn Sie einen Mailordner öffnen. Außerdem können Sie einen Zeitplan für das Synchronisieren festlegen (weitere Informationen). Nachrichten synchronisieren, wenn bereits ein Ordner geöffnet ist: Tippen Sie auf .
