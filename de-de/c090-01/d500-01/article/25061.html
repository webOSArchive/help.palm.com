Mit Unterhaltungen arbeiten
  1. Öffnen Sie Messaging .
  2. Sie haben folgende Möglichkeiten:
    • Eine neue Unterhaltung beginnen: Erstellen Sie eine Nachricht oder tippen Sie auf eine Nachricht, um sie zu beantworten.
    • Eine bestehende Unterhaltung fortführen: Tippen Sie auf die Unterhaltung.
  3. Geben Sie die Nachricht ein > Tippen Sie auf die Eingabetaste.

Wenn Sie mehr als eine Nachricht mit einem Gesprächspartner austauschen, werden diese Nachrichten in einer Unterhaltung gruppiert. Eine Unterhaltung kann SMS- und Sofortnachrichten enthalten. Wenn Sie eine Unterhaltung beginnen oder fortfahren, werden im oberen Teil des Bildschirms alle Nachrichten angezeigt, die Sie mit diesem Gesprächspartner ausgetauscht haben. Im unteren Bereich geben Sie Ihre nächste Nachricht ein.

Zum Senden und Empfangen von SMS muss Ihr Gerät mit einem kompatiblen webOS-Smartphone verbunden sein (weitere Informationen).

Sie können mehrere Unterhaltungen gleichzeitig führen.
