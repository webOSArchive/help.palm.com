Dateien vom Web herunterladen
  1. Öffnen Sie ! Web .
  2. Gehen Sie zu der Webseite, auf der sich die Datei befindet, die Sie herunterladen möchten.
  3. Tippen Sie auf den Dateinamen.
  4. Wenn eine Anwendung auf Ihrem Gerät den Dateityp unterstützt, wird der Dateiname im Fenster „Downloads“ angezeigt. Tippen Sie auf Öffnen.

Alle Dateien werden in den USB-Speicher des Geräts heruntergeladen. Wenn eine Anwendung auf Ihrem Gerät den Dateityp unterstützt, können Sie die Datei außerdem in dieser Anwendung finden. Sie können beispielsweise eine PDF-Datei in Adobe Reader und eine MP3-Datei in der Musikanwendung finden.
