Facebook Tablet erstmalig verwenden

Wenn Sie auf Ihrem Gerät noch kein Facebook-Konto eingerichtet haben, müssen Sie eines einrichten, wenn die Anwendung Facebook Tablet erstmals geöffnet wird.

  1. Öffnen Sie ! Facebook Tablet .
  2. (Optional) Sie haben folgende Möglichkeiten:
    • Ändern, wie häufig die Anwendung Facebook Tablet nach neuen Benachrichtigungen sucht: Tippen Sie Häufigkeit > Tippen Sie auf eine Häufigkeit.
    • HP webOS-Benachrichtigungen aktivieren oder deaktivieren: Tippen Sie auf Ein oder Aus. Wenn diese Einstellung eingeschaltet ist, werden neue Facebook-Benachrichtigungen auf dem Dashboard angezeigt.
  3. Tippen Sie auf Konto hinzufügen > Geben Sie die Kontodaten ein > Anmelden.
  4. Es werden Optionen dazu angezeigt, welche Anwendungen Daten von diesem Konto herunterladen sollen. Tippen Sie für jede Anwendung auf Ein oder Aus > Konto erstellen.
  5. Tippen Sie auf Fertig.
