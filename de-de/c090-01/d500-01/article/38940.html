E-Mail-Nachrichten erstellen und senden
  1. Öffnen Sie E-Mail .
  2. Wenn Sie noch kein E-Mail-Konto eingerichtet haben, müssen Sie zuerst ein E-Mail-Konto einrichten (Weitere Informationen).
  3. Tippen Sie im Nachrichtenlistenfenster am unteren Bildschirmrand auf .
  4. (Optional) Tippen Sie auf VON, um das E-Mail-Konto zu ändern, das Sie zum Senden von E-Mails verwenden. Diese Option wird nur angezeigt, wenn mehrere E-Mail-Konten vorhanden sind.
  5. Tippen Sie auf das Feld An und gehen Sie wie folgt vor:
    • Geben Sie einen Kontaktnamen, Initialen oder eine E-Mail-Adresse ein > Tippen Sie auf die Adresse.
    • Tippen Sie auf , um Ihre Kontakte anzuzeigen > Tippen Sie auf den gewünschten Kontakt oder geben Sie einen Namen bzw. eine Adresse ein > Tippen Sie auf die Adresse.
    • Geben Sie die vollständige E-Mail-Adresse eines Empfängers ein, bei dem es sich nicht um einen Kontakt handelt.
  6. (Optional) Tippen Sie auf An, um die Cc- und Bcc-Felder zu öffnen > Tippen Sie auf eines der Felder und geben Sie eine Adresse ein.
  7. Wiederholen Sie die Schritte 5 und 6, um zusätzliche Empfänger einzugeben.

    Wenn Sie mehrere Empfänger hinzufügen und Sie einen Empfänger bearbeiten möchten, tippen Sie auf die zu ändernde E-Mail-Adresse, und geben Sie die neue Adresse ein.

  8. Tippen Sie auf das Feld Betreff > Geben Sie den Betreff ein > Tippen Sie auf die Eingabetaste > Geben Sie den Nachrichtentext ein.
  9. (Optional) Um die Priorität der Nachricht festzulegen, tippen Sie auf > Wählen Sie eine Prioritätsstufe.
  10. (Optional) Nachricht verwerfen: Tippen Sie auf > Nachricht verwerfen.
  11. Tippen Sie auf Senden.

Wenn Sie ein E-Mail-Konto eingerichtet haben, können Sie auch über die Kartenansicht mit dem Verfassen einer E-Mail beginnen. Tippen Sie auf Einfach lostippen und beginnen Sie mit der Eingabe Ihrer Nachricht. Rollen Sie in der Liste der Suchoptionen nach unten und tippen Sie auf Neue E-Mail. Bei einer neuen E-Mail-Nachricht wird die E-Mail-Anwendung geöffnet und der von Ihnen eingegebene Nachrichtentext wird angezeigt.
