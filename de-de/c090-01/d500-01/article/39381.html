E-Mail einrichten: Microsoft Exchange und Lotus Notes

Bevor Sie beginnen

Erfragen Sie die folgenden Informationen vom E-Mail-Anbieter oder Systemadministrator:

  • URL des Mailservers zum Empfangen von E-Mails.
  • Domain-Name
  • Benutzername und Kennwort

BYB Close

  1. Öffnen Sie E-Mail .
  2. Sie haben folgende Möglichkeiten:
    • Falls Sie in einer anderen Anwendung noch kein Konto eingerichtet haben, das die Synergy-Funktion nutzt, tippen Sie auf einen Kontotyp oder auf Microsoft Exchange.
    • Wenn Sie bereits ein Konto eingerichtet haben, das die Synergy-Funktion nutzt, und ein weiteres einrichten möchten, tippen Sie auf Konto hinzufügen > Microsoft Exchange.
    • Sie können auch jederzeit Folgendes tun: Tippen Sie unter „E-Mail“ auf Menü (oben links) > Einstellungen und Konten > Konto hinzufügen > Microsoft Exchange.
  3. Geben Sie Ihre E-Mail-Adresse und Ihr Kennwort ein > Manuelle Einrichtung.
  4. Geben Sie bei Bedarf basierend auf den Server-Informationen, die Sie vom E-Mail-Anbieter oder Systemadministrator erhalten, Informationen für den Maileingangsserver ein (Informationen finden Sie oben unter „Bevor Sie beginnen“). Sie können unter „Posteingangsserver“ im Feld Domäne den Namen oder die IP-Adresse des Posteingangsservers eintragen.
  5. Tippen Sie auf Anmelden.
  6. Um Ihr Exchange-Konto zu schützen, müssen Sie möglicherweise ein Kennwort oder eine PIN zum Entsperren des Geräts festlegen. Tippen Sie auf PIN festlegen oder Kennwort festlegen > Geben Sie eine PIN oder ein Kennwort ein > Tippen Sie auf Fertig.
  7. Tippen Sie für Anwendungen, in die Sie Daten aus diesem Konto herunterladen möchten, auf Ein/Aus > Konto erstellen > Los.
