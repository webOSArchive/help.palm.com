PDF-Datei drucken
  1. Öffnen Sie die zu druckende Datei > (rechts oben).
  2. Sie haben folgende Möglichkeiten:
    • Wenn eine Liste von Druckern angezeigt wird, tippen Sie auf den gewünschten.
    • Wenn Sie bereits einen Drucker in diesem Netzwerk verwendet haben: Der zuletzt verwendete Drucker wird gespeichert und ausgewählt. Wenn Sie einen anderen Drucker auswählen möchten, navigieren Sie nach unten und tippen Sie auf Anderen Drucker auswählen > einen Drucker. 
    • Drucker manuell oder in einem Unternehmensnetzwerk einrichten: Navigieren Sie nach unten und tippen Sie auf Drucker hinzufügen. Tippen Sie auf Geben Sie eine IP-Adresse ein > Geben Sie die IP-Adresse ein. (Optional) Tippen Sie auf Diesen Drucker benennen > Geben Sie einen Namen ein. Tippen Sie auf Fertig.
  3. (Optional) In Abhängigkeit des verwendeten Druckers und des Dateityps können Sie die folgenden Druckerattribute festlegen:

    Anzahl Kopien: Tippen Sie auf 1 und tippen Sie auf die Anzahl der Kopien, die Sie drucken möchten.

    Beidseitiger Druck: Tippen Sie auf Aus, um den beidseitigen Druck von „Aus“ in „Ein“ umzuschalten.

    Farbdruck: Tippen Sie auf Ein, um den Farbdruck von „Ein“ in „Aus“ zu ändern.

    Seitenbereich: Tippen Sie auf den aktuell angezeigten Seitenbereich und geben Sie den gewünschten Bereich ein.

    Druckqualität: Tippen Sie auf die aktuell angezeigte Option und tippen Sie auf die gewünschte Qualität (Entwurf, Normal oder Optimal).

  4. Tippen Sie auf Drucken.

Um einen Druckauftrag abzubrechen, tippen Sie auf Abbrechen, während die Datei zum Drucken vorbereitet wird oder der Druckvorgang läuft, tippen Sie auf > die Benachrichtigung > rechts neben dem Druckauftrag.

Sie können eine Datei auf einem HP-Drucker drucken, der mit einem WLAN-Netzwerk verbunden ist. Informationen zum Einrichten eines Druckers in einem WLAN-Netzwerk finden Sie in den Anweisungen, die im Lieferumfang des Druckers enthalten waren.
