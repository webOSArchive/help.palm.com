Webseite drucken
  1. Öffnen Sie die zu druckende Internetseite und warten Sie, bis die Statusleiste am oberen Bildschirmrand vollständig blau ist und nicht mehr angezeigt wird > Menü (oben links) > Tippen Sie auf Drucken.

    Alle Drucker, die in Ihrem aktuellen WLAN-Netzwerk verfügbar sind, werden angezeigt.

  2. Sie haben folgende Möglichkeiten:
    • Wenn Sie das erste Mal einen Drucker in diesem Netzwerk verwenden: Wenn eine Liste mit Druckern angezeigt wird, tippen Sie auf einen Drucker.
    • Wenn Sie bereits einen Drucker in diesem Netzwerk verwendet haben: Der zuletzt verwendete Drucker wird gespeichert und ausgewählt. Wenn Sie einen anderen Drucker auswählen möchten, navigieren Sie nach unten und tippen Sie auf Anderen Drucker auswählen > Tippen Sie auf einen Drucker.
    • Drucker manuell oder in einem Unternehmensnetzwerk einrichten: Zum Hinzufügen eines Druckers navigieren Sie nach unten und tippen Sie auf Drucker hinzufügen. Tippen Sie auf Geben Sie eine IP-Adresse ein > Geben Sie die IP-Adresse des Druckers ein. (Optional) Tippen Sie auf Diesen Drucker benennen > Geben Sie einen Namen ein, um den Drucker zu identifizieren. Tippen Sie auf Fertig.
  3. (Optional) In Abhängigkeit des verwendeten Druckers können Sie die folgenden Druckerattribute festlegen:

    Anzahl Kopien: Tippen Sie auf 1 und tippen Sie auf die Anzahl der Kopien, die Sie drucken möchten.

    Farbdruck: Tippen Sie auf Ein, um den Farbdruck von „Ein“ in „Aus“ zu ändern.

  4. Tippen Sie auf Drucken.

    Wenn Sie einen Druckauftrag abbrechen möchten, tippen Sie auf Abbrechen, während die Datei zum Drucken vorbereitet wird. Falls der Druckvorgang bereits ausgeführt wird, tippen Sie oben auf der Seite auf > Tippen Sie auf die Benachrichtigung > Tippen Sie rechts vom Druckauftrag in der Druckwarteschlange auf .
