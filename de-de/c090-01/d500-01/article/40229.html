Einstellungen des HP webOS-Kontos ändern
  1. Öffnen Sie Konten > Tippen Sie auf den Namen Ihres HP webOS-Kontos.
  2. Geben Sie das Kennwort Ihres HP webOS-Kontos ein > Tippen Sie auf Anmelden.
  3. Bearbeiten Sie die folgenden Elemente:

    Name: Ändern Sie den Vornamen bzw. Nachnamen, der Ihrem HP webOS-Konto zugeordnet ist.

    E-Mail: Ändern Sie die E-Mail-Adresse, die Ihrem HP webOS-Konto zugeordnet ist. An die neue Adresse wird eine Bestätigungs-E-Mail gesendet. Klicken Sie auf die Verknüpfung in dieser E-Mail, um die neue Adresse zu bestätigen.

    Kennwort: Tippen Sie auf Kennwort ändern > Geben Sie das neue Kennwort ein > Geben Sie das neue Kennwort erneut ein > Fertig.

    Sicherheitsfrage/Antwort: Tippen Sie auf die aktuelle Frage > Tippen Sie auf eine neue > Antwort > Geben Sie eine Antwort ein.

  4. Wenn Sie die Informationen zu einem Gerät anzeigen möchten, das diesem Konto zugewiesen ist, tippen Sie unter Geräte auf das Gerät. Wenn Sie die Informationen nicht mehr anzeigen möchten, tippen Sie auf Fertig.
