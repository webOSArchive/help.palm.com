Uso de Calendario por primera vez
  1. Abre ! Calendario .
  2. Realiza una de las siguientes acciones:
    • Para configurar una cuenta en línea para que se sincronice: toca Añadir una cuenta y ve al paso 3.
    • Para abrir Calendario sin crear una cuenta en línea o para utilizar cuentas ya configuradas en otra aplicación: toca Aceptar. De este modo se abre la cuenta de HP webOS y cualquier otra cuenta en línea y puedes empezar a crear eventos del calendario en esas cuentas.
  3. Para configurar una cuenta que aparezca en la lista, toca el tipo de cuenta y avanza hasta el paso 4. Para configurar un tipo de cuenta que no aparezca en la lista, toca Buscar más para buscar una aplicación y sigue las instrucciones que aparecen en pantalla.
  4. Introduce la información de la cuenta > toca Iniciar sesión.
  5. Si configuras una cuenta de Exchange, para mantener su seguridad, puede que se te solicite que introduzcas una contraseña o PIN para desbloquear el smartphone. Toca Establecer contraseña o Establecer PIN > introduce una contraseña o PIN > Aceptar.
  6. Según la cuenta, es posible que se te proporcionen opciones para las cuales las aplicaciones deban descargar datos de esa cuenta. Para cada aplicación, toca Activo o Inactivo > Crear.
