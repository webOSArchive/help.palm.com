Chat con un grupo
  1. Abre Mensajería .
  2. Realiza una de las siguientes acciones:
    • Para iniciar un nuevo chat de grupo: en Conversaciones o Amigos, toca > introduce dos o más amigos de Skype o toca para buscar amigos de Skype. Introduce un mensaje para crear el tema inicial del chat > toca para configurar el chat. En la vista Conversaciones, toca el nuevo chat que has creado.
    • Para continuar con un chat de grupo existente: en la vista Conversaciones, toca la conversación.
  3. Introduce un mensaje y toca .
  4. Cuando los destinatarios del mensaje respondan, la respuesta se mostrará debajo de tu mensaje en la conversación. Repite el paso 3 para responder a todo el grupo.
  5. (Optativo) Realiza una de las siguientes acciones:
    • Para añadir o cambiar un tema del chat: toca la conversación > Menú (parte superior izquierda) > Establecer tema. Introduce o edita el tema > Establecer tema.
    • Para añadir otro amigo a un chat: toca la conversación > Menú (parte superior izquierda) > Añadir amigo a este chat. Introduce el nombre de usuario Skype del amigo o toca para buscar un nombre de Skype > toca .
    • Para ver todos los participantes de un chat de grupo: toca el encabezado de la conversación. Toca un participante del chat para ver sus detalles de contacto.
    • Para abandonar un chat: toca la conversación > Menú (parte superior izquierda) > Abandonar chat.

Todos los miembros de un chat de grupo deben ser amigos de Skype.
