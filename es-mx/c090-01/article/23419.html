Borrado de datos y restablecimiento del teléfono

IMPORTANTEIMPORTANTE:

Realiza una copia de seguridad manual (consulta cómo) de los datos antes del restablecimiento del teléfono. Important Close

Abre Información del dispositivo  > Opciones de Restablecer.

  • Para borrar parcialmente datos del teléfono:

    • Toca Borrar aplicaciones y datos para borrar aplicaciones que hayas instalado y toda la configuración de aplicaciones y datos.

    Toca Borrar unidad USB para borrar archivos personales guardados en la unidad USB, incluidos los archivos transferidos desde iTunes mediante la sincronización de medios. Al borrar la unidad USB no se eliminan las aplicaciones que hayas instalado.

  • Para borrar todos los datos del teléfono:

    • Toca Borrado completo para borrar las aplicaciones que hayas instalado, toda la configuración de aplicaciones y datos, además de todos los archivos guardados en la unidad USB. (También puedes mantener presionada la tecla Sym  + la tecla Opción  + encendido para realizar un borrado completo.)

    Tras un borrado completo, inicia sesión en tu perfil de Palm. De este modo se restauran los datos del perfil de Palm en el teléfono. Inicia sesión en tus cuentas en línea para restaurar los datos. Los archivos personales guardados en la unidad USB, como fotos, videos y música, no pueden restaurarse. También puedes iniciar sesión en el perfil de Palm en un teléfono nuevo y restaurar los datos en él.

    Toca Borrado completo > Borrado completo seguro para borrar todas las aplicaciones, la configuración de todas las aplicaciones y datos y todos los archivos guardados en la unidad USB. Selecciona esta opción para borrar por completo el teléfono, de modo que no se puedan recuperar los datos (por ejemplo, si vendes el teléfono).
