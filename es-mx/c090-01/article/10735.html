Compra de una aplicación
  1. Abre App Catalog .
  2. Navega hasta la aplicación que desees > toca el nombre de la aplicación > Descargar por [cantidad].
  3. Si la aplicación utiliza Servicios de ubicación, se mostrará un aviso. Toca Continuar.
  4. Realiza una de las siguientes acciones:
    • Para comprar la aplicación: si se requiere, introduce la contraseña de tu perfil de Palm para autorizar la compra > Continuar.
    • Para configurar una cuenta de facturación: toca Continuar > introduce la contraseña del perfil de Palm > Continuar > configura la cuenta de facturación (consulta cómo).

      • Si introduces una contraseña del perfil de Palm incorrecta tres veces, aparecerá un mensaje que indica cómo contactar con el servicio de atención al cliente para obtener ayuda.


  5. Toca OK.
  6. Para abrir una aplicación descargada, toca Tocar para iniciar. El icono de la aplicación también aparece en la primera página del Iniciador, por lo que también puede abrirse desde allí.
