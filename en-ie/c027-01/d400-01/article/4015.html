Connect to a new secure Wi-Fi network
  1. Open ! Wi-Fi .
  2. Make sure Wi-Fi is on (learn how) and do one of the following:
    • If the network you want is displayed: Tap the network name > enter your Username and/or Password > Sign In. That's it.
    • If the network you want is not displayed: Tap Join network > enter the network name.
  3. Tap the Security field > tap the network type: Open, WPA-personal, WEP or Enterprise.
  4. Tap Connect.
  5. A series of different screens appears depending on the network type. Enter or select the following if you are prompted to do so: Username, Password and/or Certificate.
  6. Tap Sign In.
