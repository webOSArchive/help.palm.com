Add a Cisco AnyConnect VPN profile

Before You Begin

Get this info from your email provider or system administrator:

  • VPN server name
  • Username and password
  • Security token or other password

BYB Close

  1. Open ! VPN  > Add Profile.
  2. Tap the Connection Type field > Cisco AnyConnect.
  3. Tap Enter server name > enter the server name > Next.
  4. If required, tap Agree to accept the Cisco licence agreement. Tap  to view the full agreement.
  5. Enter your login information > Connect.
  6. If required, enter your security token or other password > Done > Continue.
