Envío y recepción de mensajes de IM

Antes de empezar

Asegúrate de que el smartphone está encendido y de que te encuentras dentro de un área de cobertura (consulta cómo). BYB Close

  1. Abre Mensajería .
  2. Realiza una de las siguientes acciones:
    • En la vista Amigos, toca el nombre de un amigo. Si hay más de una cuenta para ese amigo, se abrirá la cuenta de IM utilizada más recientemente. Para cambiar a otra cuenta de IM, toca el nombre de la cuenta de IM en la esquina superior derecha y, a continuación, toca una dirección de IM.
    • En la vista Conversaciones, toca una conversación existente.
    • En la vista Conversaciones o Amigos, toca  para comenzar un nuevo mensaje > introduce una dirección de IM o toca para buscar un contacto.
  3. Introduce el texto del mensaje > . Puede usar emoticonos en el mensaje (consulta cómo).
  4. Cuando el destinatario del mensaje responda, la respuesta se mostrará debajo de tu mensaje en la conversación. Repite el paso 3 si deseas enviar otro mensaje.
  5. (Optativo) Realiza una de las siguientes acciones en un mensaje recibido:
    • Para llamar a un número: toca el número > .
    • Para añadir un número a Contactos: toca y mantén presionado el número > toca Añadir a Contactos. Toca Guardar como nuevo para crear un contacto nuevo o bien, toca Añadir a existente para añadir este número a un contacto existente.
    • Para enviar un mensaje de correo electrónico: toca la dirección de correo electrónico.
    • Para acceder a una dirección Web: tócala.
