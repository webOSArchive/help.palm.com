Administración de aplicaciones descargadas
  1. Abre Administrador de software .
  2. Toca Aplicaciones para administrar las aplicaciones descargadas que aparecen en el Lanzador, o bien toca Otro para administrar las aplicaciones y los servicios en segundo plano que no aparecen en el Lanzador.
  3. Toca el nombre de una aplicación o servicio.
  4. En la pantalla de detalles de la aplicación, realiza una de las siguientes acciones:
    • Para leer las reseñas: toca Reseñar aplicación > toca una reseña.
    • Para añadir una reseña: toca Reseñar aplicación > Reseña.
    • Para compartir información sobre la aplicación: toca Reseñar aplicación > Enviar > Correo electrónico o Mensaje de texto.
    • Para informar de un problema con una aplicación: toca Informar de un problema > Falla > toca un tipo de problema. Introduce la descripción del problema > .
    • Para eliminar una aplicación: toca Eliminar > Eliminar.
    • Para volver a instalar una aplicación borrada: busca la aplicación (consulta cómo) > toca el nombre de la aplicación > Volver a descargar gratis. Si ya pagaste por la aplicación, no es necesario comprarla de nuevo.
