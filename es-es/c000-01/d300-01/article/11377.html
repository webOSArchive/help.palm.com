Actualización de una aplicación descargada desde una notificación

Cuando hay una actualización disponible para una aplicación que has descargado, recibes una notificación. Toca la notificación y realiza una de las siguientes acciones:

  • Para mostrar los detalles sobre la aplicación y la actualización disponible: toca el nombre de la aplicación.
  • Para actualizar una aplicación: toca el icono de flecha que aparece a la derecha de su nombre.
  • Para actualizar todas las aplicaciones: toca Instalar actualizaciones.

También puedes buscar actualizaciones de aplicaciones de forma manual en la aplicación Administrador de software (consulta cómo).
