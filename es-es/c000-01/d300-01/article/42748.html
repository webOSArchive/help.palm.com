Búsqueda de información

Con la función Escribe y listo, puedes buscar contactos y aplicaciones en el smartphone, buscar información dentro de una aplicación o buscar en la Web.

  1. Abre la vista Carta (consulta cómo) o el Lanzador (consulta cómo).
  2. Comienza a escribir el término de búsqueda.
  3. Realiza una de las siguientes acciones:
    • Para abrir una aplicación: toca el icono.
    • Para ponerte en contacto con alguien: toca un nombre para mostrar la información de contacto de esa persona. Toca una dirección de correo electrónico para comenzar un mensaje nuevo. Toca un número de teléfono para llamar a esa persona o toca para enviar un mensaje de texto.
    • Para buscar un contacto en la lista global de direcciones (GAL) de la empresa: toca Búsqueda global de dirección.
    • Para buscar en la Web: toca una opción de búsqueda en la Web.
    • Para buscar mensajes de correo electrónico que contengan el término introducido: toca Correo electrónico.
    • Para buscar favoritos Web o páginas visitadas que contengan el término introducido: toca Favoritos e historial.
    • Para mostrar un mapa de la ubicación introducida: toca Mapas.
    • Para buscar en HP webOS App Catalog aplicaciones que coincidan con el término introducido, toca HP App Catalog.
    • Para crear un nuevo mensaje, evento del calendario u otro tipo de elemento: toca Nuevo [elemento].

    Para buscar una aplicación, puedes introducir el nombre de la aplicación o una palabra clave (por ejemplo, Reuniones para Calendario).

    Si no se encuentran los contactos o aplicaciones, las opciones de búsqueda en la Web se muestran automáticamente.
