Personalización de la configuración del navegador Web
  1. Abre ! Web  > Menú (parte superior izquierda) > Preferencias.
  2. Configura cualquiera de las siguientes preferencias:

    Motor de búsqueda web predet.: toca para cambiar el motor de búsqueda predeterminado.

    Bloquear ventanas emergentes: impide a los sitios Web abrir nuevas ventanas del navegador. Toca Sí para cambiar de Sí a No.

    Aceptar cookies: permite las cookies de cualquier sitio Web que visites, incluso si otro sitio Web te ha redirigido al mismo. Toca Sí para cambiar de Sí a No. 

    JavaScript: si se desactiva Javascript, es posible que algunos sitios Web no funcionen correctamente. Toca Activo para cambiar el estado de activo a inactivo.

    Permitir contenido Flash: si se desactiva esta opción no se puede reproducir contenido Flash. Toca Activo para cambiar el estado de activo a inactivo.

    Cargar automáticamente contenido Flash: permite reproducir contenido Flash inmediatamente al abrir un sitio Web. Toca Inactivo para cambiar el estado de inactivo a activo.

    Borrar historial: toca para eliminar el historial de navegación.

    Borrar cookies: toca para eliminar las cookies utilizadas por algunos sitios Web para recordar el historial de visitas y la configuración de usuario.

    Borrar memoria caché: toca para eliminar cualquier contenido de páginas Web que se haya almacenado en la memoria caché local. Así se garantiza que estás viendo el contenido más reciente cuando visitas un sitio Web.
