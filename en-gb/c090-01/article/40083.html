Print a Word document or a PowerPoint presentation
  1. Open the file you want to print > (upper-right corner). If the icon is not displayed, tap the screen to display it.
  2. Do one of the following:
    • If a list of printers is displayed: Tap the one you want.
    • If you have already used a printer on this network: The last printer you used is remembered and is selected. To select another printer, scroll down and tap Select Another Printer > a printer. 
    • To set up a printer manually or on a corporate network: Scroll down and tap Add a Printer. Tap Enter Printer IP address > enter the IP address. (Optional) Tap Name Printer > enter a name. Tap Done.
  3. (Optional) Depending on the printer you are using and the file type, you can set the following printer attributes:

    Number of Copies: Tap 1 and tap the number of copies you want to print.

    2-Sided Printing: Tap Off to switch 2-sided printing from Off to On.

    Color Printing: Tap On to switch color printing from On to Off.

    Print Quality: Tap the currently displayed option, and tap the quality you want (Draft, Normal, or Best).

  4. Tap Print.

To cancel a print job, tap Cancel while the file is preparing to print, or while printing is in progress, tap > the notification >  to the right of the print job.

You can print a document or presentation on an HP printer that is connected to a wireless network. See the instructions that came with your printer for information about setting up a printer on a wireless network.
