Enter advanced account settings

These preferences apply only to the account you select.

  1. Open  Email .
  2. Menu (upper left) > Preferences & Accounts > tap an account.
  3. Tap to set any of the following:
    • Account Name: Enter the name that appears in the account list.
    • Full Name: Enter the name you want to appear in the From field for messages you send.
    • Show Icon: Set whether a notification icon appears onscreen when a new message arrives.
    • Alert: When Show Icon is on, set whether a sound plays when new messages arrive (learn how).
    • Signature: Tap to create a signature that's added to outgoing messages (learn how).
    • Reply-to Address: Enter the address you want recipients to see and reply to on your outgoing messages, if this is different from the email address you send the message from.
    • Show Email: Set how many days' worth of messages to retrieve from the server.
    • Get Email: Set how frequently to synchronize email for this account.
    • Sync deleted emails: For POP accounts only. Set whether messages should be deleted on the server when you delete them on your device.
    • Sync server to device: For POP accounts only. Set whether messages should be deleted on your device when they are deleted on the server.
    • Default Folders: For IMAP accounts only. Specify the folder where messages you send, save as drafts, or delete are stored.
    • Change Login Settings: Learn how.
    • Remove Account: Learn how.
  4. Tap Done > Done.
