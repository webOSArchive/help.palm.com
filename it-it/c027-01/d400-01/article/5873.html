Personalizzazione delle impostazioni del browser Web
  1. Aprire ! Web  > Menu (in alto a sinistra) > Preferenze.
  2. Impostare una delle seguenti opzioni:

    Motore di ricerca Web predefinito : toccare questa opzione per cambiare il motore di ricerca predefinito.

    Blocca popup : impedisce ai siti Web di aprire nuove finestre del browser. Toccare Sì per impostare No.

    Accetta cookie: consente di accettare i cookie dei siti Web visitati, anche se si viene reindirizzati a un sito da un altro sito. Toccare Sì per impostare No. 

    JavaScript: la disattivazione di JavaScript potrebbe impedire il corretto funzionamento di alcuni siti Web. Toccare On per impostare Off.

    Attiva contenuto Flash: la disattivazione di questa opzione impedisce la visualizzazione del contenuto Flash. Toccare On per impostare Off.

    Caricamento automatico contenuto Flash: attiva il contenuto Flash non appena si accede al sito Web. Toccare Off per impostare On.

    Cancella cronologia: toccare per eliminare la cronologia del browser.

    Cancella cookie: toccare per eliminare i cookie usati da alcuni siti Web per memorizzare la cronologia di visita e le impostazioni utente.

    Cancella cache: toccare per eliminare il contenuto delle pagine Web memorizzato nella cache locale. Questa operazione consente di visualizzare il contenuto più recente di una pagina Web.
