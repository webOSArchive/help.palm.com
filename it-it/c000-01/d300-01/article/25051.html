Aggiunta di una suoneria a un contatto
  1. Mentre si crea (istruzioni) o si modifica (istruzioni) un contatto > Imposta suoneria.
  2. Effettuare una delle seguenti operazioni:
    • Toccare il nome di una suoneria.
    • Per utilizzare un file musicale come suoneria: toccare > toccare il nome del file > toccare il nome file nell'elenco suonerie.
  3. (Facoltativo) Toccare per sentire l'anteprima della suoneria.
  4. Toccare Chiudi.
