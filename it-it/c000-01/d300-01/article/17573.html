Strumento per la risoluzione di problemi inerenti all'aggiornamento di software

Durante l'esecuzione di un aggiornamento wireless del software, esiste la possibilità, seppure infima, che il telefono si blocchi o non risponda più ai tocchi o alle pressioni dei tasti. Palm® webOS Doctor™ consente di ripristinare la funzionalità del telefono aggiornandone il sistema operativo e altro software di sistema alle ultime versioni. Dopo l'utilizzo di tale strumento, i dati di backup nel profilo Palm vengono automaticamente ripristinati nel telefono.

Per utilizzare Palm® webOS Doctor™ sono necessari:

  • Il cavo USB fornito con il telefono
  • Un computer desktop con:
    • Windows XP, Vista® o 7 oppure Mac OS 10.5
    • Una porta USB disponibile (non tramite un hub o una connessione USB secondaria)
    • L'ultima versione di Java installata (disponibile sul Web all'indirizzo java.com)
    • 200 MB di spazio libero su disco
  • Una connessione Internet in grado di supportare il download di file di grandi dimensioni (10 MB-200 MB)

Per scaricare lo strumento, visitare il sito Web all'indirizzo palm.com/support.
