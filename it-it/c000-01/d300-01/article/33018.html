Impostazione di account e-mail per Microsoft Exchange e Lotus Notes

Prima di iniziare

Richiedere le seguenti informazioni al provider del servizio e-mail utilizzato o all'amministratore del sistema:

  • Il nome del server di posta o l'indirizzo IP per la ricezione di posta
  • Nome del dominio del server
  • Il proprio nome utente e la password

BYB Close

  1. Aprire E-mail .
  2. Effettuare una delle seguenti operazioni:
    • Quando si apre l'applicazione E-mail per la prima volta: toccare Microsoft Exchange.
    • Se è già stato configurato un account che utilizza la funzionalità Synergy e si desidera configurarne un altro, toccare Aggiungi un account > Exchange.
    • In qualsiasi altro momento in cui si apre E-mail, toccare Menu (in alto a sinistra) > Preferenze e account > Aggiungi account > Microsoft Exchange.
  3. Inserire l'indirizzo e-mail e la password > Impostazione manuale.
  4. Inserire le informazioni per il server di posta in arrivo ricevute dal provider del servizio e-mail o dall'amministratore di sistema (vedere la sezione Prima di iniziare, all'inizio di questa procedura). È possibile inserire sia il nome del server sia l'indirizzo IP nel campo Server post in arrivo.
  5. Toccare Accedi.
  6. Per mantenere sicuro l'account Exchange, potrebbe essere necessario creare una password o un PIN per sbloccare il telefono. Toccare Imposta password o Imposta PIN > inserire la password o il PIN > Chiudi.
  7. Toccare On/Off per ogni applicazione in cui si intende scaricare dati dall'account.
  8. Toccare Crea > Chiudi.
