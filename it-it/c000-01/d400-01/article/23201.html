Modifica delle applicazioni predefinite
  1. Aprire Informazioni sul dispositivo  > Menu (in alto a sinistra) > Applicazioni predefinite.
  2. Per trovare i tipi di dati e i tipi di file per i quali è disponibile più di un'applicazione, cercare i nomi delle applicazioni visualizzati in carattere nero.
  3. Eseguire una o entrambe le seguenti operazioni:
    • Per cambiare l'applicazione predefinita da utilizzare per diverse funzioni: Tipi di dati > toccare l'applicazione attualmente visualizzata per un determinato tipo di dati > toccare un'applicazione diversa.
    • Per cambiare l'applicazione predefinita da utilizzare per diversi tipi di file: Tipi di file > toccare l'applicazione attualmente visualizzata per un determinato tipo di file > toccare un'applicazione diversa.

Per ignorare l'impostazione di applicazione predefinita quando si cerca un indirizzo su una cartina: tenere premuto Opzione , toccare l'indirizzo, quindi toccare un'applicazione.

Se nello smartphone sono presenti più applicazioni in grado di eseguire una determinata funzione, è possibile cambiare l'applicazione predefinita che viene utilizzata per la funzione in questione.

Inoltre, se nello smartphone sono presenti più applicazioni in grado di aprire un determinato tipo di file, è possibile cambiare l'applicazione predefinita che viene utilizzata per il file in questione.
