Trasferimento di file musicali allo smartphone

È possibile ascoltare file musicali copiati da un computer (istruzioni), ricevuti come allegati di e-mail (istruzioni), ricevuti tramite MMS (istruzioni), acquistati e scaricati dall'applicazione Amazon MP3, se disponibile sullo smartphone (istruzioni) o ottenuti mediante una sincronizzazione da HP Play sul computer (istruzioni).

I file musicali devono essere nei seguenti formati:

  • File ricevuti tramite messaggi MMS o acquistati da Amazon MP3: MP3.
  • File copiati da un computer: MP3, AAC, AAC+,WAV.
  • File ricevuti come allegati di e-mail: tutti i formati indicati sopra, nonché AMR.

È possibile scaricare e ascoltare file MP3, WAV e AMR da E-mail se allegati a messaggi di e-mail o da Messaggistica se contenuti in messaggi MMS. Tuttavia i file scaricati da E-mail o Messaggistica non vengono salvati nell'unità di memorizzazione USB e non sono elencati in Musica. Gli allegati dei messaggi MMS inviati allo smartphone in formato WAV o AMR vengono convertiti e copiati sullo smartphone come file MP3.
