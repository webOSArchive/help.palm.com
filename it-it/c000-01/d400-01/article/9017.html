Aggiunta di un profilo VPN Cisco AnyConnect

Prima di iniziare

Richiedere le seguenti informazioni al provider del servizio e-mail utilizzato o all'amministratore del sistema:

  • Nome server VPN
  • Nome utente e password
  • Token di protezione o altra password

BYB Close

  1. Aprire ! VPN  > Aggiungi profilo.
  2. Toccare il campo Tipo di collegamento > Cisco AnyConnect.
  3. Toccare Inserire il nome del server > Inserire il nome del server > Avanti.
  4. Se richiesto, toccare Accetto per accettare il contratto di licenza Cisco. Toccare  per visualizzare l'intero contratto.
  5. Immettere i dati di accesso > Collega.
  6. Se richiesto, immettere il token di protezione o un'altra password > Chiudi > Continua.
