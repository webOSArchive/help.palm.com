Aggiunta di un profilo VPN VPNC

Prima di iniziare

Richiedere le seguenti informazioni al provider del servizio e-mail utilizzato o all'amministratore del sistema:

  • Nome profilo
  • Server VPN
  • Nome utente e password
  • ID di gruppo
  • Segreto gruppo
  • Dominio
  • Dead Peer Detection: quando selezionata, se non viene ricevuto alcun traffico VPN per cinque minuti, il client verifica se il server VPN è ancora disponibile.
  • Metodo di crittografia: Protetto forza il client a utilizzare algoritmi AES o 3DES più protetti. Debole consente l'utilizzo dell'algoritmo meno sicuro 1DES. In base a quanto viene stabilito dal server VPN, è possibile che venga utilizzata una crittografia più sicura. Nessuno: non viene utilizzata alcuna crittografia. La crittografia non è tuttavia disattivata e in base a quanto viene stabilito dal server VPN, è possibile che venga utilizzata. Con l'opzione Nessuno viene semplicemente consentito al profilo VPN di non utilizzare alcuna crittografia.
  • Metodo Nat Traversal: Cisco-UDP, NAT-T (sempre), NAT-T (rileva automaticamente) o Disattivato

BYB Close

  1. Aprire ! VPN  > Aggiungi profilo.
  2. Se VPNC non è già selezionato nel campo Tipo di collegamento, toccare il campo Tipo di collegamento > VPNC.
  3. Toccare il campo Server VPN > immettere il nome del server.
  4. Toccare Avanti > Immettere le informazioni ottenute dal proprio amministratore di sistema > Collega.
  5. Se richiesto, immettere il token di protezione o un'altra password > Chiudi.
  6. Toccare Continua.
