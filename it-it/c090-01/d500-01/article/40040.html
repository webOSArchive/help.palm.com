Aggiunta di suonerie al dispositivo

È possibile aggiungere suonerie al dispositivo nei seguenti formati di file: MP3, AAC, AAC+, AMR, QCELP o WAV. Il modo in cui utilizzare il file come suoneria dipende dal tipo di file, da come viene acquisito e a seconda se si intende utilizzarlo solo come suoneria oppure riprodurlo nell'applicazione Musica.

È possibile acquisire file da utilizzare come suonerie nei seguenti modi:

  • Scaricando una suoneria da una delle molte applicazioni di suonerie presenti in App Catalog (ulteriori informazioni).
  • Copiando un file dal computer (istruzioni).

Per utilizzare un file AMR o QCELP come suoneria, è necessario che il file si trovi nella cartella delle suonerie nell'unità USB del dispositivo. Per acquisire il file nella cartella corretta:

  • File scaricati da App Catalog. Il file viene posizionato automaticamente nella cartella delle suonerie.
  • File copiati dal computer. È necessario posizionare il file nella cartella delle suonerie (istruzioni).

Per i file MP3, AAC, AAC+ e WAV, sono disponibili due opzioni:

  • Per utilizzare il file soltanto come suoneria: posizionarlo nella cartella delle suonerie (istruzioni). Se il file si trova già nel dispositivo, collegare il dispositivo al computer come descritto nella relativa procedura, individuare il file e spostarlo nella cartella delle suonerie. Se si scarica un file da un'applicazione di suonerie presente in App Catalog, viene posizionato automaticamente nella cartella delle suonerie.
  • Per utilizzare il file come suoneria e per riprodurlo nell'applicazione Musica: copiare il file nello smartphone e posizionarlo in una cartella qualsiasi nell'unità USB del dispositivo a eccezione della cartella delle suonerie (istruzioni).
