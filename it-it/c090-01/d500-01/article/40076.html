Apertura di un file
  1. Aprire ! Quickoffice .
  2. La prima volta che si apre l'applicazione Quickoffice è possibile scegliere di impostare l'applicazione in modo che utilizzi i file di un account online (istruzioni). Per utilizzare i file nel dispositivo o un account già impostato, toccare Vai.
  3. Toccare un account per visualizzarne i file. Toccare Il mio TouchPad per visualizzare i file archiviati nel dispositivo.
  4. Toccare Mostra > Documenti, PDF, Powerpoint o Fogli di calcolo, per visualizzare un elenco di soltanto quei tipi di file, oppure toccare Tutti i file. Se si tocca un file PDF viene aperta l'applicazione Adobe Reader nel dispositivo.
  5. Toccare il file.
  6. Per tornare all'elenco dei file, toccare Documenti nell'angolo in alto a sinistra dello schermo.

È possibile visualizzare file disponibili in un account online creato dall'utente (istruzioni), file copiati dal computer (istruzioni) o file allegati a messaggi e-mail (istruzioni). Questa operazione è possibile con file in formato DOC, DOCX, TXT, XLS, XLSX, PPT e PPTX.
