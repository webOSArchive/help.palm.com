Compte HP webOS. De quoi s'agit-il ?

Votre compte HP webOS vous apporte les avantages suivants :

  • Les mises à jour automatiques du système et du logiciel sont envoyées à votre smartphone.
  • Un compte HP webOS est automatiquement créé sur votre smartphone. Dans ce compte, vous pouvez stocker des informations sur les Contacts et le Calendrier que vous ne stockez pas dans des comptes en ligne tels que Google ou Exchange.
  • Les informations stockées sur votre compte HP webOS, ainsi que celles stockées dans les applications de votre appareil (notamment l'application Mémos), sont automatiquement sauvegardées sur les serveurs webOS.
  • Vous pouvez effacer à distance les informations de votre smartphone s'il venait à être perdu ou volé : rendez-vous sur hpwebos.com/account dans le navigateur Web de votre ordinateur et connectez-vous à votre compte HP webOS > Options de l'appareil > J'ai perdu mon appareil.

Une fois que vous l'avez configuré, vous pouvez modifier les informations de votre compte HP webOS (procédure).

REMARQUE : il est impossible d'accéder aux données d'un compte HP webOS sur le site Web du compte. Vous pouvez uniquement accéder aux données sur votre smartphone.
