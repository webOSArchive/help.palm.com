Personnalisation de l'application Contacts
  1. Ouvrez Contacts  > Menu (en haut, à gauche) > Préférences et comptes.
  2. Définissez les options suivantes :

    Ordre dans la liste : appuyez pour sélectionner le critère selon lequel la liste des contacts est triée. Si vous sélectionnez Nom pour effectuer le tri, les noms sont triés par ordre alphabétique, mais ils apparaissent toujours dans la liste Contacts en tant que prénom suivi du nom. Si vous sélectionnez Prénom pour le tri, les noms sont triés par ordre alphabétique des prénoms.

    Comptes : appuyez sur un compte pour en modifier la configuration d'ouverture de session, puis connectez-vous à l'aide de cette nouvelle configuration.

    Compte par défaut : appuyez sur le compte par défaut actuel > appuyez sur un compte. Tout nouveau contact créé est stocké dans le compte par défaut, sauf si vous spécifiiez un autre compte.

    Synchroniser maintenant : appuyez sur cette option pour lancer la synchronisation de vos comptes en ligne.

    Ajouter un compte : appuyez sur un type de compte > entrez les informations sur le compte. Avant de pouvoir configurer un compte sur votre smartphone, vous devez disposer d'un compte en ligne auprès du fournisseur.
