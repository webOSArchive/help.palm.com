Démarrer Exposition

Effectuez l'une des opérations suivantes :

  • Lorsque l'écran est éteint et le clavier fermé, placez le smartphone sur une station de charge HP Touchstone. Le mode Exposition démarre automatiquement.
  • Si vous placez votre smartphone sur une station de charge Touchstone lorsque l'écran est allumé, ou quand l'écran est éteint et le clavier sorti, le mode Exposition démarre quand le smartphone est resté inactif pendant la période définie dans les paramètres Verrouillage écran. L’écran ne se verrouille pas.
  • Ouvrir Exposition  > Démarrer Exposition. Le mode Exposition démarre alors, que votre smartphone soit placé ou non sur une station de charge Touchstone.

Remarque : si vous placez votre smartphone sur une station de charge Touchstone lorsque vous êtes en cours d'appel ou en train de visionner une vidéo dans les applications Vidéos ou YouTube, le mode Exposition ne démarre pas.
