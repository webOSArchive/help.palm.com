Effacement des données et réinitialisation du smartphone

Ouvrez ! Infos appareil  > Options de réinitialisation.

  • Pour effacer partiellement les données de votre smartphone :

    Appuyez sur Effacer Applications & Données pour supprimer les applications installées ainsi que tous les paramètres et données associés.

    Appuyez sur Effacer le lecteur USB pour supprimer les fichiers personnels stockés sur votre lecteur USB, y compris les photos, vidéos et fichiers musicaux. Effacer le lecteur USB ne supprime pas les applications installées.

  • Pour effacer toutes les données de votre smartphone :

    Appuyez sur Effacement intégral pour supprimer les applications installées, les paramètres et données associés, ainsi que tous les fichiers stockés sur le lecteur USB. (Vous pouvez aussi maintenir enfoncés Sym  + Option  + le bouton de marche/arrêt pour effectuer un effacement intégral.)

    Après un effacement intégral, connectez-vous à votre compte HP webOS afin de restaurer les données de votre compte HP webOS dans votre smartphone. Connectez-vous à vos comptes en ligne pour rétablir ces données. Les fichiers personnels stockés sur le lecteur USB, comme les photos, les vidéos et les fichiers musicaux ne sont pas rétablis. Vous pouvez également vous connecter à votre compte HP webOS sur un nouveau smartphone et restaurer vos données sur le nouveau smartphone.

    Appuyez sur Effacement intégral > Effacement intégral sécurisé pour supprimer toutes les applications, les paramètres et données associés, ainsi que tous les fichiers stockés sur le lecteur USB. Choisissez cette option pour formater intégralement votre smartphone, de sorte que vos données ne puissent plus être restaurées, si, par exemple, vous vendez votre smartphone.

Effectuez une sauvegarde manuelle des données (procédure) avant de procéder à la réinitialisation du smartphone. Important Close
