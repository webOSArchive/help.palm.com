Utilisation de l'Assistant de transfert de données (DTA)

L'Assistant de transfert de données installé sur votre smartphone permet d'exporter vos contacts, événements de calendrier, tâches et mémos/notes par le biais d'un processus qui commence sur votre ordinateur. Pour débuter la partie du transfert qui se déroule sur votre ordinateur, rendez-vous sur hpwebos.com/migrate-webOS pour télécharger le DTA et préparer les données pour le transfert vers votre smartphone.

Il est recommandé d'utiliser l'Assistant de transfert de données une seule fois pour chaque source de données de votre ordinateur. Si vous utilisez l'Assistant de transfert de données pour transférer des données plusieurs fois à partir d'une même source, vous risquez d'obtenir des données dupliquées.

L'Assistant de transfert de données n'est pas une application de synchronisation. Si vous modifiez les données enregistrées sur le smartphone une fois le transfert effectué, ces changements ne sont pas pris en compte dans le logiciel de bureau. Si vous souhaitez poursuivre la synchronisation avec les logiciels de bureau, rendez-vous sur hpwebos.com/sync-solutions pour vérifier si des solutions tierces sont disponibles pour votre smartphone. Si aucune solution n'est disponible pour votre smartphone, consultez les sites de développeurs de logiciels de synchronisation existants pour d'autres appareils webOS afin de savoir s'ils développent une solution pour votre smartphone.

L'Assistant de transfert de données vous permet de transférer les données vers l'un de vos comptes en ligne (Exchange ou Google par exemple) ou vers votre compte HP webOS. Il est possible de transférer une partie seulement des données vers le compte en ligne sélectionné. L'Assistant de transfert de données vous indique les données transférées et l'emplacement auquel elles sont transférées au moment où vous sélectionnez le compte. À ce stade, vous pouvez encore changer d'avis. Par la suite, ce ne sera plus possible.

 

  1. Une fois que vous avez fini de transférer les données depuis votre smartphone, déconnectez le smartphone de votre ordinateur.
  2. Sur votre smartphone, sélectionnez le compte sur lequel vous souhaitez transférer vos données. Effectuez l'une des opérations suivantes :

    • Si votre compte est répertorié : appuyez dessus. Si nécessaire, saisissez les informations relatives au compte > Me connecter.
    • Si vous voulez ajouter un compte en ligne sur votre smartphone : appuyez sur Ajouter un compte > appuyez sur un type de compte > saisissez les informations relatives au compte > Me connecter.

      Pour configurer un type de compte non répertorié : appuyez sur Autres > appuyez sur une application > suivez les instructions à l'écran pour ajouter le compte.

      IMPORTANTIMPORTANT :

      Si vous souhaitez annuler ou reporter le transfert, vous devez le faire avant d'effectuer l'étape suivante. Pression

      Menu (en haut, à gauche) > Annuler le transfert > Annuler le transfert ou Reprendre ultérieurement.

      Pour reprendre un transfert reporté : connectez votre smartphone à votre ordinateur à l'aide du câble USB > Lecteur USB. Interrompez ensuite la connexion en toute sécurité. Appuyez sur ou ajoutez un compte sur l'écran de transfert de données.

       

      Important

  3. Appuyez sur Terminer le transfert > OK.

    REMARQUE : l'importation s'effectue application par application et peut prendre de quelques minutes à une heure.

  4. Lorsqu'une notification indique que le transfert de données est terminé, appuyez sur Terminé.

Qu'advient-il des données une fois le transfert effectué ? Si vous transférez des données vers un compte en ligne, les modifications apportées aux données sur le smartphone sont synchronisées avec le compte figurant sur le serveur et inversement. Si vous transférez des données vers votre compte HP webOS, les données sont sauvegardées dans votre compte HP webOS. Ces données ne sont disponibles par le biais d'aucun compte en ligne.

Une version locale d'Outlook et Palm® Desktop est installée sur mon ordinateur. Est-il possible de transférer les données des deux applications ? Oui, c'est possible. Toutefois, si des données identiques sont stockées aux deux emplacements, le smartphone risque de contenir des données dupliquées.
