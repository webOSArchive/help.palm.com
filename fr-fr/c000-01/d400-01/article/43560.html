Création d'un hotspot sécurisé
  1. Ouvrir Hotspot mobile HP .
  2. Si vous ouvrez le hotspot mobile HP pour la première fois, appuyez sur OK pour continuer.
  3. Appuyez sur Non pour activer le hotspot mobile Palm.
  4. Si vous êtes connecté à un réseau Wi-Fi, appuyez sur Continuer pour vous déconnecter.

    REMARQUE : le smartphone doit rester connecté au réseau de votre fournisseur de services mobiles. N'utilisez pas le mode avion pour désactiver le Wi-Fi.

  5. Effectuez l'une des opérations suivantes :
    • Si vous exécutez le hotspot mobile HP pour la première fois : entrez une phrase secrète > Terminé.
    • Si vous disposez déjà d'un hotspot ouvert : appuyez sur Ouvrir > WPA/WPA2 Personal > entrez une expression de passe > Terminé.
  6. (Facultatif) Pour personnaliser le nom de votre hotspot, sélectionnez Informations réseau > entrez un nouveau nom > Terminé.
  7. Après avoir créé votre hotspot, accédez à l'appareil que vous souhaitez connecter à Internet (votre ordinateur portable, par exemple) puis utilisez ses paramètres de connexion Wi-Fi pour vous connecter à votre hotspot mobile HP. Une fois la connexion établie, vous pouvez naviguer sur le Web avec votre ordinateur portable.
