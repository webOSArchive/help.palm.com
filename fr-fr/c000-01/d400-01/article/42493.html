Obtention d'aide en cas d'échec de la mise à jour du logiciel

Bien que cela se produise rarement, il est possible, lors d'une mise à jour logicielle sans fil, que votre smartphone se fige et cesse de répondre aux pressions exercées sur l'écran ou sur les touches. HP webOS Doctor restaure les fonctionnalités de votre smartphone et le met à jour avec les dernières versions du système d'exploitation et des autres logiciels système. Une fois l'outil utilisé, les données sauvegardées dans votre compte HP webOS sont automatiquement restaurées sur le smartphone.

Pour utiliser l'outil, vous devez disposer des éléments suivants :

  • Le câble USB fourni avec votre smartphone
  • Un ordinateur de bureau présentant les caractéristiques suivantes :
    • Windows XP, Vista® ou 7 ou Mac OS 10.5
    • Un port USB disponible (pas via un hub ou une connexion USB secondaire)
    • La dernière version de Java (disponible sur le site java.com)
    • 200 Mo d'espace disque disponible
  • Une connexion Internet permettant de télécharger de gros fichiers (de 10 Mo à 200 Mo)

Pour télécharger l'outil, visitez la page hpwebos.com/support.
