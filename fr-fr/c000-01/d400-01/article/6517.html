Navigation dans les applications de l'App Catalog de HP webOS
  1. Ouvrir App Catalog .
  2. Prenez connaissance des conditions d'utilisation et appuyez sur Accepter.
  3. Appuyez sur l'icône d'une application présentée ou effectuez l'une des opérations suivantes :
    • Pour rechercher des applications par nom : saisissez le terme à rechercher > appuyez sur Entrée .
    • Pour rechercher des applications par catégorie : appuyez sur l'en-tête Parcourir les catégories, puis appuyez sur une catégorie. Si l'en-tête continue de s'afficher, appuyez dessus pour sélectionner des sous-catégories.
    • Pour restreindre la recherche à l'aide d'un filtre : appuyez sur une icône au bas de l'écran.

    • Pour lancer une recherche portant uniquement sur les applications proposées, les applications en vente, les applications gratuites ou les applications les plus récentes, appuyez d'abord sur l'icône correspondante au bas de l'écran, puis saisissez un terme à rechercher.
  4. Appuyez sur le nom d'une application pour afficher les détails correspondants.
  5. À partir de l'écran donnant des renseignements précis sur l'application sélectionnée, effectuez l'une des opérations suivantes :
    • Pour agrandir la vue : appuyez dessus.
    • Pour consulter les critiques disponibles : appuyez sur Critiques.
    • Pour partager avec un ami les informations concernant cette application : appuyez sur Partager > Email ou Message SMS.
    • Pour accéder à la page d'accueil du développeur : appuyez sur Accueil développeur.
    • Pour accéder à la page de support produit du développeur : appuyez sur Support (si l'option est disponible).
    • Pour visionner une vidéo de l'application dans l'application YouTube de votre smartphone, appuyez sur le lien Youtube (s'il est disponible).
