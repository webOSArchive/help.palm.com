Ajout d'un ami
  1. Ouvrir Messagerie  > Menu (en haut, à gauche) > Ajouter un ami.
  2. Appuyez sur le champ Service IM > appuyez sur le compte auquel vous souhaitez ajouter cet ami.
  3. Effectuez l'une des opérations suivantes :
    • Pour ajouter un ami Skype : appuyez sur Ajouter un contact > entrez le nom d'écran de l'ami que vous souhaitez ajouter >  > entrez le nom de l'ami.
    • Pour ajouter un ami dans tout autre compte de messagerie instantanée, appuyez sur le champ Nom de l'ami et saisissez les informations requises par le compte, par exemple, une adresse email ou une adresse de messagerie instantanée.
  4. (Facultatif, si disponible) Appuyez sur le champ Message > saisissez un message pour votre ami.
  5. Appuyez sur Ajouter un ami.

Pour ajouter un ami à un compte de messagerie instantanée, vous devez envoyer une demande, qui doit être acceptée par la personne concernée.
