Utilisation des liens d'une page Web
  • Pour accéder à une autre page par le biais du lien correspondant : effectuez un zoom avant sur la page (en appuyant deux fois dessus ou en effectuant le geste pincer/ouvrir) > appuyez sur le lien.
  • Pour composer un numéro de téléphone affiché sous forme de lien : appuyez sur le lien pour ouvrir le clavier numérique avec le numéro déjà saisi > appuyez sur .
  • Pour ouvrir un lien dans une nouvelle carte de navigateur : appuyez sur Option  et maintenez-la enfoncée > appuyez sur le lien > appuyez sur Ouvrir dans nouvelle carte.
  • Pour partager un lien : appuyez sur Option  et maintenez-la enfoncée > appuyez sur le lien > appuyez sur Partager le lien. Un nouveau message email s'ouvre avec le lien en pièce jointe.
  • Pour copier un lien ou une URL : appuyez sur Option  et maintenez-la enfoncée > appuyez sur le lien. Appuyez sur Copier l'URL puis collez-la dans n'importe quelle application permettant l'édition de texte.
