Achat d'une application directement à partir d'un lien de code promotionnel
  1. Si vous recevez un lien avec un code promotionnel (tel qu'un email ou un SMS), cliquez sur le lien.
  2. Effectuez l'une des opérations suivantes :
    • Si la page principale de l'App Catalog est affichée : appuyez sur OK pour ignorer la notification de code promotionnel > parcourez la liste jusqu'à l'application de votre choix > appuyez sur le nom de l'application.
    • Si la page d'une application spécifique est affichée : appuyez sur OK pour ignorer la notification de code promotionnel.
  3. Appuyez sur Télécharger : [quantité].
  4. Si une application utilise des services de positionnement, une notification s'affiche. Appuyez sur Continuer.
  5. Si vous y êtes invité, entrez le mot de passe de votre compte HP webOS pour autoriser l'achat > Continuer.
  6. Si vous n'avez pas configuré de compte de facturation, vous êtes invité à en configurer un (procédure).
  7. Effectuez l'une des opérations suivantes :
    • Pour acheter l'application avec le code promotionnel : appuyez sur Utiliser le code promo.
    • Pour acheter l'application à l'aide de votre compte de facturation : appuyez sur Acheter.
  8. Pour ouvrir une application téléchargée, appuyez sur Appuyer pour lancer. L'icône de l'application s'affiche aussi sur la première page de l'écran de lancement, d'où vous pouvez également l'ouvrir.

Une offre de code promotionnel peut être utilisée pour une application spécifique ou pour n'importe quelle application (jusqu'à une certaine quantité). Vous devez avoir un compte de facturation configuré dans App Catalog pour acheter des applications avec un code promotionnel. Toutefois, vos achats ne vous sont pas facturés quand vous utilisez le code.
