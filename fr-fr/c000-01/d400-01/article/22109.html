Comment faire pour envoyer et recevoir des emails sur mon smartphone ?

Utilisez l'application Email de votre smartphone pour accéder à vos nombreux comptes de messagerie : société (comme Exchange), FAI (comme EarthLink ou Comcast) et Web (comme Gmail). Vous devez créer le compte en ligne avant de pouvoir le configurer sur votre smartphone.

Vous pouvez configurer vos comptes de messagerie dans ces différents emplacements. La procédure reste la même.

  • L'application Comptes (procédure)
  • L'application Email

La fonction HP Synergy offre un autre avantage : durant la procédure de configuration d'un compte de messagerie, vous pouvez également configurer le même compte pour toutes les autres applications pouvant afficher des données à partir de ce compte.

Par exemple, supposons que vous vouliez configurer votre compte Gmail sur votre smartphone. La dernière étape de la configuration consiste à sélectionner toutes les autres applications pour lesquelles vous souhaitez recevoir des données de votre compte Google (outre l'application Email, vous pouvez inclure Contacts, Messagerie et Calendrier). Une fois la configuration terminée, votre smartphone commence à télécharger votre messagerie et toutes les données pour les autres applications sélectionnées dont la synchronisation est activée.

Si vous le souhaitez, vous pouvez également utiliser le navigateur Web de votre smartphone pour consulter votre messagerie Web simplement en accédant au site Web du fournisseur de messagerie.
