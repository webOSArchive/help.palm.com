Ajout de sonneries à votre smartphone

Vous pouvez ajouter un fichier de l'un des types suivants comme sonnerie pour votre smartphone : MP3, AAC, AAC+, AMR, QCELP ou WAV. La manière d'utiliser le fichier comme sonnerie dépend des éléments suivants : le type de fichier, le mode d'acquisition du fichier et si vous souhaitez l'utiliser uniquement comme sonnerie ou le lire dans l'application Musique.

Vous pouvez obtenir des fichiers à utiliser comme sonneries à l'aide de l'une des méthodes suivantes :

  • Téléchargez une sonnerie à partir de l'une des nombreuses applications de sonnerie dans l'App Catalog HP webOS (procédure).
  • Téléchargez un fichier MP3 que vous souhaitez utiliser comme sonnerie à partir de l'application Amazon MP3 (si disponible sur votre smartphone) (procédure).
  • Copiez un fichier depuis votre ordinateur (procédure).

Pour utiliser un fichier AMR ou QCELP comme sonnerie, le fichier doit être enregistré dans le dossier des sonneries du lecteur USB de votre smartphone. Voici comment placer le fichier dans le dossier approprié :

  • Fichiers téléchargés depuis l'App Catalog : le fichier est automatiquement placé dans le dossier des sonneries.
  • Fichiers copiés depuis votre ordinateur : vous devez placer le fichier dans le dossier des sonneries (procédure).

En ce qui concerne les fichiers MP3, AAC, AAC+ et WAV, vous disposez de deux options :

  • Pour utiliser un fichier comme sonnerie uniquement : placez le fichier dans le dossier des sonneries (procédure). Si le fichier est déjà sur votre smartphone, connectez votre smartphone à votre ordinateur comme décrit dans cette procédure, accédez au fichier et déplacez-le dans le dossier des sonneries. Si vous téléchargez un fichier d'une application de sonnerie dans l'App Catalog, il est automatiquement placé dans le dossier des sonneries.
  • Pour utiliser le fichier comme sonnerie et l'écouter dans l'application Musique : copiez le fichier dans votre smartphone et placez le fichier dans n'importe quel dossier du stockage sur le lecteur USB de votre smartphone, à l'exception du dossier des sonneries (procédure). Si vous téléchargez un fichier MP3 de l'application Amazon MP3, il est automatiquement enregistré dans un dossier où vous pouvez à la fois l'écouter dans l'application Musique et l'utiliser comme sonnerie.
