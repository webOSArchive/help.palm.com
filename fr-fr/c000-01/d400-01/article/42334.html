Réception et lecture des emails
  1. Ouvrir Email .
  2. Appuyez sur pour afficher les dossiers de comptes > appuyez sur un dossier.
  3. Appuyez sur un message pour l'ouvrir.
  4. (Facultatif) Procédez de l'une des façons suivantes :
    • Pour appeler un numéro : appuyez sur le numéro > .
    • Pour envoyer un message SMS, appuyez de manière prolongée sur le numéro > appuyez sur SMS.
    • Pour ajouter un numéro de téléphone à vos contacts : appuyez longuement sur le numéro > appuyez sur Ajouter aux Contacts. Appuyez sur Enregistrer comme nouveau pour créer un contact contenant le numéro de téléphone, ou appuyez sur Ajouter aux éléments existants pour ajouter le numéro de téléphone à un contact existant.
    • Pour envoyer un email : appuyez sur l'adresse email > entrez votre message > .
    • Pour vous rendre sur une adresse Web : appuyez dessus.
    • Pour partager un lien vers un site Web : appuyez sur Option  et maintenez-la enfoncée > appuyez sur le lien > Partager le lien.
    • Pour copier un lien vers un site Web : appuyez sur Option  et maintenez-la enfoncée > appuyez sur le lien > Copier l'URL.
    • Pour copier un email en vue de le coller dans un mémo ou un autre document : Menu (en haut, à gauche) > Éditer > Copier tout.
  5. Appuyez sur ou sur pour afficher le message précédent ou suivant.

L'application Email synchronise les messages chaque fois que vous ouvrez un dossier de courrier. L'application synchronise également les messages sur un planning que vous pouvez définir (procédure). Pour synchroniser vos messages alors qu'un dossier est déjà ouvert : appuyez sur .
