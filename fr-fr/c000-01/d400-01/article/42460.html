Recherche d'informations dans une application de votre smartphone
  1. Ouvrez la vue Cartes (procédure) ou l'écran de lancement (procédure).
  2. Commencez la saisie des informations que vous recherchez. En fonction de l'application dans laquelle vous souhaitez effectuer une recherche, ces informations peuvent être les suivantes : objet d'un email; adresse physique à localiser sur une carte, nom de signet, nom d'une application à télécharger ou tout autre terme.
  3. En fonction de l'application dans laquelle vous souhaitez effectuer une recherche, procédez comme suit. Vous devrez peut-être faire défiler l'écran pour afficher les résultats de recherche de l'application.
    • Signets et historique : votre smartphone affiche le nombre de sites Web dans votre liste des signets et votre historique de navigation dont le nom contient le terme recherché. Appuyez sur Signets et historique pour voir une liste des sites Web. Appuyez sur une entrée pour vous rendre sur le site Web concerné.
    • Email : votre smartphone affiche le nombre d'emails dont le nom de l'expéditeur ou l'objet contient le terme recherché. Appuyez sur Email pour afficher la liste des messages correspondants. Appuyez sur un message pour l'afficher.
    • Cartes : pour localiser le terme que vous recherchez sur une carte, sous Lancement et recherche, appuyez sur Cartes. Si l'application Cartes n'est pas installée sur votre smartphone, l'App Catalog ouvre la page de l'application Cartes.
    • App Catalog de HP : pour rechercher des applications correspondant au terme à rechercher dans l'App Catalog, sous Lancement et recherche, appuyez sur App Catalog de HP.

Si aucun résultat ne s'affiche pour le terme à rechercher dans une application donnée, vous devez peut-être activer une préférence pour inclure l'application concernée dans la recherche Tapez maintenant (procédure).
