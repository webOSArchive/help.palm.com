Recherche sur le Web
  1. Ouvrez la vue Cartes (procédure) ou l'écran de lancement (procédure).
  2. Commencez à saisir le terme de recherche ou une adresse de site Web, telle que hp.com.
  3. Effectuez l'une des opérations suivantes. Vous devrez peut-être faire défiler l'affichage pour afficher les moteurs de recherche Web.
    • Pour accéder à un site Web : si vous avez saisi l'adresse d'un site Web, appuyez sur Aller au site Web.
    • Pour effectuer une recherche Web à l'aide de votre moteur de recherche par défaut, appuyez sur le nom du moteur de recherche. Le moteur de recherche apparaît directement au-dessous des entrées de contact correspondantes.
    • Pour afficher les termes de recherche proposés dans votre moteur de recherche par défaut à partir des caractères saisis, appuyez sur l'option Suggestion figurant à droite du nom de votre moteur de recherche par défaut. Si votre recherche ne renvoie aucun contact, aucune application, ou aucun autre élément sur votre téléphone, les suggestions s'affichent automatiquement. Appuyez sur une suggestion pour effectuer une recherche sur ce terme.
    • Pour effectuer une recherche Web à l'aide d'un autre moteur de recherche : sous Lancement et recherche, appuyez sur le nom d'un moteur de recherche. Si le moteur de recherche est spécifique à un site Web, tel que Wikipedia, votre téléphone effectue une recherche sur ce site uniquement. Pour afficher plus d'options de recherche, appuyez sur Plus sous Lancement et recherche.
