Personnalisation de Facebook
  1. Ouvrez ! Facebook  > Menu (en haut, à gauche) > Préférences et comptes.
  2. Définissez les options suivantes :
    • Afficher les commentaires dans le flux : appuyez sur On/Off pour déterminer si les commentaires sont affichés avec vos flux d'informations.
    • Secouer pour recharger : appuyez sur Activé/Désactivé pour actualiser la page Actualités en secouant l'appareil.
    • Toutes les 20 minutes : définit l'intervalle de vérification des notifications entrantes. Pour changer cet intervalle, appuyez sur Toutes les 20 minutes > appuyez sur un nouvel intervalle.
    • Notifications HP webOS : appuyez pour activer/désactiver ce paramètre. Quand ce paramètre est activé, les nouvelles notifications sont annoncées sur le tableau de bord.
    • Comptes : appuyez sur un compte pour en modifier les paramètres de connexion ou le supprimer. Ne changez les paramètres de connexion qu'en cas de modification de ces informations dans votre compte Facebook en ligne. En effet, les informations saisies ici doivent correspondre à celles du compte en ligne.
    • Ajouter le compte : ajouter d'autres comptes Facebook. Le dernier compte ajouté devient votre compte par défaut. Pour modifier votre compte par défaut, appuyez sur Compte par défaut > appuyez sur un autre compte. Facebook affiche uniquement les informations de votre compte par défaut.
