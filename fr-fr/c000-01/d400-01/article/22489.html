Saisie de paramètres de compte avancés

Ces préférences s'appliquent uniquement au compte sélectionné à l'étape 2.

  1. Ouvrir Email .
  2. Menu (en haut, à gauche) > Préférences et comptes > appuyez sur un compte.
  3. Configurez les options de votre choix :
    • Nom du compte : entrez le nom qui apparaît dans la liste de comptes.
    • Nom complet :entrez le nom que vous voulez voir apparaître dans le champ De des messages que vous envoyez.
    • Afficher l'icône : déterminez si une icône de notification doit s'afficher à l'arrivée d'un nouveau message.
    • Alerte : lorsque l'icône Afficher est activée, indiquez si un son doit être émis à l'arrivée de nouveaux messages (procédure).
    • Signature : appuyez sur cette option pour créer une signature à ajouter aux messages sortants (procédure).
    • Adresse pour réponse : autre adresse email fournie aux destinataires de vos messages et qu'ils peuvent utiliser pour vous répondre.
    • Afficher : nombre de jours de messages qui doivent être récupérés sur le serveur.
    • Récupérer : fréquence de synchronisation des emails pour ce compte.
    • Synchroniser les emails supprimés : (pour les comptes POP uniquement) permet de supprimer les messages sur le serveur lorsque vous les supprimez sur votre smartphone.
    • Synchroniser le serveur avec l'appareil : (pour les comptes POP uniquement) permet de supprimer les messages sur votre smartphone lorsque vous les supprimez sur le serveur.
    • Dossiers par défaut : (pour les comptes IMAP uniquement) permet de spécifier les dossiers contenant vos messages envoyés et supprimés et vos brouillons.
    • Modifier config. ouv. session : procédure.
    • Supprimer le compte : procédure.
