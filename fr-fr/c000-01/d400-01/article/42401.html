Lecture de messages SMS/multimédia
  1. Ouvrir Messagerie .
  2. Appuyez sur une conversation pour afficher le message.
  3. Pour afficher un fichier en pièce jointe dans un message MMS, appuyez sur le nom d'un fichier vidéo, musical ou vCard.
  4. Les photos en pièce jointe s'affichent dans le message. Pour enregistrer une image jointe, appuyez sur l'image > Afficher > Copier dans Photos.
  5. (Facultatif) Procédez de l'une des façons suivantes dans le corps d'un message :
    • Pour appeler un numéro : appuyez sur le numéro > .
    • Pour ajouter un numéro de téléphone à vos contacts : appuyez longuement sur le numéro > appuyez sur Ajouter aux Contacts. Appuyez sur Enregistrer comme nouveau pour créer un contact contenant le numéro de téléphone, ou appuyez sur Ajouter aux éléments existants pour ajouter le numéro de téléphone à un contact existant.
    • Pour envoyer un email : appuyez sur l'adresse email.
    • Pour vous rendre sur une adresse Web : appuyez dessus.
