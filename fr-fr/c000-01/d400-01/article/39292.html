Configuration d'un compte en ligne dans Contacts
  1. Ouvrir Contacts .
  2. Effectuez l'une des opérations suivantes :
    • Configurez un compte en ligne lors de votre première utilisation de Contacts (procédure).
    • Pour configurer un compte en ligne à tout autre moment : appuyez sur Menu (en haut, à gauche) > Préférences et comptes > Ajouter un compte > appuyez sur un type de compte > passez à l'étape 3. Pour configurer un type de compte non répertorié, appuyez sur Rechercher autres, puis sur une application et suivez ensuite les instructions à l'écran.
  3. Si vous configurez un compte Skype, appuyez sur Me connecter.
  4. Saisissez les informations du compte > Me connecter.
  5. Si vous configurez un compte Exchange, vous devez entrer un mot de passe ou un code PIN qui sera requis pour déverrouiller le smartphone afin de préserver la sécurité de votre compte Exchange. Appuyez sur Définir le mot de passe ou Définir le code PIN > entrez un mot de passe ou un code PIN > Terminé.
  6. Selon le compte, vous pouvez être invité à sélectionner les applications téléchargeant des données depuis ce compte. Pour chaque application, appuyez sur Activé ou Désactivé, puis sur Créer.
