Insertion des coordonnées d'un contact dans Contacts

Si vous souhaitez intégrer les coordonnées d'un contact à l'application Contacts, vous avez le choix entre plusieurs procédures :

  • Se connecter à un carnet d'adresses en ligne : si vous disposez d'un carnet d'adresses dans un compte en ligne fonctionnant avec la fonction HP Synergy (par exemple, Google ou Microsoft Exchange), vous pouvez configurer Contacts sur votre smartphone pour qu'il synchronise et affiche les contacts du carnet d'adresses en ligne. Après avoir configuré la connexion au carnet d'adresses en ligne, les contacts que vous saisissez en ligne apparaissent automatiquement dans Contacts et les contacts que vous entrez sur votre smartphone se synchronisent avec le compte en ligne, à condition que ce dernier permette l'écriture à partir du smartphone sur le compte en ligne.

    Vous pouvez configurer vos comptes de contacts soit dans l'application Comptes (procédure) soit dans l'application Contacts.

    La fonction Synergy offre un autre avantage : durant la procédure de configuration d'un compte de contacts, vous pouvez également configurer le même compte pour toutes les autres applications pouvant afficher des données à partir de ce compte.

    Par exemple, supposons que vous vouliez configurer votre compte de contacts Google sur votre smartphone. La dernière étape de la configuration consiste à sélectionner toutes les autres applications pour lesquelles vous souhaitez recevoir des données de votre compte Google (outre l'application Contacts, vous pouvez inclure Messagerie, Calendrier et Email). Une fois la configuration terminée, votre smartphone commence à télécharger vos contacts et toutes les données pour les autres applications sélectionnées dont la synchronisation est activée.

  • Utiliser une solution de synchronisation tierce : si vous souhaitez utiliser une application de bureau pour stocker vos contacts, vous pouvez utiliser une solution tierce (vendue séparément) pour synchroniser vos contacts directement avec votre ordinateur. Consultez la page Web hpwebos.com/sync-solutions pour savoir si des solutions tierces sont disponibles pour votre smartphone. Si aucune solution n'est disponible pour votre smartphone, consultez les sites de développeurs de logiciels de synchronisation existants pour d'autres appareils webOS afin de savoir s'ils développent une solution pour votre smartphone.
  • Exportez des contacts à l'aide de l'Assistant de transfert de données (DTA) : après avoir exporté des contacts depuis un logiciel de bureau à l'aide du DTA (procédure), sélectionnez le compte auquel assigner ces contacts. Recherchez ces événements dans le compte sélectionné.
  • Entrez les contacts directement dans Contacts (procédure). Assignez le contact à l'un de vos carnets d'adresses en ligne ou créez le contact dans votre compte HP webOS. Les informations de votre compte HP webOS ne se synchronisent pas mais elles sont incluses dans la sauvegarde régulière des informations de votre compte HP webOS.
  • Téléchargez une vCard qui vous a été envoyée par email : une vCard peut être jointe à un email ou à un message SMS, et vous pouvez télécharger les informations d'une vCard dans votre application Contacts (procédure).
