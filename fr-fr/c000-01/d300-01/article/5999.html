Téléchargement de fichiers sur Internet
  1. Ouvrir ! Web .
  2. Accédez à la page Web contenant le fichier à télécharger.
  3. Appuyez sur le nom du fichier.
  4. Si une application de votre smartphone prend en charge ce type de fichier, le nom du fichier apparaît en bas de l'écran. Appuyez pour ouvrir le fichier.

Tous les fichiers sont téléchargés dans la mémoire USB du smartphone. Si une application de votre smartphone prend en charge ce type de fichier, vous pouvez également trouver le fichier dans l'application en question. S'il s'agit par exemple d'un fichier PDF, vous le trouverez dans l'Afficheur de PDF ; un fichier MP3, vous le trouverez dans Musique.
