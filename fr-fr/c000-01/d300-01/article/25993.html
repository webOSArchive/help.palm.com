Création d'un événement
  1. Ouvrir Calendrier .
  2. Dans le coin supérieur droit, appuyez sur Tous ou sur le nom d'un compte en ligne > appuyez sur le calendrier que vous voulez utiliser pour cet événement.
    Si vous créez un événement dans la vue Tous les calendriers, l'événement est créé dans le calendrier défini comme étant le calendrier par défaut (procédure).
  3. Recherchez le jour souhaité (procédure) > appuyez sur un créneau horaire vide > entrez le nom de l'événement > appuyez sur Entrée .
    Inutile de chercher un bouton d'enregistrement : vos informations sont automatiquement enregistrées lorsque vous fermez l'événement.

Pour utiliser Tapez maintenant pour créer un événement : dans l'écran de lancement ou la vue Cartes, commencez à saisir le texte de l'événement. Sous Actions rapides, appuyez sur Plus > Nouvel événement du calendrier. L'application Calendrier s'ouvre sur un nouvel événement avec le texte que vous avez saisi dans la ligne d'objet.

Dans la vue quotidienne, les créneaux horaires disponibles entre les événements sont comprimés et appelés X heures libres. Pour développer les créneaux libres afin d'y entrer un événement, appuyez sur X heures libres.

Si vous ajoutez un événement à l'un de vos comptes en ligne, les informations sont également synchronisées avec le serveur de votre fournisseur sur le Web. Si vous ajoutez l'événement à votre compte de profil Palm, les informations sont sauvegardées dans votre profil Palm.
