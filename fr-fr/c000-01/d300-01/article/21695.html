Personnalisation de Tapez maintenant
  1. En vue Cartes ou dans l'écran de lancement, commencez à saisir du texte pour afficher l'écran des résultats de recherche.
  2. Faites défiler l'affichage jusqu'en bas de l'écran, puis appuyez sur Préférences.
  3. Sélectionnez l'une des options ci-après :

    Moteur de recherche par défaut : moteur de recherche utilisé par votre smartphone pour effectuer une recherche sur le Web lorsque vous saisissez un terme à rechercher. Pour changer le moteur de recherche, appuyez sur le moteur actuellement affiché, puis appuyez sur le moteur souhaité. Cette liste contient deux types de moteurs de recherche : des moteurs effectuant des recherches sur le Web, tels que Google, et des moteurs effectuant des recherches sur des sites Web spécifiques, tels que Wikipédia.

    Applications : si vous incluez des applications dans vos paramètres de recherche, tous les éléments correspondants compris dans ces applications apparaissent dans les résultats de recherche. Pour ajouter une application à vos résultats de recherche, appuyez sur Ajouter des recherches d'application, puis sur le nom de l'application.

    Lancement et recherche : cette liste affiche d'autres moteurs de recherche et les applications dans lesquelles votre smartphone effectue une recherche lorsque vous saisissez un terme de recherche. Les moteurs de recherche sont désignés par Web ; tous les moteurs de recherche répertoriés peuvent être utilisés en tant que moteur de recherche par défaut. Les applications incluses dans vos recherches sont désignées par recherche d'applications. Pour inclure d'autres moteurs de recherche ou applications dans vos recherches, appuyez sur Ajouter d'autres recherches, puis sur l'élément souhaité.

    Actions rapides : permet d'afficher les types d'éléments que vous créez à partir des résultats de votre recherche, tel qu'un email ou une note. Pour ajouter un nouveau type d'élément, appuyez sur Ajouter des actions rapides, puis sur l'élément souhaité.

    Rechercher autres : permet de rechercher dans l'App Catalog des applications et services tiers permettant d'exécuter l'une au moins des fonctions Tapez maintenant suivantes : recherche Web, recherche d'applications ou création d'un nouvel élément (Action rapide). Rechercher autres peut vous permettre par exemple de télécharger un nouveau moteur de recherche Web. Pour ce faire, appuyez sur Rechercher autres, puis téléchargez une application de la liste affichée dans l'App Catalog (procédure).

  4. Pour réorganiser les éléments d'une liste : appuyez sur l'élément souhaité et maintenez-le enfoncé, attendez l'indication visuelle, puis faites glisser l'élément à l'emplacement de votre choix.
