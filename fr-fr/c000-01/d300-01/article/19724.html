Achat d'une application en utilisant votre compte de facturation ou en entrant un code promotionnel
  1. Ouvrir App Catalog .
  2. Naviguez jusqu'à l'application recherchée > appuyez sur son nom > Télécharger pour [montant].
  3. Si une application utilise des services de positionnement, une notification s'affiche. Appuyez sur Continuer.
  4. À l'invite, saisissez le mot de passe de votre profil Palm pour autoriser l'achat > Continuer.
  5. Si vous n'avez pas configuré de compte de facturation, vous êtes invité à en configurer un (procédure).
  6. Effectuez l'une des opérations suivantes :
    • Pour acheter l'application avec un code promotionnel : appuyez sur Utiliser le code promo > saisissez le code > appuyez sur Utiliser le code promo.
    • Pour acheter l'application à l'aide de votre compte de facturation : appuyez sur Acheter.
  7. Pour ouvrir une application téléchargée, appuyez sur Appuyer pour lancer. L'icône de l'application s'affiche aussi sur la première page de l'écran de lancement, d'où vous pouvez également l'ouvrir.

Une offre de code promotionnel peut être utilisée pour une application spécifique ou pour n'importe quelle application (jusqu'à une certaine quantité). Vous devez avoir un compte de facturation configuré dans App Catalog pour acheter des applications avec un code promotionnel. Toutefois, vos achats ne vous sont pas facturés quand vous utilisez le code.
