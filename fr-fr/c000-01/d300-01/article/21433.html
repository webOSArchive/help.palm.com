Gestion des informations de positionnement communiquées à des applications
  1. Ouvrir Services de positionnement .
  2. Si nécessaire, appuyez sur Activer le GPS.
  3. Dans Pour les applications, appuyez sur l'une des options suivantes :

    Pos. automatique : permet aux applications d'identifier automatiquement votre position.

    Toujours demander : les applications affichent une demande d'autorisation d'identifier votre position. Vous autorisez les applications à identifier votre position pour la session actuelle uniquement.

  4. Si vous sélectionnez Pos. automatique, appuyez sur Activé/Désactivé pour les options suivantes :

    Photos Geotag : disponible uniquement quand Pos. automatique est sélectionné. Si Photos Geotag est activé, les photos sont marquées avec la longitude et la latitude du lieu auquel elles ont été prises.

    Récupération des données contextuelles : option disponible uniquement lorsque les services Google sont activés (procédure). Si la récupération des données contextuelles est activée, des données de positionnement anonymes sont transmises depuis votre téléphone. Ces données sont utilisées pour améliorer la qualité des services de positionnement.

L'application Services de positionnement vous permet de contrôler si votre positionnement est donné aux applications qui le demandent. Une application peut demander votre position actuelle pour vous fournir des informations pertinentes en fonction de votre localisation. Par exemple, Google Maps peut se servir de votre position actuelle pour vous fournir l'itinéraire vers une destination.
