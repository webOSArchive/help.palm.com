Transfert d'un message
  1. Ouvrir Messagerie .
  2. Appuyez sur une conversation > appuyez sur un message.
  3. Effectuez l'une des opérations suivantes :
    • Pour transférer un message via la messagerie : appuyez sur Transférer > entrez le numéro ou l'adresse du destinataire dans le champ A.
    • Pour transférer un message dans un email : appuyez sur Transférer par email > entrez le numéro ou l'adresse du destinataire dans le champ A.
  4. (Facultatif) Appuyez sur le texte du message pour le modifier.
  5. Appuyez sur .
