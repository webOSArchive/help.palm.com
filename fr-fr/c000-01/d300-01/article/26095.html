Comment insérer des événements dans le Calendrier ?

Plusieurs options s'offrent à vous si vous souhaitez que des événements s'affichent dans Calendrier :

  • Se connecter à un calendrier en ligne : si vous disposez d'un calendrier dans un compte en ligne fonctionnant avec la fonction HP Synergy (par exemple, Google ou Microsoft Exchange), vous pouvez configurer Calendrier sur votre smartphone pour qu'il synchronise et affiche les événements du calendrier en ligne. Après avoir configuré la connexion au calendrier en ligne, les événements que vous saisissez en ligne apparaissent automatiquement dans le Calendrier et les événements que vous entrez sur votre smartphone se synchronisent avec le compte en ligne, à condition que ce dernier permette l'écriture à partir du smartphone sur le compte en ligne.

    Vous pouvez configurer vos comptes du calendrier soit dans l'application Comptes (procédure) soit dans l'application Calendrier.

    La fonction Synergy offre un autre avantage : durant la procédure de configuration d'un compte du calendrier, vous pouvez également configurer le même compte pour toutes les autres applications pouvant afficher des données à partir de ce compte.

    Par exemple, supposons que vous vouliez configurer votre compte du calendrier Google sur votre smartphone. La dernière étape de la configuration consiste à sélectionner toutes les autres applications pour lesquelles vous souhaitez recevoir des données de votre compte Google (outre l'application Calendrier, vous pouvez inclure Contacts, Messagerie et Email). Une fois la configuration terminée, votre smartphone commence à télécharger vos événements du calendrier et toutes les données pour les autres applications sélectionnées dont la synchronisation est activée.

  • Saisie directe d'un événement dans Calendrier (procédure) : vous pouvez assigner l'événement à l'un de vos calendriers en ligne. Il se synchronise alors avec le calendrier en ligne. Vous pouvez également créer l'événement dans votre compte de profil Palm. Dans ce cas, l'événement s'affiche uniquement sur le téléphone. Les événements du profil Palm sont inclus dans la sauvegarde quotidienne de vos informations de profil Palm, ce qui permet de les restaurer s'ils sont involontairement effacés.

  • Utilisez une solution de synchronisation tierce : si vous souhaitez utiliser une application de bureau pour stocker les événements de votre calendrier, vous pouvez utiliser une solution tierce (vendue séparément) pour synchroniser vos événements du calendrier directement avec votre ordinateur (rendez-vous sur palm.com/sync-solutions).

  • Transfert d'événements à l'aide de l'Assistant de transfert de données (DTA) : lorsque vous exportez des événements du calendrier depuis un logiciel de bureau (procédure), vous devez sélectionner le compte auquel vous souhaitez assigner ces événements. Recherchez ces événements dans le compte sélectionné.
