Installation du hotspot mobile Palm

Cette application n'est pas prise en charge par tous les fournisseurs de services mobiles. Contactez votre opérateur afin d'obtenir des informations sur les tarifs et la disponibilité de ce service.

Vérifiez dans le programme de lancement si le hotspot mobile Palm est installé sur votre téléphone. S'il ne se trouve pas dans le programme de lancement, téléchargez-le gratuitement depuis l'App Catalog.

  1. Ouvrez App Catalog .
  2. Saisissez hotspot mobile dans le champ de recherche.
  3. Appuyez sur Palm mobile hotspot .
  4. Appuyez sur Télécharger gratuitement.

Si l'application Hotspot Mobile Palm n'est pas disponible dans le catalogue d'applications, il se peut que votre fournisseur de service mobile ne prenne pas en charge cette application.
