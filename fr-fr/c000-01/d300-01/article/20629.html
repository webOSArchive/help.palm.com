Qu'est-ce que le hotspot mobile Palm ?

Transformez en un clin d'œil votre téléphone Palm webOS™ en hotspot Wi-Fi pouvant être utilisé simultanément par cinq appareils. Lors de vos déplacements, bénéficiez d'un accès Internet mobile pour votre ordinateur portable tout en offrant une connexion à vos amis ou collègues. Développée par Palm tout spécialement pour la plateforme webOS™, cette application vous permet de créer un hotspot Wi-Fi où que vous vous trouviez dans une zone de couverture de votre fournisseur de service mobile.

Lorsque vous utilisez le hotspot mobile Palm, nous vous recommandons de créer un hotspot sécurisé (procédure). Un hotspot sécurisé est protégé par une expression de passe. Communiquez ou non votre expression de passe pour contrôler qui accède à Internet via votre téléphone. Si vous créez un hotspot ouvert (procédure), vous ne pourrez pas contrôler l'accès d'autres utilisateurs à Internet via votre téléphone. Vous pouvez cependant vous en tenir à un hotspot ouvert, si vous travaillez en groupe restreint et n'éprouvez pas le besoin d'établir un hotspot sécurisé et de définir une expression de passe.

Le hotspot mobile Palm a une portée d'environ 50 mètres. La portée réelle varie selon l'environnement, comme l'existence d'obstacles physiques et d'interférences radio.

Si vous souhaitez seulement parcourir le Web sur votre téléphone, utilisez le navigateur Web de votre téléphone (procédure).

REMARQUE : cette application nécessite la fourniture d'une connexion de données rapide de la part de votre fournisseur de service mobile, ce qui peut entraîner des frais supplémentaires ou un surcoût de l'abonnement mensuel. Renseignez-vous sur les tarifs applicables auprès de votre fournisseur de service mobile avant d'utiliser cette application. Sachez que si vous ne disposez pas d'un forfait de données illimité, vous risquez de cumuler des frais importants en utilisant cette application, en particulier si vous l'utilisez en itinérance. Lorsque l'application Hotspot Mobile Palm est active, les autres applications de votre téléphone continuent de fonctionner et toutes les données envoyées ou reçues sont comptabilisées dans votre forfait, le cas échéant. (disponible uniquement dans les zones de couverture sans fil).
