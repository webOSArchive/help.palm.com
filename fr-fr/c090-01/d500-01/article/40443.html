Recherche d'informations

À l'aide de la fonctionnalité Tapez maintenant, vous pouvez rechercher des contacts et applications sur votre smartphone ou lancer des recherches sur le Web.

  1. Ouvrez la vue Cartes (procédure) > appuyez sur le champ Tapez maintenant.
  2. Commencez par taper le terme recherché.
  3. Effectuez l'une des opérations suivantes :
    • Pour ouvrir une application : appuyez sur l'icône.
    • Pour contacter quelqu'un : appuyez sur une adresse email pour rédiger un message. Appuyez sur le nom d'une personne pour afficher ses coordonnés. Si un smartphone est connecté à votre appareil, appuyez sur un numéro de téléphone pour appeler la personne ou appuyez sur pour envoyer un SMS.
    • Pour rechercher un contact dans la Liste d'adresses globale (LAG) de votre société : appuyez sur Recherche d'adresses globale.
    • Pour effectuer une recherche sur le Web : appuyez sur une option de recherche.
    • Pour rechercher des emails contenant le terme que vous avez saisi : appuyez sur Email.
    • Pour rechercher des signets ou des pages Web que vous avez visitées et contenant le terme que vous avez saisi : appuyez sur Liens et historique.
    • Pour afficher une carte d'un lieu que vous avez saisi : appuyez sur Cartes.
    • App Catalog : pour rechercher des applications correspondant au terme de recherche dans l'App Catalog, appuyez sur App Catalog HP.

    • Pour créer un événement de calendrier, un message ou un autre élément : appuyez sur Nouveau [élément].

    Pour rechercher une application, vous pouvez saisir le nom de l'application ou un mot clé. Par exemple, Réunions pour Calendrier.

    Lorsqu'aucun contact ni aucune application ne correspond aux critères de recherche saisis, les options de recherche Web s'affichent automatiquement.
