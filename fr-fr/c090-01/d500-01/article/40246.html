Mise à jour ou suppression d'un compte de facturation
  1. Ouvrez ! App Catalog  > Menu (en haut, à gauche) > Préférences et comptes.
  2. Appuyez sur le nom de votre compte.
  3. Pour modifier les informations d'une carte de crédit : appuyez sur le nom de la carte de crédit.
  4. Pour modifier le mot de passe ou l'email de reçu : appuyez sur l'entrée actuelle sous Le mot de passe est requis ou Envoyer les reçus, et sélectionnez un nouveau mot de passe ou entrez une nouvelle adresse email.
  5. Si vous y êtes invité, entrez le mot de passe de votre compte HP webOS > appuyez sur Soumettre.
  6. Effectuez l'une des opérations suivantes :
    • Pour modifier les informations de votre compte : modifiez les informations sur le compte > appuyez sur Terminé.
    • Pour supprimer les informations du compte : appuyez sur Supprimer la carte de crédit > Supprimer la carte de crédit.
