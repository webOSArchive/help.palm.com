Configuration des options de déverrouillage de l'écran
  1. Ouvrir ! Écran et verrouillage .
  2. Dans Verrouillage de sécurité, appuyez sur Non, puis appuyez sur l'une des options suivantes :

    Code de sécurité : entrez un code de votre choix de longueur quelconque > entrez à nouveau ce code de sécurité > Terminé.

     

    Mot de passe : entrez un mot de passe constitué de la combinaison de chiffres et de lettres de votre choix > entrez à nouveau ce mot de passe > Terminé

     

     

    IMPORTANTIMPORTANT :

     Pensez à noter votre code secret et votre mot de passe et à les conserver dans un endroit sûr. Si vous oubliez votre code PIN ou votre mot de passe, vous devrez réinitialiser votre appareil à distance et vous risquez de perdre des données.

    Important Close
  3. Appuyez sur Verrouillage après pour déterminer si l'écran se verrouille après une période donnée, ou après désactivation de l'écran.
  4. Pour désactiver le déverrouillage sûr : dans Déverrouillage sûr, appuyez sur Non > entrez le code secret et le mot de passe > Terminé.
