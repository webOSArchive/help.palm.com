Gestion des applications téléchargées
  1. Ouvrir ! Gestionnaire de logiciels .
  2. Appuyez sur Applications pour gérer les applications téléchargées qui apparaissent dans l'écran de lancement ou appuyez sur Autres pour gérer les applications et services en arrière-plan qui n'apparaissent pas dans l'écran de lancement.
  3. Appuyez sur un nom d'application ou de service.
  4. À partir de l'écran donnant des renseignements précis sur l'application sélectionnée, effectuez l'une des opérations suivantes.
    • Pour ajouter une critique, appuyez sur Faire une critique de l'app. Appuyez sur les étoiles pour évaluer l'application. Appuyez sur le nom que vous voulez afficher avec votre critique. Appuyez sur Appuyer ici pour saisir > saisissez votre critique > Envoyer. 
    • Pour signaler un problème avec une application : appuyez sur Signaler un problème > Bogue > appuyez sur un type de problème. Appuyez sur Appuyer ici pour saisir > saisissez la description du problème > Envoyer.
    • Pour supprimer une application : appuyez sur Supprimer > Supprimer.
