Réception et lecture des emails
  1. Ouvrir Email .
  2. Si les dossiers du compte requis sont masqués, appuyez sur le nom du compte dans le volet de gauche pour les afficher.
  3. Appuyez sur le dossier dont vous voulez vérifier les messages . Si la synchronisation ne démarre pas, appuyez sur .
  4. Appuyez sur un message dans le volet central pour l'ouvrir.
  5. (Facultatif) Pour afficher un message en plein écran, appuyez sur dans le panneau de droite.
  6. (Facultatif) Procédez de l'une des façons suivantes :
    • Pour appeler un numéro : appuyez sur le numéro > . Pour passer un appel téléphonique, votre appareil doit être connecté à un smartphone webOS (procédure).
    • Pour envoyer un email à une adresse email : appuyez sur l'adresse email > entrez votre message > appuyez sur Envoyer.
    • Pour vous rendre sur une adresse Web : appuyez dessus.
    • Pour copier le corps d'un email en vue de le coller dans un mémo ou un autre document : appuyez sur le texte du message > appuyez sur Menu (en haut, à gauche) > appuyez sur Modifier > Tout sélectionner. Appuyez sur le texte sélectionné > appuyez sur Copier.

L'application Email synchronise les messages chaque fois que vous ouvrez un dossier de courrier. L'application synchronise également les messages sur un planning que vous pouvez définir (procédure). Pour synchroniser vos messages alors qu'un dossier est déjà ouvert : appuyez sur .
