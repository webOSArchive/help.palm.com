Uso de Servicios de ubicación
  1. Abre Servicios de ubicación .
  2. Toca Activo o Inactivo para alguna de las siguientes opciones:

    3. Ubicación automática: cuando la opción Ubicación automática está activada, las aplicaciones pueden identificar automáticamente tu ubicación. Cuando está desactivada, una aplicación pide permiso para identificar la ubicación. Toca Listo para activar Servicios de ubicación solo en esa sesión.

    Marcar fotos con Geotags: solo está disponible cuando la opción Ubicación automática está activada. Cuando la opción Marcar fotos con Geotags está activada, se marcan las fotos con las coordenadas de latitud y longitud de la ubicación en la que se tomaron.

    Recopilación de datos en segundo plano: solo está disponible cuando los servicios de Google están activados. Cuando está activada Recopilación de datos en segundo plano, se recopilan datos de ubicación anónimos del teléfono. Estos datos se utilizan para mejorar la calidad de Servicios de ubicación.


  3. (Optativo) Para activar y desactivar el GPS: Menú (parte superior izquierda) > Ubicarme mediante > GPS.

    4. La función GPS, activada de forma predeterminada, proporciona información de ubicación a las aplicaciones que lo soliciten.


  4. (Optativo) Para activar o desactivar los servicios de Google: Menú (parte superior izquierda) > Ubicarme mediante > Usar los Servicios de Google.

    La primera vez que actives los servicios de Google se te pedirá que aceptes los términos de uso y decidas si deseas o no permitir la recopilación de datos anónima. Realiza las siguientes acciones:

    • Lee las condiciones de uso.
    • Si decides no permitir la recopilación de datos anónima, toca la marca de verificación para quitarla.
    • Toca Permitir o No permitir.

Servicios de ubicación proporciona información sobre la ubicación a aplicaciones que lo soliciten. Por ejemplo, una aplicación de navegación podría necesitar tu ubicación para proporcionarte las indicaciones precisas.
