Búsqueda de información

Puedes buscar contactos y aplicaciones en el teléfono, así como en la Web.

  1. Abre la vista Carta (consulta cómo) o el Iniciador (consulta cómo).
  2. Realiza una de las siguientes acciones:

     

    • Comienza a escribir el término de búsqueda.
    • Utiliza Editar > Pegar para introducir un término de búsqueda.
  3. Realiza una de las siguientes acciones:

     

    • Para abrir una aplicación: toca el icono.
    • Para ponerte en contacto con alguien: toca un número de teléfono para realizar una llamada, una dirección de correo electrónico para comenzar un mensaje nuevo, etc. Toca un nombre para mostrar la información de contacto de la persona.
    • Para buscar un contacto en la lista global de direcciones (GAL) de la empresa: toca Búsqueda global de dirección.
    • Para buscar en la Web: toca el término de búsqueda introducido para abrir una lista de opciones de búsqueda en la Web > toca la opción.

    Para buscar una aplicación, puedes introducir el nombre de la aplicación o una palabra clave (por ejemplo, Reuniones para Calendario o Texto para Mensajería).

    Si no se encuentran los contactos o aplicaciones, las opciones de búsqueda en la Web se muestran automáticamente.

    También puedes buscar entradas o archivos en aplicaciones como Contactos, Memos, Vista de documentos o Vista de PDF introduciendo algunas palabras del nombre de un archivo o memo, o el nombre o las iniciales de un contacto.
