Exploración de App Catalog
  1. Abre App Catalog .
  2. Lee las condiciones de uso y toca Aceptar.
  3. Desliza el dedo hacia la derecha o hacia la izquierda sobre las miniaturas para ver aplicaciones destacadas, o realiza una de las siguientes acciones:
    • Para buscar por nombre: toca Buscar > introduce el término de búsqueda > Intro .
    • Para buscar aplicaciones por categoría: toca Categorías. Toca para ver una lista de categorías. Toca para ver una nube de etiquetas. 
    • Para ver las aplicaciones añadidas más recientemente: toca Novedades.
    • Para ver las aplicaciones más populares: toca Aplicaciones más populares.
  4. Para ordenar la lista de aplicaciones o ver los resultados por fecha, orden alfabético, clasificación o precio, toca uno de los iconos de la parte inferior de la pantalla.
  5. Toca el nombre de una aplicación para mostrar sus detalles.
  6. En la pantalla de detalles de la aplicación, realiza una de las siguientes acciones:
    • Para obtener una vista más grande: toca una captura de pantalla.
    • Para leer las reseñas: toca Reseñas.
    • Para enviar a un amigo un enlace a esta aplicación: toca > Compartir > Correo electrónico o Mensaje de texto.
    • Para ir a la página de inicio del desarrollador: toca Página de inicio del desarrollador.
    • Para ir a la página de asistencia del producto del desarrollador: toca Asistencia (si está disponible).
