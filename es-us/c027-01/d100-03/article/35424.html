Obtención de ayuda si una actualización de software falla

Durante una actualización de software inalámbrica, existe una pequeña posibilidad de que el teléfono se bloquee y deje de responder a los toques o la presión de las teclas. Palm® webOS Doctor™ restaura las funciones del teléfono y lo actualiza con las últimas versiones del sistema operativo y otro software del sistema. Después de utilizar la herramienta, los datos del perfil de Palm de los que hayas hecho copia de seguridad se restaurarán de forma automática en el teléfono.

Necesitarás lo siguiente para utilizar la herramienta:

  • Cable USB que se incluye con el teléfono
  • Una computadora de escritorio con lo siguiente:
    • Windows XP, Vista® o 7 o Mac OS 10.5
    • Un puerto USB disponible (no a través de un concentrador ni una conexión USB secundaria)
    • La última versión de Java instalada en la computadora de escritorio (disponible desde java.com)
    • 200 MB de espacio de almacenamiento disponible
  • Conexión a Internet con capacidad para descargar archivos de gran tamaño (de 10 MB a 200 MB)

Para descargar la herramienta, visita palm.com/support.
