Creación de una lista de reproducción
  1. Instala software multimedia de terceros que permita la sincronización de música con un teléfono webOS, por ejemplo, el software que se describe en palm.com/sync-solutions.
  2. Crea la lista de reproducción siguiendo las instrucciones del software de terceros.
  3. Conecta el teléfono a la computadora con el cable USB y toca Unidad USB en el teléfono.
  4. Sigue las instrucciones del software de terceros para sincronizar la lista de reproducción con el teléfono.
  5. Para encontrar la lista de reproducción en el teléfono: Abre Música  > Listas reproducción.
