Marcado del número nacional de emergencias

Para marcar el número nacional de emergencias (como 911, 112 o 999), realiza lo siguiente:

  1. Pulsa el botón de encendido para activar la pantalla si es necesario.
  2. Arrastra hacia arriba para desbloquear la pantalla, si es necesario.
  3. Realiza una de las siguientes acciones:
    • Si Desbloqueo seguro no está activado (consulta cómo), toca el centro del área de trazos para mostrar la vista Carta, si no se muestra ya. Introduce el número de emergencia y toca el número para marcarlo. No tienes que pulsar Opción para introducir los números.
    • Si Desbloqueo seguro está activado, toca Tocar para hacer una llamada de emergencia y, a continuación, toca Llamada de emergencia. Edita el número si es necesario y, a continuación, toca para marcarlo.

Si el teléfono está desactivado (es decir, el smartphone se encuentra en modo Avión), no tienes que activarlo. Intenta automáticamente conectarse a una red para realizar la llamada.
