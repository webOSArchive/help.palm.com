Exploración de App Catalog
  1. Abre App Catalog .
  2. Lee las condiciones de uso y toca Aceptar.
  3. Toca el icono de una aplicación destacada o realiza una de las siguientes acciones:
    • Para buscar por nombre: escribe un término de búsqueda > Intro .
    • Para buscar aplicaciones por categoría: toca el encabezado Buscar categorías y, a continuación, toca una categoría. Si el encabezado sigue apareciendo, tócalo para seleccionar subcategorías.
    • Para limitar la búsqueda mediante filtro: toca un icono en la parte inferior de la pantalla.


      		•  Aplicaciones destacadas (solo en la pantalla de inicio)

      		Todas las aplicaciones de la categoría (solo en la pantalla de una categoría/subcategoría)
      Principales aplicaciones de pago

      		Principales aplicaciones gratuitas

      		Aplicaciones más recientes

       

    • Para buscar solo en las aplicaciones destacadas, de pago principales, gratuitas principales o nuevas: toca en primer lugar el icono correspondiente de la parte inferior de la pantalla y, a continuación, introduce un término de búsqueda.
  4. Toca el nombre de una aplicación para mostrar sus detalles.
  5. En la pantalla de detalles de la aplicación, realiza una de las siguientes acciones:
    • Para obtener una vista más grande: toca una captura de pantalla.
    • Para leer las reseñas: toca Reseñas.
    • Para compartir información sobre esta aplicación con un amigo: toca Compartir > Correo electrónico o Mensaje de texto.
    • Para ir a la página de inicio del desarrollador: toca Página de inicio del desarrollador.
    • Para ir a la página de asistencia del producto del desarrollador: toca Asistencia (si está disponible).
    • Para ver un video de la aplicación en la aplicación YouTube del teléfono: toca el enlace a YouTube (si está disponible).
