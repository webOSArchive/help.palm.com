Cambio de aplicaciones predeterminadas
  1. Abre Información del dispositivo  > Menú (parte superior izquierda) > Aplicaciones predeterminadas.
  2. Para encontrar tipos de datos y tipos de archivos para los cuales hay más de una aplicación disponible, busca nombres de aplicaciones que aparezcan en negro.
  3. Realiza cualquiera de las siguientes acciones o ambas:
    • Para cambiar la aplicación predeterminada que se utilizará para distintas funciones: en Tipos de datos, toca la aplicación que se muestra actualmente para un tipo de dato determinado y toca una aplicación diferente.
    • Para cambiar la aplicación predeterminada que se utilizará para distintos tipos de archivos: en Tipos de archivo, toca la aplicación que se muestra actualmente para un tipo de archivo determinado y toca una aplicación diferente.

Para anular el ajuste predeterminado de la aplicación al buscar la ubicación de una dirección en el mapa: mantén presionada la tecla Opción , toca la dirección y toca una aplicación.

Si tienes más de una aplicación en el teléfono que puede realizar una función determinada, puedes cambiar la aplicación predeterminada que se utilizará para esa función.

Además, si tienes más de una aplicación en el teléfono que puede abrir un tipo de archivo concreto, puedes cambiar la aplicación predeterminada que se utilizará para abrir ese tipo de archivo.
