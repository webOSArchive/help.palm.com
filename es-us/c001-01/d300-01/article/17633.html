Selección de un archivo de la carpeta de tonos como tono de llamada
  1. Abre ! Sonidos y tonos .
  2. Toca el tono mostrado para abrir la lista completa de tonos.
  3. Toca el nombre del tono para seleccionarlo. Toca para reproducir el tono.
  4. Para establecer el Volumen de tono, arrastra el control deslizante.

El tono que se selecciona aquí se aplica de forma general a todas las llamadas entrantes. También es posible asignar un tono exclusivo para un contacto (consulta cómo).
