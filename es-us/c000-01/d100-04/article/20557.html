Creación de un Wi-Fi móvil abierto
  1. Abre Wi-Fi móvil de Palm .
  2. La primera vez que abras el Wi-Fi móvil de Palm, toca OK para continuar.
  3. Toca Inactivo para activar el Wi-Fi móvil de Palm.
  4. Si estás conectado a una red Wi-Fi, toca Continuar para desconectarla.

    NOTA: el teléfono debe seguir conectado a la red de tu proveedor de servicios inalámbricos, por lo que no debes utilizar el modo Avión para desactivar Wi-Fi.

  5. Realiza una de las siguientes acciones:
    • Si es la primera vez que ejecutas el Wi-Fi móvil de Palm: toca Crear red abierta.
    • Si ya tienes configurado un Wi-Fi móvil seguro: toca WPA/WPA2 Personal > Abre.
  6. (Optativo) Para personalizar el nombre de tu Wi-Fi móvil, toca el nombre en Información de la red > introduce un nuevo nombre > Listo.
  7. Después de crear el Wi-Fi móvil, en el dispositivo que quieras conectar a Internet (por ejemplo, el portátil), utiliza la configuración de conexión Wi-Fi para conectarte a tu Wi-Fi móvil de Palm. Una vez establecida la conexión, puedes navegar por la Web en el portátil.
