Selección manual de la configuración de red

Antes de empezar

Obtén la siguiente información de tu proveedor de servicios inalámbricos:

  • El nombre de punto de acceso a Internet (APN)
  • El nombre de usuario y la contraseña para acceder al APN de Internet
  • El nombre APN de mensajería multimedia (MMS APN)
  • El nombre de usuario y la contraseña para acceder al APN de mensajería multimedia
  • El nombre del centro de servicios de mensajería multimedia (MMSC)
  • El nombre del servidor proxy de mensajería multimedia (proxy de MMS)
  • El tamaño máximo de mensajería multimedia

BYB Close

  1. Abre Teléfono  > Menú (parte superior izquierda) > Preferencias y cuentas.
  2. Dentro de Red, en Manual, toca Inactivo para cambiar la configuración de red manual de Inactivo a Activo.
  3. Toca Editar configuración de red.
  4. Toca APN de Internet e introduce la configuración obtenida del proveedor de servicios inalámbricos.
  5. Toca APN de MMS e introduce la configuración obtenida del proveedor de servicios inalámbricos.
  6. Toca Cambiar la configuración.

Si el teléfono no puede leer la configuración de red para la tarjeta SIM durante la configuración, puedes introducir la configuración de forma manual.
