Lectura de mensajes de texto/multimedia
  1. Abre Mensajería .
  2. Toca una conversación para ver el tema del mensaje.
  3. (Optativo) Realiza una de las siguientes acciones:
    • Para ver un archivo adjunto en un mensaje multimedia: toca el nombre de un archivo de música, video o vCard. Las fotos adjuntas se muestran dentro del mensaje. Para guardar una imagen JPG adjunta, toca la imagen > Copiar foto.
    • Para llamar a un número: toca el número > .
    • Para enviar un mensaje de correo electrónico: toca la dirección de correo electrónico > Enviar correo.
    • Para acceder a una dirección Web: tócala.
