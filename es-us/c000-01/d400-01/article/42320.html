Configuración del correo electrónico: Microsoft Exchange y Lotus Notes

Antes de empezar

Obtén la siguiente información de tu proveedor de correo electrónico o del administrador del sistema:

  • Nombre del servidor de correo o dirección IP para recibir correo
  • Nombre de dominio del servidor
  • Nombre de usuario y contraseña

BYB Close

  1. Abrir Correo electrónico .
  2. Realiza una de las siguientes acciones:
    • La primera vez que abras Correo electrónico, toca Microsoft Exchange.
    • En cualquier otro momento: en Correo electrónico, toca Menú (parte superior izquierda) > Preferencias y cuentas > Añadir cuenta > Microsoft Exchange.
  3. Introduce la dirección de correo electrónico y la contraseña > Configuración manual.
  4. Introduce la información del servidor de correo entrante, de acuerdo con la información del servidor que te proporcionó el proveedor de correo electrónico o el administrador del sistema (consulta Antes de empezar, más arriba). Introduce el nombre del servidor o la dirección IP en el campo Servidor de correo entrante.
  5. Toca en Iniciar sesión.
  6. Para mantener segura la cuenta de Exchange, es posible que tengas que crear una contraseña o PIN, que se solicitará para desbloquear el teléfono. Toca Establecer contraseña o Establecer PIN > introduce una contraseña o PIN > Aceptar.
  7. Toca Activo/Inactivo para cada aplicación para la que desees descargar datos desde esta cuenta de Exchange.
  8. Toca Crear > Aceptar.
