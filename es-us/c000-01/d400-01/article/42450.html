Búsqueda en la Web
  1. Abre la vista Carta (consulta cómo) o el Lanzador (consulta cómo).
  2. Comienza e escribir el término de búsqueda o la dirección de un sitio Web como, por ejemplo, hp.com.
  3. Realiza una de las siguientes acciones. Puede que necesites desplazarte hacia abajo para ver los motores de búsqueda en la Web.
    • Para ir a un sitio Web: si has escrito la dirección de un sitio Web, toca Ir al sitio web.
    • Para buscar en la Web mediante el motor de búsqueda predeterminado: toca el nombre del motor de búsqueda. El motor de búsqueda aparece directamente debajo de las entradas de contactos coincidentes.
    • Para ver términos de búsqueda sugeridos desde el motor de búsqueda predeterminado según los caracteres que has introducido: toca Sugerir a la derecha del nombre del motor de búsqueda predeterminado. Si no se encuentran contactos, aplicaciones u otros elementos en el teléfono, se muestran sugerencias automáticamente. Toca una sugerencia para buscar ese término.
    • Para buscar en la Web con un motor de búsqueda diferente: en Inicio & búsqueda, toca el nombre de un motor de búsqueda. Si el motor de búsqueda es para un sitio Web específico, como Wikipedia, el teléfono busca solo en ese sitio. Para ver más opciones de búsqueda, toca Más en Inicio & búsqueda.
