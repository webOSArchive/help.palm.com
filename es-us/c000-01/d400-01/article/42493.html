Obtención de ayuda si una actualización de software falla

Durante una actualización inalámbrica del software, existe una pequeña posibilidad de que el smartphone se bloquee y deje de responder a los toques o las pulsaciones de las teclas. HP webOS Doctor restaura las funciones del smartphone y lo actualiza con las últimas versiones del sistema operativo y otro software del sistema. Después de utilizar la herramienta, los datos de la cuenta de HP webOS de los que se haya hecho copia de seguridad se restaurarán de forma automática en el smartphone.

Necesitarás lo siguiente para utilizar la herramienta:

  • Cable USB que se incluye con el smartphone
  • Un ordenador de escritorio con lo siguiente:
    • Windows XP, Vista® o 7 o Mac OS 10.5
    • Un puerto USB disponible (no a través de un concentrador ni una conexión USB secundaria)
    • La última versión de Java instalada en el ordenador de escritorio (disponible desde java.com)
    • 200 MB de espacio de almacenamiento disponible
  • Conexión a Internet con capacidad para descargar archivos de gran tamaño (de 10 MB a 200 MB)

Para descargar la herramienta, visita hpwebos.com/support.
