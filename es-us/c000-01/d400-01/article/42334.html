Recepción y lectura de mensajes de correo electrónico
  1. Abrir Correo electrónico .
  2. Toca para mostrar las carpetas de cuenta > toca una carpeta.
  3. Toca un mensaje para abrirlo.
  4. (Optativo) Realiza una de las siguientes acciones:
    • Para llamar a un número: toca el número > .
    • Para enviar un mensaje de texto: toca y mantén presionado el número > toca Texto.
    • Para añadir un número a Contactos: toca y mantén presionado el número > toca Añadir a Contactos. Toca Guardar como nuevo para crear un contacto nuevo o bien, toca Añadir a existente para añadir este número a un contacto existente.
    • Para enviar un mensaje de correo electrónico: toca la dirección de correo electrónico > introduce el mensaje > .
    • Para acceder a una dirección Web: tócala.
    • Para compartir un enlace a un sitio Web: mantén presionada Opción  > toca el enlace > Enviar enlace.
    • Para copiar un enlace a un sitio Web: mantén presionada Opción  > toca el enlace > Copiar URL.
    • Para copiar un correo electrónico, de forma que puedas pegarlo en una nota o en otro documento: Menú (parte superior izquierda) > Editar > Copiar todo.
  5. Toca o  para ver el mensaje anterior o siguiente.

La aplicación Correo electrónico sincroniza los mensajes cuando abres una carpeta de correo. También sincroniza los mensajes de acuerdo a una programación predeterminada (consulta cómo). Para sincronizar mensajes cuando ya tienes una carpeta abierta: toca .
