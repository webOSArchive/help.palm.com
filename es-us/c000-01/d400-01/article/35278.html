Iniciar el modo exposición

Realiza una de las siguientes acciones:

  • Cuando la pantalla esté apagada y el teclado cerrado, coloca el smartphone en una base HP Touchstone. El modo exposición se iniciará automáticamente.
  • Si colocas el smartphone en una base Touchstone mientras la pantalla está activa, o cuando la pantalla está apagada pero el teclado está abierto, el modo exposición se inicia cuando el smartphone haya estado inactivo durante el periodo definido en la configuración de Bloqueo de pantalla. La pantalla no se bloquea.
  • Abre Exposición  > Comenzar exposición. De este modo se activa el modo exposición, independientemente de si el smartphone se encuentra en una base Touchstone.

Nota: si colocas el dispositivo en una base Touchstone cuando tienes una llamada activa o estás viendo un vídeo en la aplicación YouTube, el modo exposición no se inicia.
